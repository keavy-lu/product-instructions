# 新增全新地圖

### 如何新增全新地圖？

<mark style="background-color:purple;">設定位置：選定地圖欄 -> 新增地圖</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-09-04 12.21.20.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊選定地圖欄的「新增地圖」
2. 輸入該地圖的地圖名稱、尺寸與地圖的描述

{% hint style="info" %}
尺寸為 50x50 的地圖適合 20 人的賽場；尺寸為 100x100 的地圖適合 110 人的賽場。細則可以參考[此連結](https://boniotw-my.sharepoint.com/:x:/r/personal/bonio_share_bonio_com_tw/_layouts/15/Doc.aspx?sourcedoc=%7BFA5BBD32-15F1-4568-8889-4963A6D499E1%7D\&file=%E9%9B%BB%E7%AB%B6%E4%BA%BA%E6%95%B8%E8%88%87%E5%9C%9F%E5%9C%B0%E5%A4%A7%E5%B0%8F.xlsx\&action=default\&mobileredirect=true)
{% endhint %}
