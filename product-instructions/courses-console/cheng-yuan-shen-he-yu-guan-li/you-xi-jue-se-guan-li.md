# 遊戲角色管理

### 如何在課程管理介面中打開遊戲角色的權限？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 成員審核與管理 -> 遊戲角色管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-14 11.59.55.png" alt="" width="563"><figcaption></figcaption></figure>

找到需要調動遊戲角色權限的帳號，可以從前三列的權限欄位中查看該帳號當前所持有的權限類別 e.g. 課程管理權限、課程題庫管理權限老師權限

1. 課程管理權限：開啟後，即可在課程管理介面進行任務設置、成員管理及課程題庫管理等功能
2. 課程題庫管理權限：開啟後，即可管理課程題庫
3. 老師權限：開啟後，即可在教師後台進行學生管理、作業管理及個人題庫管理等功能

如需要進一步編輯權限，可點擊對應權限下方的「更改」，調整權限。

{% hint style="info" %}
課程擁有者與主題式管理權限的角色權限開啟需要使用管理者介面進行設置，詳請可參考[此連結](../../admin-courses/gamecharacters.md#ru-he-da-kai-you-hu-jiao-se-de-quan-xian)
{% endhint %}
