# 我的課程列表

{% hint style="info" %}
此功能為「管理者介面的課程管理」的閹割版本，僅能調整部份內容。

如需要調整更多設置項目，請參考[此連結](../guan-li-zhe-hou-tai-quan-ke-cheng-shi-jie-she-ding/xi-tong-she-ding/ke-cheng-guan-li.md)
{% endhint %}

### 如何在課程管理介面中編輯課程？

<mark style="background-color:purple;">設定位置：課程管理介面 -> Landing Page / Home Page -> 某一世界的編輯課程欄位的「編輯」</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-14 10.15.06.png" alt="" width="563"><figcaption></figcaption></figure>

點擊該課程的「編輯」後，即可

1. 更動該課程的課程名稱
2. 調整該課程是否為公開課程（玩家可於首頁的課程列表中直接進入該課程）
3. 編輯該課程簡介
4. 變動該課程的圖示

{% hint style="warning" %}
如果我的課程列表中並無出現欲編輯的世界，原因可能有二：

1. 該課程在系統設定的課程管理裡，其顯示課程設定值為「否」，詳請請參考[此連結](../guan-li-zhe-hou-tai-quan-ke-cheng-shi-jie-she-ding/xi-tong-she-ding/ke-cheng-guan-li.md#ru-he-bian-ji-ke-cheng-shi-jie-de-zi-xun)
2. 尚未取得該課程的「課程管理權限」，詳請請參考[管理者後台的做法](../admin-courses/gamecharacters.md#ru-he-da-kai-you-hu-jiao-se-de-quan-xian)或是課程管理介面的作法
{% endhint %}

{% hint style="warning" %}
如果無法調整該課程是否成為公開課程，可能是因為權限不足，詳請請查看[此連結](../admin-courses/gamecharacters.md#ru-he-da-kai-you-hu-jiao-se-de-quan-xian)
{% endhint %}
