# 課程題庫（舊版）

### 如何更動課程題庫的名稱？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫</mark>

<figure><img src="../../.gitbook/assets/截圖 2024-01-10 10.35.08.png" alt="" width="563"><figcaption></figcaption></figure>

點擊目前使用中的題庫，並編輯題庫名稱，儲存後即可更動該課程題庫的題庫名稱。

### 如何更動課程題庫的使用權限？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫</mark>

<figure><img src="../../.gitbook/assets/截圖 2024-01-10 10.35.08.png" alt="" width="563"><figcaption></figcaption></figure>

點擊目前使用中的題庫，並點選使用權限的下拉式選單，儲存後即可更動該課程題庫的使用權限。

{% hint style="info" %}
權限種類：

1. 僅限教師使用：僅有教師能夠在教師後台創建作業時能夠選用
2. 師生皆可使用：除了教師能夠在教師後台創建作業時能夠選用外，學生也可以於自選章節中選擇，進行答題
{% endhint %}

### 如何綁定內容供應商？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-29 10.41.28 (1).png" alt="" width="563"><figcaption></figcaption></figure>

將課程題庫指定至想要的內容供應商。

{% hint style="warning" %}
如果此功能並無顯示於側邊功能欄，原因可能是沒有權限，若需要打開該權限，可查看[該連結](../admin-courses/gamecharacters.md#ru-he-da-kai-you-hu-jiao-se-de-quan-xian)
{% endhint %}

### 如何建立題目分類架構？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫 -> 主頁面或是導覽欄的「編輯章節」</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-11 17.49.27 2 (1).png" alt="" width="563"><figcaption></figcaption></figure>

1. 初次進入畫面時，會有一個已經設置好的科目，為了將題目更好地進行分類，可以選擇替已經設置好的科目改名成需要的科目名稱
2. 若需要更多科目，可以點選「新增科目」進行新增
3. 請依照需求依序建立冊次與章節

{% hint style="warning" %}
建立科目後才能建立冊次；建立冊次後才能建立章節，不可跳過
{% endhint %}

### 如何上傳題目？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫 -> 主頁面或是導覽欄的「上傳題目」</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-11 17.49.27 3.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇題型
   * 選擇題：可設為單選或多選，也可以自由增設選項（最多八個選項）
   * 是非題：選項僅有是與否兩個選項
   * 閱讀題組：玩家於閱讀完長文後，根據文章內容作答（目前僅支援選擇題，規則同選擇題）
   * 填充題：玩家根據問題的前後文填入簡短答案
   * 申論題：玩家根據問題內容填入長篇論述（目前功能僅能從導覽欄中點選）
2. 選擇出題範圍（此處的範圍選項會連動到編輯章節時所創建的章節架構）
3. 選擇該題所屬的難度類別（此處的難度類別為出題時的其中一個篩選條件）
4. 填入題目內容（閱讀題組的組內題數最多十題）、選項（是非題、選擇題或是閱讀題組才需要填入）、答案（填充題才需要填入）詳解或影音詳解、答題時限

{% hint style="info" %}
可使用注音字型的欄位有：文章（閱讀題組）、題目內容、選項內容（選擇題）、詳解

使用說明：

1. 在文字輸入區將工具列的字型切換為 Bpmf Zihi Sans
2. 進入線上工具「[字嗨注音 IVS 字型輸入](https://buttaiwan.github.io/bpmfvs/)」的頁面，輸入框內輸入文字，若需切換破音字，可以透過右下方的「開始」進行調整
3. 確認完所有注音皆正確後，將文字複製、並貼到文字輸入區中
4. 完成後，玩家可在該題目的作答介面切換顯示成注音字型
{% endhint %}

{% hint style="info" %}
題目、文章、答案、詳解、影音詳解可以另外設定自動播放影片，選擇時間後，當用戶進入指定頁面時會自動播放影片
{% endhint %}

{% hint style="info" %}
答題時限若選擇自動調整，系統會依據作答情形[^1]自動調整至最佳時間；若自己選擇時限，則會固定不再變動
{% endhint %}

{% hint style="warning" %}
題目內的單張圖片大小最大不要超過 5MB，否則會讀取非常久
{% endhint %}

### 如何編輯題目？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫 -> 題目列表</mark>

<figure><img src="../../.gitbook/assets/截圖 2024-01-09 17.20.04.png" alt="" width="563"><figcaption></figcaption></figure>

可透過科目、冊次、章節與狀態[^2]等篩選器找尋特定題目，或是使用題目 ID 來搜尋題目。若需編輯題目，需點擊欲編輯的題目右側設定欄位的「編輯」，即可：

1. 變更題目所屬的章節範圍
2. 變更題目難度
3. 編輯題目內容
4. 編輯詳解與語音詳解
5. 調整答題時限

### 如何刪除題目？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫 -> 題目列表</mark>

<figure><img src="../../.gitbook/assets/截圖 2024-01-09 17.20.04拷貝.png" alt="" width="563"><figcaption></figcaption></figure>

可透過科目、冊次、章節與狀態[^2]等篩選器找尋特定題目，或是使用題目 ID 來搜尋題目。若需刪除題目，需點擊需要欲編輯的題目右側設定欄位的「刪除」，並點擊「確認」後即可成功刪除該題目。

### 如何批量上傳題目？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫 -> 上傳題目 -> 批量上傳</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-11 17.53.10.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇該次批量上傳所使用的題目類型
2. 下載上傳範例的 Excel 檔案
3. 按照 Excel 內的格式，填上題目
4. 填完題目後，回到此頁面，點選「選擇檔案」，將編輯好的 Excel 檔與多媒體檔一併選起，並點選「上傳」
5. 上傳後分為兩種情境
   * 若有錯誤，請依照錯誤訊息修正後重新選擇檔案上傳
   * 若成功，將顯示成功訊息，耐心等候 1～2 小時即可上傳完畢

{% hint style="warning" %}
如果課程題庫並無批量上傳的功能，原因是因為該使用者並無該權限，詳情請參考[該連結](../admin-courses/game-setting/ke-cheng-guan-li-jie-mian-kai-guan.md#ru-he-da-kai-ke-cheng-guan-li-jie-mian-zhong-ke-cheng-ti-ku-de-pi-liang-shang-chuan-gong-neng)
{% endhint %}

### 如何批量上傳多語系的題目？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫 -> 上傳題目 -> 批量上傳</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-11 17.53.10.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇該次批量上傳所使用的題目類型（目前僅閱讀題組不開放使用）
2. 下載上傳範例的 Excel 檔案
3. 按照 Excel 內的格式，填上題目
4. 命名各多語系檔案：以 \_locale\_xxxxx.xlsx 結尾 e.g. 慈濟環保題\_locale\_en.xlsx
   * 英文：慈濟環保題\_locale\_en.xlsx
   * 繁體中文：慈濟環保題\_locale\_zh-TW.xlsx
   * 簡體中文：慈濟環保題\_locale\_zh-CN.xlsx
   * 印尼文：慈濟環保題\_locale\_id.xlsx
   * 泰文：慈濟環保題\_locale\_th.xlsx
5. 壓縮多個不同語言題目檔案成 zip 檔，並點選此頁面「選擇檔案」、選取 zip 檔後點擊「上傳」
6. 上傳後分為兩種情境
   * 若有錯誤，請依照錯誤訊息修正後重新選擇檔案上傳
   * 若成功，將顯示成功訊息，耐心等候 1\~2 小時即可上傳完畢

{% hint style="info" %}
1) 所有相同內容但不同語系的題目，必須編號順序相同、題目數量相同
2) 除了題目相關的題目、答案、詳解等項目是做多個語系，其他科目、冊次、章節、難度填入的內容，都必須要和主語系一樣
3) 創建完多語系題庫後，可用於任務出題與電競用題
{% endhint %}

{% hint style="info" %}
FAQ：

1. 若只有上傳兩種語系：繁中、英文，若切換成簡中時會以繁中代替；若切換成印尼文時會以英文代替
2. 與個人題庫不同，課程題庫開放編輯或更新多語系的題目。同一題可以藉由切換不同語系檢查或是在錯題回報管理裡進行修正
{% endhint %}

{% hint style="warning" %}
1. 如果課程題庫並無批量上傳的功能，原因是因為該使用者並無該權限，詳情請參考[該連結](../admin-courses/game-setting/ke-cheng-guan-li-jie-mian-kai-guan.md#ru-he-da-kai-ke-cheng-guan-li-jie-mian-zhong-ke-cheng-ti-ku-de-pi-liang-shang-chuan-gong-neng)
2. 單個檔案限制為 5MB，總檔案限制為 100MB
{% endhint %}

### 如何處理錯誤回報？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 課程題庫 -> 導覽欄的「錯題回報管理」</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-11 17.49.27 2 (2).png" alt="" width="563"><figcaption></figcaption></figure>

1. 以科目、冊次與章節來篩選錯題（若選擇不篩選，可以點擊篩選項目右側的Ｘ解除篩選）
2. 以不同的 Tab 查看錯題回報
   * 問題回報：未處理的錯題（包含處理後又被回報的錯題）
   * 暫時下架：已處理且當時選擇<mark style="color:red;">暫時下架</mark>的錯題
   * 已審核：已處理且當時選擇<mark style="color:red;">此題正常</mark>或是<mark style="color:red;">立即修正</mark>的錯題
3. 處理錯題回報
   * 暫停下架：適用於<mark style="color:red;">該題有誤</mark>但<mark style="color:red;">無法當下解決</mark>，當選擇「暫停下架」，有引用該題庫的世界的教師不能於出題時看到、選擇該題目
   * 此題正常：適用於<mark style="color:red;">該題無誤</mark>，當選擇「此題正常」，教師可以於出題時正常出題
   * 立即修正：適用於<mark style="color:red;">該題有誤</mark>且<mark style="color:red;">可於當下解決</mark>，當選擇「立即修正」，有引用該題庫的世界的教師可以直接編輯該錯題的題目內容

[^1]: 若選擇「自動調整」，系統會自動調整成平均答題時間。一開始會預設為 30 分鐘，如果很少人答題，或是大家都花很久時間，時間會非常長。

[^2]: 已上架、待審核、未上架
