# 任務設置

{% hint style="info" %}
此功能為「管理者介面的世界任務管理」的閹割版本，僅能派發世界任務至該課程世界、為所有取得世界任務的課程世界提早結束該世界任務。

如需要調整更多設置項目，請參考[此連結](../../admin-all-courses-setting/ying-yun-guan-li/shi-jie-ren-wu-guan-li.md)。
{% endhint %}

### 如何在課程管理介面中派發世界任務至該世界？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 任務設置 -> 任務設置 -> 一般任務的「指派任務」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-10-23 10.50.47.png" alt="" width="563"><figcaption></figcaption></figure>

點擊某尚未指派成世界任務的一般任務的「指派任務」，即可將該一般任務派發成該課程世界內的世界任務。

{% hint style="info" %}
僅有課程管理者於教師後台創建的一般任務（尚未派發給任何班級）才會出現在此清單中
{% endhint %}

### 如何在課程管理介面中提早結束某世界任務？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 任務設置 -> 任務設置 -> 現有世界任務的「提早結束」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-14 10.36.04 (1).png" alt="" width="563"><figcaption></figcaption></figure>

點擊某世界任務的「提前結束」，即可替「所有取得該世界任務的世界」提前結束該任務。

{% hint style="info" %}
使用提早結束功能將導致此任務從所有課程世界的任務書消失，且此任務在管理者介面的世界任務管理會跑到「已結束」的分頁（無法單一地為某課程提前結束世界任務）
{% endhint %}
