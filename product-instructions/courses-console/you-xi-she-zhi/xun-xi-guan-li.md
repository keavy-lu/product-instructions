# 訊息管理

### 如何在課程管理介面中發送至課程世界的公告訊息？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 遊戲設置 -> 訊息管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-14 12.59.20 (2).png" alt="" width="563"><figcaption></figcaption></figure>

點擊「新增訊息」後即可進入該新訊息的設定頁面​。輸入訊息標題與訊息內容後選擇發佈時間，並選擇是否以彈窗顯示訊息，即可點擊「新增」，將訊息傳送給該課程世界的所有玩家。

發送完的訊息細節（包括標題、內文與發送時間）也可以於課程管理介面的訊息管理頁面中查看。

{% hint style="info" %}
訊息管理的訊息內容輸入窗已套用 HTML 編輯器
{% endhint %}

### 如何在課程管理介面中編輯發送給課程世界的公告訊息？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 遊戲設置 -> 訊息管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-14 12.59.20 (1).png" alt="" width="563"><figcaption></figcaption></figure>

點擊「編輯 icon 」後即可進入訊息內容的編輯頁面，可進一步編輯訊息標題與訊息內容、彈窗顯示期間與其顯示頻率。如需要取消顯示該訊息，可以點擊「垃圾桶 icon 」刪除該訊息。

{% hint style="info" %}
訊息管理的訊息內容輸入窗已套用 HTML 編輯器
{% endhint %}
