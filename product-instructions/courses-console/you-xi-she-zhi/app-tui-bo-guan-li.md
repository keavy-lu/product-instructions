# APP 推播管理

### 如何新增發送 APP 推播通知？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 遊戲設置 -> APP 推播管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2024-01-09 14.53.53.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「新增通知」後即可進入該則通知的設定頁面​。輸入通知標題與通知內容後選擇發佈時間，並選定發布的對象：課程世界內所有玩家或課程世界內的特定玩家[^1]，另外，也可以選擇附上圖片，當通知成功推播時，此圖可以顯示於通知中。

{% hint style="info" %}
APP 推播管理的通知內容輸入窗僅支援 Emoji，並無完整支援 HTML
{% endhint %}

### 如何查看 APP 推播通知的詳細資訊？

<mark style="background-color:purple;">查看位置：課程管理介面 -> 遊戲設置 -> APP 推播管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2024-01-09 14.53.53拷貝.png" alt="" width="563"><figcaption></figcaption></figure>

找到欲查看通知細節的通知，點擊其操作欄位的「查看」開啟側邊欄，即可查看此通知的細節，包括：標題、內容、圖片、發送時間、發送對象。

### 如何編輯 APP 推播通知？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 遊戲設置 -> APP 推播管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2024-01-09 14.53.53拷貝.png" alt="" width="563"><figcaption></figcaption></figure>

#### 編輯**未發送的 APP 推播通知**

找到欲編輯通知設定的未發送通知，點擊其操作欄位的「查看」開啟側邊欄，並接續點擊「編輯」，即可編輯此通知的設定，包括：標題、內容、圖片、發送時間、發送對象。

#### 編輯**已發送的 APP 推播通知**

找到欲編輯通知設定的已發送通知，點擊其操作欄位的「查看」開啟側邊欄，並接續點擊「編輯」，即可編輯此通知的設定，包括：標題、內容、圖片。而發送時間與發送對象則不可編輯。

{% hint style="info" %}
APP 推播管理的通知內容輸入窗僅支援 Emoji，並無完整支援 HTML
{% endhint %}

{% hint style="info" %}
修改已發送 APP 推播通知的邏輯為

1. 新通知會替換掉舊通知
   * 如用戶已經點擊舊通知並開啟 APP，課程管理者隨後才更新通知內容，則用戶會收到新的一則 APP 通知
   * 如用戶尚未點擊通知，課程管理者隨後更新通知內容，則用戶裝置內通知中心的舊通知會消失，改以呈現新通知的資訊
2. 通知若發送給所有課程用戶，則首次發送～修改通知之間加入課程的新用戶，不會收到修改後的通知
{% endhint %}

### 如何刪除 APP 推播通知？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 遊戲設置 -> APP 推播管理</mark>

#### **刪除未發送的 APP 推播通知**

找到欲刪除通知的未發送通知，點擊其操作欄位的「刪除」，並於確認視窗點擊「確認」，即可刪除此通知。

#### **刪除已發送的 APP 推播通知**

找到欲刪除通知的已發送通知，點擊其操作欄位的「刪除」，並於確認視窗點擊「確認」，即可刪除此通知。

{% hint style="info" %}
成功刪除後，已經點擊過通知的用戶仍會收到「Notification deleted.」的通知；尚未點擊過通知的用戶，其舊通知會被新通知「Notification deleted.」覆蓋
{% endhint %}

[^1]: 

    能夠以下方式作為輸入方式：

    1. 發送對象帳號
    2. 發送對象 email
    3. 發送對象 UID
