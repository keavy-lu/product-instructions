# 參數管理

<figure><img src="../../../.gitbook/assets/截圖 2023-08-14 13.10.56.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何調整課程世界的道具掉落機率與能量回復程度參數？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 遊戲設置 -> 參數管理</mark>

直接調整道具掉落機率與能量回復程度的倍率。如需要另外顯示常駐訊息於遊戲介面的右上角，可另外開啟「顯示常駐訊息」的 Toggle。

{% hint style="info" %}
1. 道具掉落機率為一倍時，其機率為 10%，約攻擊或訓練 10 次掉落一次道具
2. 能量回復程度為一倍時，其每分鐘自動補充的能量值為 0.2 / 分（一小時恢復 12 能量）
{% endhint %}

### 如何開放課程世界玩家使用省力模式？

<mark style="background-color:purple;">設定位置：課程管理介面 -> 遊戲設置 -> 參數管理</mark>

關閉「禁止省力模式」的 Toggle。

{% hint style="info" %}
省力模式為一種特別的能量消耗答題模式，詳情參考[此連結](../../game-interface/ge-ren-tou-xiang/sheng-li-mo-shi.md#sheng-li-mo-shi-shi-shen-me)
{% endhint %}
