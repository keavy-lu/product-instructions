# 賽後設定

### 如何設定電競賽的勝利條件？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 當前世界 -> 賽後設定</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-30 12.27.22.png" alt="" width="563"><figcaption></figcaption></figure>

設定勝利者的條件：個人排名 / 年級排名 / 校內排名 / 學校總排名 / 隊伍排名 / 隊內排名。

### 如何設定賽後顯示資訊？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 當前世界 -> 賽後設定</mark>

1. 設定是否顯示結算介面、顯示排行榜前 N 名
2. 設定勝利者前往的課程世界與訊息文字
3. 設定落敗者前往的課程世界與訊息文字
