# 一般電競賽世界設定

### 如何設定一般電競賽的世界設定？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 當前世界 -> 世界設定</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-30 18.27.25.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="success" %}
完成設定時，請點擊「儲存」保存設定
{% endhint %}

#### **賽場開放設定**

1. 設定賽場開放時間
2. 設定是否禁賽黑名單

{% hint style="info" %}
若電競賽沒有設定開賽時間，而玩家進入該為設定開放時間的賽場時，會導到錯誤頁
{% endhint %}

#### **題目設定**

1. 搜尋帳號與帳號下的題目集（作業），並將其設定為電競賽所使用的題目來源
2. 設定出題順序：全部隨機、照作業包順序
3. 設定出題規則：只出現未答過的、未答過和答錯的兩者隨機出現（機率相同），直到所有題目答對為止、未答過的都答過一遍後隨機出答錯的，直到所有題目答對為止[^1]
4. 設定完成出題規則後的出題方式：不再出題、全部題目隨機出題
5. 設定詳解：不能看詳解、可以看詳解、強制跳詳解

{% hint style="info" %}
1) 使用多語系題目語言，請配合使用[課程管理的編輯功能設定語言](../../../admin-all-courses-setting/xi-tong-she-ding/ke-cheng-guan-li.md#ru-he-bian-ji-ke-cheng-shi-jie-de-zi-xun)
2) 若修改題目來源，且有玩家在地圖，下一題就會以新的規則與順序進行
3) 出題規則中若含有重複出題，則當申論題有出現在題目中，會排除申論題的重複出題
{% endhint %}

#### **機制設定**

1. 設定即時排行：我的排名、前十名排名（排名會出現在畫面右側）
2. 設定點擊土地時是否顯示隊名
3. 設定是否顯示倒數計時
4. 設定是否開放使用省力模式
5. 設定是否開放攻擊同隊的領土
6. 設定土地是否以顏色標示

#### **其他設定**

1. 設定能量值上限
2. 設定土地血量是否為滿血（野地、對手領地皆為滿血）
3. 設定是否會掉落能量道具、商店是否會販售能量道具
4. 設定是否會掉落傳送道具、商店是否會販售傳送道具
5. 設定是否會掉落攻擊道具、商店是否會販售攻擊道具
6. 設定是否允許玩家重生

{% hint style="info" %}
1) 此課程的課程管理者，可至課程管理後台下，遊戲設置的參數管理，調整此課程的能量回復速度與掉寶率，詳情可參考[此連結](../../../courses-console/you-xi-she-zhi/can-shu-guan-li.md#ru-he-diao-zheng-ke-cheng-shi-jie-de-dao-ju-diao-luo-ji-lv-yu-neng-liang-hui-fu-cheng-du-can-shu)
2) 如需開放手動重生，仍需要至管理者介面下，課程設定的進階設定，開啟免費重生功能，詳情可參考[此連結](../../../admin-courses/game-setting/shi-jie-she-ding.md#ru-he-kai-fang-wan-jia-mian-fei-zhong-sheng)
{% endhint %}

#### **背包道具**

點擊新增道具，設定角色出生時背包即有的道具。

{% hint style="info" %}
如列表中並無欲選擇的道具，需要至管理者介面下，遊戲設定的本課程道具數值，開啟可使用的道具，詳情可參考[此連結](../../../admin-all-courses-setting/you-xi-she-ding/ben-ke-cheng-dao-ju-shu-zhi.md#ru-he-bian-ji-chan-yi-ke-cheng-shi-jie-de-dao-ju-shu-zhi)
{% endhint %}

#### **寶藏地形**

只需要設定玩家出生點

* 隨機排列：若將寶藏地形的排列方式設定為隨機排列，其將鎖定選為出生點隨機，無法修改
* 同心圓排列：若將寶藏地形的排列方式設定為同心圓排列，其將鎖定選為出生點同心圓排列，無法修改。最早入場的玩家為圓心，其他玩家以繞圓的方式逐一入場，越晚入場越在外圈

{% hint style="info" %}
當設定完玩家出生點為同心圓排列、點擊儲存後，就會出現「重新排列」的按鈕，點擊「重新排列」後，系統將所有參賽者的座標隨機掉換位置，地圖需要一段時間處理換位置， 此時使用者還是可以繼續做其他設定，重新排列按鈕則變成處理中狀態，重複點擊無效，直到位置交換完畢，按鈕回復原狀態
{% endhint %}

{% hint style="warning" %}
重新排列的按鈕只會在以下條件下有效

1. 比賽尚未開始
2. 玩家出生點為同心圓排列
3. 比賽開始前（該世界沒有任何答題紀錄）
{% endhint %}

#### **排行榜設定**

1. 設定土地數、答對數的積分比例，如選擇 60%:40%，則佔據 9 塊土地、答對 15 題，則該玩家獲得 6\*9 + 4\*15 = 114 分
2. 設定頭像內戰績排名的「隊伍/班級排名」開關
3. 設定頭像內戰績排名的「隊伍/班級內排名」開關
4. 設定頭像內戰績排名的「學校排名」開關
5. 設定頭像內戰績排名的「學校內排名」開關
6. 設定頭像內戰績排名的「年級排名」開關

{% hint style="info" %}
若修改土地數量、答對數量的積分比例，且有玩家在地圖，下一題就會以新的積分規則計算積分
{% endhint %}

#### **權限設定**

1. 新增非本課程帳號的使用者轉播者權限
2. 新增非本課程帳號的使用者觀賽者權限

{% hint style="info" %}
1) 轉播者沒有土地，但可以使用即時排行榜快速移動與即時事件轉播的功能
2) 觀賽者沒有土地，但可以使用即時排行榜快速移動的功能
{% endhint %}

{% hint style="info" %}
一般電競賽不需設定寶藏地形、地形積分設定，此設定值為奪寶信號所用
{% endhint %}

[^1]: 假設甲乙丙丁戊 五題為任務內容，並做錯 甲丙戊三題。在隨機順序出題的情形下，在答完五題後，隨機出現甲丙戊三題，但同一題不會在同一個三題循環內出現兩次，假設第一次答錯題時出現 丙 題，則下兩次答題則隨機出現甲或戊（隨機但不重複）
