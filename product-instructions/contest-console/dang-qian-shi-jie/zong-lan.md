# 總覽

### 如何得知電競賽或是奪寶信號的資訊？

<mark style="background-color:purple;">檢視位置：競賽管理介面 -> 當前世界</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-30 12.21.32.png" alt="" width="563"><figcaption></figcaption></figure>

於當前競賽世界的競賽總覽頁面，可查看：

* 課程世界（電競世界）名稱
* 該課程世界（電競世界）代碼
* 該電競賽創立時間
* 電競賽開放時間
* 電競賽報名時間
* 參賽者人數
* 是否有使用共用報名頁
* 電競賽地圖大小
* 地圖模式
* 電競賽題目來源

{% hint style="info" %}
共用報名頁為使用城市、身份、年級、班級的分流條件，進行篩選後分派至不同賽場的機制
{% endhint %}
