# 連結課程

### 如何連結原課程世界與奪寶信號電競世界？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 當前世界 -> 連結課程</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-30 12.22.14.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇欲開放的課程世界（下拉選單僅出現已打開奪寶開關的課程）
2. 競賽說明：有預設文字，可刪掉自行輸入或修改
3. 題目範圍：可自行輸入，例如：國一國文1\~2章
4. 設定可加入班級代碼：在連結課程裡任何老師創的班級都可以加入，一行貼一個班級代碼，只要玩家有在列表上任一個班級中，不需要輸入密碼就可以參加該場奪寶競賽
5. 設定入場密碼：如果玩家並沒有加入指定班級，可以改以輸入密碼的方式參加該場奪寶競賽

{% hint style="warning" %}
若密碼欄空白，則不論加沒加班級都可以無密碼直接加入奪寶競賽。
{% endhint %}
