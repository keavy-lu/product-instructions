# 特殊電競賽報名頁

### 如何設定特殊電競賽的報名頁設定？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 當前世界 -> 報名頁設定</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-30 12.24.20.png" alt="" width="563"><figcaption></figcaption></figure>

1. 輸入欲設定的報名頁網址後綴（只能輸入小寫英文、數字、底線）
2. 決定是否開啟「[共用報名頁](/broken/pages/RPtf1JlTR42c67fbxLpi#sai-chang-gong-yong-bao-ming-ye-shi-shen-me)」，若開啟，請先設定「報名條件」
   * 設定為主賽場：選擇報名條件中的某項，並勾選對應分流條件，也可以另外新增分流條件
   * 設定為分賽場：選擇該分賽場所對應到的主賽場
3. 決定是否開放具「管理者權限」者參加
4. 決定是否發送報名成功訊息，若開啟，請輸入訊息
5. 設定通知信 Footer（可直接帶入不同預設值）\
   &#x20;\[ 客服信箱 / 客服專線 / Footer ]
6. 設定並輸入報名頁橫幅資訊\
   &#x20;\[ 比賽標題 / 活動橫幅 / 實施對象 / 競賽時間 / 報名時間 / 其他賽制連結（請能輸入 URL）]
7.  設定並輸入區塊資訊（以下資訊會以不同分頁呈現）

    &#x20;\[ 報名須知 / 賽程賽制 / 比賽獎項 / 注意事項 / 聯絡窗口 ]
8. 設定並上傳單位圖片\
   \[ 主辦單位 LOGO / 指導單位 LOGO / 協辦單位 LOGO / 其他贊助 LOGO ]
9. 設定並輸入連結資訊\
   &#x20;\[ 報名手冊連結 / 報名簡章連結 / FB 連結 / YT 連結 / Line 連結 / 微信連結 / QQ 號連結 ]&#x20;
10. 設定報名是否需要手機資訊（帳號、真實姓名、年齡為必填，可新增備註說明必填原因）
11. 設定報名填寫的資訊（聯絡信箱為必填，可新增備註說明必填原因）
    * 身分證字號、國家（可新增備註說明必填原因）
    * 城市、身份：輸入選項（可新增備註說明必選原因）
    * 學校：決定是否輸入預設選項（開啟後才能夠查看「學校排名」與「學校內排名」）
    * 年級：選擇選項（開啟後才能夠查看「年級排名」）
    * 班級：輸入選項（開啟後才能夠查看「隊伍 / 班級排名」和「隊伍 / 班級內排名」）
12. 設定並輸入自訂報名頁題目，可設定單選題或是開放式問題
13. 設定並輸入報名來源：輸入選項
14. 設定該賽制的模式
    * 選擇個人競賽，設定是否需要參賽者填寫家長教師姓名、家長教師信箱
    * 選擇手動分組，設定隊伍各組人數，輸入隊伍名稱備註、隊伍成員備註，並設定是否需要參賽者填寫帶隊者姓名、帶隊者手機、帶隊者信箱
    * 選擇自動分組，設定分組條件（若需開啟，請先設定「報名條件」）
15. 儲存設定

{% hint style="info" %}
1) 網址設定完畢儲存後，即無法更改
2) 如果有使用到共用報名頁的功能，則僅有設定主賽場時才需要實作第三項到第十四項的設定
3) 具管理者權限者具關閉賽場的權限，如有開放這項功能，可能會因為離開賽場而誤關賽場
4) 通知信 footer 會出現您選擇的 icon、名稱，點擊後會前往後方放置的連結
5) 若輸入學校的預設值，會改變所有參賽者的學校（意即所有參賽者皆來自該預設值的學校）若空白則不影響參賽者原設定
6) 具管理者權限者具備關閉賽場的權限，如有開放具「管理者權限」者參加，可能會因為離開賽場而誤關賽場
{% endhint %}

{% hint style="warning" %}
理論上，特殊電競賽是可以使用「手動分組」，但是當分組數量超過選定的特殊地圖的預設組數時，會出現「沒有土地的情況」。因此，建議在現場比賽 e.g. 人數固定的決賽，或是其他人數已配置好的場合中，再使用手動分組。否則皆建議使用「自動分組」功能
{% endhint %}
