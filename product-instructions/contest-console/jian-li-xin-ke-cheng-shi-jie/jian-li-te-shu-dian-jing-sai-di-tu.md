# 建立特殊電競賽地圖

{% hint style="success" %}
在選定地圖前，請確保地圖編輯界面中已有欲使用的地圖，如沒有，請參考[此連結](/broken/pages/UMaqgWzHyRZgLensP0gf)進行設置
{% endhint %}

### 如何建立新課程世界並生成特殊電競賽的地圖？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 建立新課程世界</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-30 18.15.56.png" alt="" width="563"><figcaption></figcaption></figure>

1. 填入競賽名稱
2. 選擇競賽所屬的課程分類
3. 選擇特殊地圖並選定特殊地圖形式

{% hint style="info" %}
如果不確定是否為分組的特殊地圖，可先至地圖編輯介面確認，土地上的兩個數字中間須有逗號
{% endhint %}
