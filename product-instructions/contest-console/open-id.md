# Open ID

### 如何設定 Open ID？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> Open ID</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-30 12.31.21.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇 Open ID 作為來源
2. 選擇 Open ID 裡的縣市欄位
3. 選擇 Open ID 裡的學校欄位（可多選）
4. 選擇 Open ID 裡的年級欄位（可多選）
5. 輸入報名網址或比賽網址
6. 設定開始生效的時間，以及結束的時間
7. 點擊「儲存」

### 如何設定白名單？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> Open ID</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-30 12.31.54.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇 white\_emails 作為來源
2. 輸入欲加入白名單的 Email
3. 輸入報名網址或比賽網址
4. 設定開始生效的時間，以及結束的時間
5. 點擊「儲存」

### 如何設定 tw\_edu？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> Open ID</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-30 12.32.04.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇 tw\_edu 作為來源
2. 選擇欲分流至不同賽場的年級（由 Open ID 的年級欄位去判斷分流條件）
3. 輸入該分流條件下欲進入的報名網址或比賽網址
4. 設定開始生效的時間，以及結束的時間
5. 點擊「儲存」
