# 全課程使用同一任務打魔王戰役

{% hint style="info" %}
執行以下步驟後，其所創建的魔王是屬於任務魔王，而非官方魔王
{% endhint %}

### 如何讓全課程使用同一任務打魔王戰役？

<details>

<summary>第一步：建立作業</summary>

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 新增作業</mark>

1. 選擇題目來源、科目、學年度、冊次與作業章節作為題目來源
2. 選擇出題模式
   * 全自動：輸入總出題數 系統自動隨機出題
   * 半自動：先依難易挑選題數 系統自動隨機出題
   * 手動：手動勾選題目 完全按照您的意願
3. 選擇出題順序
   * 隨機出題：題目由系統隨機出現，每位學生答題順序都會不同
   * 順序出題：由教師設定出題順序，每位學生答題順序皆同
4. 命名作業

</details>

<details>

<summary>第二步：將作業打包成魔王任務（但不進一步派發至班級）</summary>

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單 -> 新增任務</mark>

<img src="../../../.gitbook/assets/截圖 2023-06-30 14.50.25.png" alt="" data-size="original">

（請確保已有所需魔王）

1. 請先選擇「魔王模式」作為任務的模式
2. 填入任務名稱
3. 選擇任務的開始時間與結束時間（時間僅限於 1～20 分鐘）
4. 選擇答案顯示設定（此設定值無意義，皆不會顯示正確答案與詳解）
5. 點選「魔王設定」，並完成以下設定
   * 選擇魔王難度（不同難度的魔王在後台為不同隻魔王，若無法選擇其他難度，代表後台並無設定該難度的魔王）
   * 填入參加人數（參加人數設定越高，魔王血量越高）

</details>

<details>

<summary>第三步：指派該魔王任務成為世界任務，並指派至該課程世界</summary>

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 世界任務管理</mark>

<img src="../../../.gitbook/assets/截圖 2023-08-10 11.08.34.png" alt="" data-size="original">

1. 點擊「新增世界任務」
2. 輸入創建任務時所使用的帳號，並選擇欲派發成世界任務的「任務」
3. 是否開放教師在統計分析的功能中查看班級作答數據
4. 指派任務至課程中

</details>
