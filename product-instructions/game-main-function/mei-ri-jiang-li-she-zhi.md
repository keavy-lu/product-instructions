# 每日獎勵設置

### 每日獎勵的設置流程為何？

每日獎勵分為兩種形式：小夥伴大富翁與月份簽到、週次簽到。

### 如何更新小夥伴大富翁的獎勵？

若要新增更新小夥伴大富翁的獎勵，請先參考 [此 Excel 文件](https://boniotw-my.sharepoint.com/:x:/g/personal/maggie_bonio_com_tw/EZptGDLGaAhDt22cb5SQ-AAB9riqbf3hjTaL1i7OOB6gLw?e=Juiaih) 並與 PM 討論，另外開票更換小夥伴大富翁的獎勵。

{% hint style="info" %}
更新只影響還未生成的獎勵地圖，已發生或進行中的地圖不影響。
{% endhint %}

### 如何更動未來月份的登入獎勵？

<mark style="background-color:purple;">檢視位置：管理者後台 -> 遊戲設定 -> 累積登入獎勵設定</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-02 10.16.19.png" alt="" width="563"><figcaption></figcaption></figure>

系統會呈現出當前月份所設定的登入獎勵，而管理者可以更動未來月份的登入獎勵。

{% hint style="info" %}
獎勵設定只能更動未發生的月份內容，已發生和正進行中的月份獎勵將鎖定無法修改
{% endhint %}

### 如何更新週次獎勵？

若要新增更新週次獎勵，可以先參考[該票](https://redmine.bonio.com.tw/issues/34851)，並與 PM 討論，並另外開票更換週次的七日獎勵。

{% hint style="warning" %}
若要更改週次獎勵，請先與 PM 討論，由 PM 開票更改
{% endhint %}
