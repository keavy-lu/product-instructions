# \[ 情境 ] 連結 LINE OA

### 如何連結家長後台至 LINE OA？

<mark style="background-color:purple;">設定位置：家長後台 -> 導航欄的 LINE OA 連結處</mark>\
（也可透過 \[行銷網站的官方帳號邀請連結 ] 或是 \[ 用戶自行搜尋官方帳號名稱或 ID ] 來綁定 LINE OA）

<figure><img src="../../.gitbook/assets/截圖 2023-08-23 12.13.31 2 (1).png" alt="" width="563"><figcaption></figcaption></figure>

加入 PaGamO LINE 好友後，開啟「PaGamO素養學習」LINE 官方帳號，並點選「綁定帳號」快捷鍵、完成 LINE 的的認證授權後，輸入已創建好的素養學習後台帳號與密碼，點選登入。如出現「已將 LINE 與 PaGamO 素養學習連動」等字樣，即代表完成綁定。

{% hint style="info" %}
PaGamO素養學習官方 LINE 帳號尚未提供用戶自行解除綁定，如需要更換綁定人，需要開 OP 票解除綁定，刪除掉 PaGamO 的授權，可參考[此票](https://redmine.bonio.com.tw/issues/34525)綁定 LINE OA
{% endhint %}
