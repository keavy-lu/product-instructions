# 個人設定



### 如何更改教材購買者於家長後台的密碼？

<mark style="background-color:purple;">設定位置：家長後台 -> 個人設定</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-23 12.13.31 2.png" alt="" width="563"><figcaption></figcaption></figure>

輸入舊有密碼，並反覆輸入兩次新密碼，即可點擊「變更密碼」，更新該帳號的密碼。

{% hint style="info" %}
家長後台無法更改電子信箱，需要透過客服後台更改教材購買者的信箱，詳請請參考[此連結](../customers/jiao-cai-ding-dan-guan-li/gou-mai-zhe-zi-xun.md#ru-he-jian-shi-jiao-cai-gou-mai-zhe-de-zhang-hao-zi-xun)
{% endhint %}

### 如何管理教材購買者於家長後台的登入方式？

<mark style="background-color:purple;">設定位置：家長後台 -> 個人設定</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-23 12.19.12.png" alt="" width="563"><figcaption></figcaption></figure>

點擊欲設定第三方帳號的「前往連結」，完成指定流程，即可完成連結。

{% hint style="info" %}
已經連結的第三方帳號無法從 PaGamO 平台解除連結，若想要解除連結，可參考以下方式：

1. [Facebook 帳號解除連結方式](https://www.facebook.com/help/170585223002660)
2. [Google 帳號解除連結方式](https://myaccount.google.com/connections?filters=2)
3. [Apple ID 帳號解除連結方式](https://www.unclesampig.com/2021/06/appapple-id-google.html)
{% endhint %}

### 如何更改教材購買者的個人資訊與發票設定？

<mark style="background-color:purple;">設定位置：家長後台 -> 個人設定</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-23 12.19.18 (1).png" alt="" width="563"><figcaption></figcaption></figure>

可直接更改教材購買者的真實姓名與手機號碼，並更改該發票的發票類型或是更改「同意折讓」狀態。

### 如何更改教材購買者於家長後台的付款方式設定？

<mark style="background-color:purple;">設定位置：家長後台 -> 個人設定</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-23 12.19.24 (1).png" alt="" width="563"><figcaption></figcaption></figure>

點擊「編輯」後可以直接更新信用卡的資訊。

{% hint style="info" %}
當教材購買者更新信用卡後，系統將試刷 1 元以取得授權，授權成功後會取消該筆交易。所有訂閱教材將於下一次扣款時使用新信用卡
{% endhint %}
