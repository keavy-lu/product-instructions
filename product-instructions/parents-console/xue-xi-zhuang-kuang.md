# 學習狀況

### 如何查看綁定帳號的學習狀況？

<mark style="background-color:purple;">檢視位置：家長後台 -> 學習狀況</mark>

<figure><img src="../../.gitbook/assets/學習狀況.png" alt="" width="563"><figcaption></figcaption></figure>

1. 可於帳號列表中查看各帳號綁定的教材與其教材狀態
2. 可點擊帳號內任一個訂閱狀態教材的側邊「收合欄」，進一步取消訂閱
3. 可點擊帳號內任一個取消訂閱狀態教材的「重新訂閱」，填入信用卡 / 金融卡卡號重新訂閱該教材
4. 可點擊任一個帳號的「學習數據」，連動至學習中心的學習數據（個人學習表現）頁面，選定教材類型、任務完成的時間區間、任務來源，即可：
   * 查看時間區段內的整體表現：總完成任務數、任務平均正確率[^1]、任務平均作答時間[^2]
   * 選定篩選條件，查看該難度或是領域下，時間區段內的正確率趨勢
   * 查看時間區段內的素養能力分析：素養能力下的表現雷達圖（可另外調整過去參考數據：半年[^3]、一年[^4]、過去所有時間[^5]、不顯示）
   * 查看時間區段內的素養領域表現分析：平均正確率[^6]、任務完成個數、任務平均作答時間[^7]
5. 可點擊任一個帳號的「學習數據」，連動至學習中心的學習數據（已完成任務列表）頁面，選定教材類型、任務完成的時間區間、任務來源，即可：
   * 查看單一任務正確率
   * 查看單一任務第一次作答時間
   * 查看單一任務第一次作答錯題
   * 查看文章
   * 查看題目

{% hint style="info" %}
1) 選定時間時可以調整不同時間範圍來呈現最長一年的作答數據
2) 若任務包內有多個題庫（供應商）的題目，對應到不同能力指標，則會把這些題目的能力指標都抓出來，一併加入雷達圖 e.g. 題庫（供應商） A 有 3 個能力指標， 題庫（供應商） B 有 4 個能力指標，這樣雷達圖就會出現 3 + 4 = 7 個指標
3) 當時間區間內孩子所完成的任務不滿 16 個時，學習數據的「正確率趨勢」、「素養領域表現分析」僅供參考，建議等到完成一定數量任務之後再以該數字做為評分標準
4) 每個數據區塊都可以單獨截圖下載
{% endhint %}

[^1]: 每個任務首次作答正確率平均

[^2]: 每個任務首次作答時間的平均

[^3]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去半年數據為 2022/03/01～2022/08/31

[^4]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去一年數據為 2021/09/01～2022/08/31

[^5]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去所有時間數據為 啟用日

    ～2022/08/31

[^6]: 任務正確率總和 / 指定時間區間內完成的任務數 x 100%

[^7]: 各任務首次作答時間的平均
