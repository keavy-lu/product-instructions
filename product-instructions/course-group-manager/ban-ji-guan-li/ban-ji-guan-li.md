# 班級管理

### 如何在師生帳號管理介面新增班級並自動創建帳號？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 班級管理 -> 新增班級</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 13.48.13.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「增加班級」後輸入班級名稱與班級學生數量，即可由帳號管理介面自動創建帳號。若選擇不自動創建帳號，可將班級學生數量設為零。

{% hint style="info" %}
如要批量匯入學生名單至現有班級，可參考[此連結](zhang-hao-guan-li-zhu-jie-mian.md#ru-he-pi-liang-hui-ru-xue-sheng)
{% endhint %}

### 如何替教師進行分組？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 班級管理 -> 教師分組</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 16.59.22.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「教師分組」後選擇該次分組要進入某些班級的教師，並選擇這些教師需要進入的班級。

{% hint style="info" %}
此功能僅能做到將教師加進尚未加入的班級而已，無法做到更動班級內現有的教師名單
{% endhint %}

### 如何修改現有班級的年級名稱標籤？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 班級管理 -> 修改年級</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 17.01.18.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「修改年級」，於班級列表中找到欲更改年級標籤的班級，可點擊年級欄位的下拉式選單，並選擇欲變更的年級標籤。

{% hint style="info" %}
1. 若年級欄位的下拉式選單中並無出現想要選取的年級，需先行[新增年級](nian-ji-guan-li.md#ru-he-ti-jiao-shi-hou-tai-de-ban-ji-guan-li-xin-zeng-nian-ji-xuan-xiang)
2. 若想要更動單一年級內所有班級的年級標籤，請改使用[調整年級](xin-xue-nian-guan-li.md#ru-he-jiang-chan-yi-nian-ji-nei-suo-you-ban-ji-de-nian-ji-biao-qian-zuo-diao-zheng)的功能
{% endhint %}
