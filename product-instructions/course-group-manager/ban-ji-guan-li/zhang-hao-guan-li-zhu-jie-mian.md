# 帳號管理主介面

### 如何移除班級內學生？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 某一班級的「查看詳細資料」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 14.25.01 1 (1).png" alt="" width="563"><figcaption></figcaption></figure>

切換至學生的分頁後，選取欲需移除的學生，並點擊「移除學生」。

### 如何替已建立好的班級新增或移除教師？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-17 11.35.49.png" alt="" width="563"><figcaption></figcaption></figure>

將左側教師名單中的教師拖入至指定班級區塊，即可新增至該班級。若需要將教師從班級中移除，則需要點擊該教師名稱右方的「 刪除 icon」。

{% hint style="warning" %}
每個班級至少要有一位教師，當班級內只剩一位教師時，該教師無法被移除該組別。 若需移除該位教師，請先加入其他教師或整個班級刪除再新增。
{% endhint %}

### 如何刪除班級？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 13.55.21.png" alt="" width="563"><figcaption></figcaption></figure>

點擊某一班級的「刪除」，並點擊彈窗提示的確認即可刪除該班級。

### 如何批量匯入學生？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 某一班級的「查看詳細資料」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 13.44.21.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「匯入學生」
2. 分行貼上**現有**的學生帳號
3. 點擊「儲存」

{% hint style="info" %}
1) 貼上學生帳號時，帳號必須要是現存帳號，如果要直接創建帳號，請參考[此連結](ban-ji-guan-li.md#ru-he-zai-shi-sheng-zhang-hao-guan-li-jie-mian-xin-zeng-ban-ji)
2) 批量匯入的起始座號為：該班級當前最大座號+1
{% endhint %}

### 如何替現有班級自動創建帳號？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 某一班級的「查看詳細資料」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 15.38.56 (1).png" alt=""><figcaption></figcaption></figure>

將鼠標移動至班級人數，點擊「向上 icon」或是直接編輯班級人數，即可替現有班級自動創建帳號。

{% hint style="info" %}
班級人數無法往下調整，唯有[刪除班級內成員](zhang-hao-guan-li-zhu-jie-mian.md#ru-he-yi-chu-ban-ji-nei-xue-sheng)才可以降低班級人數
{% endhint %}

### 如何批量編輯學生座號？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 某一班級的「查看詳細資料」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 14.18.16 (1).png" alt="" width="563"><figcaption></figcaption></figure>

切換至學生的分頁後，可直接編輯座號，並點擊儲存。系統並不會自動檢查座號是否重複，請小心使用。

### 如何替班級內的單一學生重設密碼？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 某一班級的「查看詳細資料」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 15.25.07.png" alt="" width="563"><figcaption></figcaption></figure>

切換至學生的分頁後，點擊密碼欄位的「Reset」，並選擇重設密碼的方式：與帳號相同或另外建立密碼。

{% hint style="warning" %}
帳號管理者僅能夠替符合以下條件的學生重設密碼

1. 該課程的課程代碼符合規則（4 \~ 5 碼英文組成）
2. 用戶的帳號前綴符合該課程代碼
3. Email 是 guest email （尚未綁定信箱）
{% endhint %}

### 如何批量替班級內學生重設密碼？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 某一班級的「查看詳細資料」</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 14.25.01.png" alt="" width="563"><figcaption></figcaption></figure>

切換至學生的分頁後，選取欲批量重設密碼的學生，並點擊「重設密碼」。

{% hint style="info" %}
自行重設過密碼的用戶無法在師生帳號管理介面中批量重設初始的密碼，僅能透過密碼欄位的「Reset」來替班級內的**單一**學生重設密碼
{% endhint %}

{% hint style="warning" %}
帳號管理者僅能夠替符合以下條件的學生重設密碼

1. 該課程的課程代碼符合規則（4 \~ 5 碼英文組成）
2. 用戶的帳號前綴符合該課程代碼
3. Email 是 guest email （尚未綁定信箱）
{% endhint %}
