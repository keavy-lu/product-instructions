# 班級管理

### 班級管理與我的班級差異為何？

帳號管理介面的「班級管理」與教師後台的「我的班級」異同如下

### 將師生帳號管理介面的班級往上或往下移動的功能為何？

將班級往上或是往下移動時，可以達到以下作用

1. 調整素養數據的班級 Selector 順序（但在教師後台的班級順序不會改變）
2. 校級管理者與班級數較多的老師能夠編排檢視順序，可以快速找到特定班級
