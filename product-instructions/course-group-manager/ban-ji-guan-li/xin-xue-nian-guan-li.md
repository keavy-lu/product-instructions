# 新學年管理

### 如何更改單一年級內所有班級的年級標籤？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 新學年管理 -> 調整年級</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 17.12.05.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「調整年級」，於年級列表中找到欲更改年級標籤的年級，可點擊動作欄位的下拉式選單，並選擇「直升」或「畢業」，若選擇直升，需要另外填寫該年級變更後的年級標籤。

{% hint style="info" %}
若選擇畢業，則該年級會從師生帳號管理介面中隱藏
{% endhint %}

{% hint style="info" %}
若想要單一調整某一班級的年級標籤，請改使用[修改年級](ban-ji-guan-li.md#ru-he-xiu-gai-xian-you-ban-ji-de-nian-ji-biao-qian)的功能
{% endhint %}

### 如何批量更改單一年級內所有班級的名稱？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理 -> 某年級 -> 新學年管理 -> 調整班級</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-18 17.12.14.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「調整班級」，並選擇欲更改班級名稱的年級。輸入年級下所有舊班級的新班級名稱後點擊「儲存」即可完成修改。
