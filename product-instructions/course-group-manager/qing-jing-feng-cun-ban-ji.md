# \[ 情境 ] 封存班級

### 如何封存班級？

目前封存班級的功能並無做在前台，僅能夠透過開 OP 票來執行該機制。

如果要執行封存班級，需參考[此表](https://redmine.bonio.com.tw/attachments/34049)，羅列出環境、指定世界代碼、要封存的班級代碼、封存班級的時間，並附在 OP 票上。
