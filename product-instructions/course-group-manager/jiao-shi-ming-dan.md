# 教師名單

### 如何新增教師所屬的學科類型？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 教師名單</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-16 18.14.14.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增學科」
2. 輸入欲新增學科的名稱
3. 點擊「確認增加」

### 如何創建該課程世界的教師帳號？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 教師名單</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-11-29 10.37.49.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「新增教師帳號」、輸入教師名稱後，點擊「繼續」即可完成新增。

{% hint style="info" %}
如果需要以帳號管理介面創建出的教師帳號登入教師後台，請先將該教師移入某學科，再點擊該學科的「查看詳細資料」，即可查看到該教師的帳號與該教師的帳號與預設密碼
{% endhint %}

### 如何移除教師帳號於該課程世界的教師權限？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 教師名單</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-11-29 10.37.49拷貝 (1).png" alt="" width="563"><figcaption></figcaption></figure>

點擊「編輯」，並接續點擊欲移除教師權限的教師帳號旁的「移除 icon」 即可移除該名教師的教師權限。

### 如何替已建立好的學科新增或移除教師？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 班級管理</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-17 10.15.16.png" alt="" width="563"><figcaption></figcaption></figure>

將左側教師名單中的教師拖入至指定學科區塊，即可新增至該學科。若需要將教師從學科中移除，則需要點擊該教師名稱右方的「 刪除 icon」。
