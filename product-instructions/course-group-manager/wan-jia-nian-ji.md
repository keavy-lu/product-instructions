# 玩家年級

### 何時需要設定玩家的年級？

帳號管理者需要於開啟進階設定後，接續設定此功能，以正確地讀取玩家年級，並加諸進階設定的限制

{% hint style="info" %}
* 此功能的影響範圍不僅限於學生，教師也會納入影響範圍
* 若教師同時在多個不同年級中任教，則系統會讀取最小年級數作為該教師的年級
{% endhint %}

### 如何設定玩家的年級？

<mark style="background-color:purple;">設定位置：師生帳號管理介面 -> 玩家年級</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-18 18.21.16.png" alt="" width="563"><figcaption></figcaption></figure>

選擇欲設定年級的群組，並輸入該群組欲設定的年級後點擊「儲存」，即可儲存此設定，並應用此年級至題目對應年級功能。

{% hint style="warning" %}
如果無法於師生帳號管理介面中看到玩家年級功能，原因是系統尚未開啟進階設定的題目對應年級功能，詳情參考[此連結](../admin-courses/game-setting/ti-mu-she-ding.md#ru-he-da-kai-shi-sheng-zhang-hao-guan-li-de-wan-jia-nian-ji-de-gong-neng-bing-she-ding-ti-mu-dui-yin)
{% endhint %}
