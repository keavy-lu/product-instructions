# 內容供應商

### 如何新增或編輯內容供應商

<mark style="background-color:purple;">設定位置：管理者後台 -> 產品管理 -> 主題式閱讀管理 -> 內容供應商</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-15 10.33.40.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增內容供應商」或是現有內容供應商的「編輯」
2. 輸入代碼
3. 輸入供使用者辨識的名稱
4. 上傳 LOGO
5. 選擇開放自動授權（開啟後，使用者將可跳過授權，直接看到此內容供應商的所有任務）
6. 選擇授權到期日
7. 輸入供應商的描述（使用者可見）

{% hint style="info" %}
編輯現有內容供應商時，無法更動代碼
{% endhint %}

{% hint style="info" %}
如果要讓內容供應商顯示 Tab 的話，需要另外開 OP 票新增
{% endhint %}
