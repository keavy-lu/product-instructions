# PaCode 管理

{% hint style="warning" %}
查詢時請確認目前所在世界為該 PaCode 的兌換世界
{% endhint %}

{% hint style="info" %}
一組 PaCode 對應到一組獎勵
{% endhint %}

### **如何查詢派發過的 PaCode 與其使用紀錄？**

<mark style="background-color:purple;">檢視位置：管理者後台 -> PaCode 管理 -> PaCode 管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-09 15.52.49 (1).png" alt="" width="563"><figcaption></figcaption></figure>

輸入已派發過的 PaCode 的 PaCode ID 或 PaCode，點擊送出即可取得該 PaCode 的紀錄。

### **如何查詢過某用戶的 PaCode 使用紀錄？**

<mark style="background-color:purple;">檢視位置：管理者後台 -> PaCode 管理 -> PaCode 管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-09 15.52.49 2.png" alt="" width="563"><figcaption></figcaption></figure>

輸入該用戶的帳號、姓名或是電子信箱，點擊送出即可取得該 GC 使用 PaCode 的紀錄。
