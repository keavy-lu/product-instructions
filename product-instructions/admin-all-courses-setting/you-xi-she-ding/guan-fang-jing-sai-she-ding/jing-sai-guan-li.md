# 競賽管理

### 如何新增官方競賽？

<mark style="background-color:purple;">設定位置：管理者後台 -> 遊戲設定 -> 官方競賽設定 -> 新增 or 複製</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-09 14.28.02.png" alt=""><figcaption></figcaption></figure>

請依照以下內容填寫該競賽的各個設定項目

<table><thead><tr><th width="193">設定項目</th><th>填入內容</th></tr></thead><tbody><tr><td>課程</td><td>填入可進入該競賽的課程世界</td></tr><tr><td>名稱</td><td>填入該競賽的名稱</td></tr><tr><td>簡介</td><td>填入該競賽題目範圍</td></tr><tr><td>模式</td><td>選擇任一種競賽模式：大亂鬥、團體戰</td></tr><tr><td>人數</td><td><p>輸入該競賽的參與人數限制</p><p>（選擇大亂鬥才須填寫）<br>（系統內的限制為 3000人）</p></td></tr><tr><td>隊伍數</td><td><p>輸入該競賽的隊伍數限制</p><p>（選擇團體戰才須填寫）</p></td></tr><tr><td>各隊人數</td><td><p>輸入該競賽的隊伍內人數限制</p><p>（選擇團體戰才須填寫）</p></td></tr><tr><td>開賽時間，每週</td><td>選擇開賽時間（星期制）</td></tr><tr><td>開賽時間，開始時間</td><td>選擇開賽時間（24 小時制）</td></tr><tr><td>開賽時間，結束時間</td><td>選擇閉賽時間（24 小時制）</td></tr><tr><td>詳解設定</td><td>選擇任一種詳解設定</td></tr><tr><td>參賽資格（非必選）</td><td>選擇可進入這場賽事的參賽資格，來源為「使用者分群管理」頁面。若未選擇，則全部使用者都可進入賽場</td></tr><tr><td><a data-footnote-ref href="#user-content-fn-1">資格說明</a>（非必填）</td><td>若有設定參賽資格，請在此處說明參賽資格為何</td></tr><tr><td><a data-footnote-ref href="#user-content-fn-2">競賽說明</a></td><td>請依照以下文字作為輸入範例<br> - 大亂鬥<br>此競賽為大亂鬥模式（個人戰）， 進入後直接點擊地圖答題，在Ｘ時間內擴張自己的領土，每攻下一個領土可獲得Ｘ分，答對一題獲得Ｘ分，競賽結束時依照積分決定勝負，名次高的將獲得稀有獎勵。<br><br> - 團體戰<br>此競賽為團體戰模式， 加入分組後直接點擊地圖答題，在Ｘ時間內擴張自己的領土，每攻下一個領土可獲得Ｘ分，答對一題獲得Ｘ分，競賽結束時依照團隊的總積分決定勝負，名次高的將獲得稀有獎勵。</td></tr><tr><td>帳號</td><td>請填寫創建競賽題目集所使用的帳號</td></tr><tr><td>題目集</td><td>選擇欲作為競賽題目集的題庫來源（可多選）<br>（此處只出現分派為 <strong>競賽之盾的作業</strong> ，或 <strong>沒有出成任務的作業）</strong></td></tr><tr><td>出題順序</td><td>選擇任一種出題順序</td></tr><tr><td>出題規則</td><td><a data-footnote-ref href="#user-content-fn-3">選擇任一種出題規則</a></td></tr><tr><td>完成出題規則後</td><td>選擇出完所題目後得出題規則</td></tr><tr><td>能量值上限</td><td>輸入該競賽的能量值上限（基本值為 300）</td></tr><tr><td>能量回復速度</td><td><p>選擇該競賽的能量回復速度，此外能量上限加倍，以上數值也加倍。</p><p>300 能量上限，１倍回復速度 = 0.2 / 分鐘</p><p>300*X能量上限，Y倍回復速度 = 0.2*X*Y / 分鐘</p></td></tr><tr><td>積分比例</td><td>選擇土地數、答對數的積分比例，如選擇 60%:40%，則佔據 9 塊土地、答對 15 題，則該玩家獲得 6*9 + 4*15 = 114 分</td></tr><tr><td>名次獎勵</td><td>選擇各名次獎勵</td></tr></tbody></table>

{% hint style="info" %}
除新增外，也可以點擊現有賽場的「複製」：複製該賽場的設定，快速新增一筆
{% endhint %}

{% hint style="info" %}
出題規則中若含有重複出題，則當申論題有出現在題目中，會排除申論題的重複出題
{% endhint %}

{% hint style="warning" %}
1. 官方競賽無法跨天，必須當天完成
2. 如果官方競賽已經有人入場後，無法再修改任何設定（包括時間）
{% endhint %}

[^1]: 沒有參賽資格的玩家點選「進場」後，會跳出「不符合參賽資格」的彈窗通知，彈窗上會顯示此欄位文字

[^2]: 此處文字會在點選「說明」按鈕後顯示

[^3]: 假設情境：甲乙丙丁戊五題為任務內容，並做錯甲丙戊三題\
    &#x20;Ａ：照順序出題，再答完戊題後，會從甲題開始出現，再來為丙，再來為戊，直到都答對為止\
    Ｂ：隨機順序出題，則在打完五題後，隨機出現甲丙戊三題， 但同一題不會在同一個三題循環內出現兩次， 假設第一次答錯題時出現 丙 題，則下兩次答題則隨機出現甲或戊（隨機但不重複）
