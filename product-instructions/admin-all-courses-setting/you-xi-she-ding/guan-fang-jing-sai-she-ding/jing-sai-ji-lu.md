# 競賽紀錄

### **如何查詢競賽之盾的戰役紀錄？**

<mark style="background-color:purple;">檢視位置：管理者後台 -> 遊戲設定 -> 官方競賽設定 -> 競賽紀錄</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-09 15.46.02 (1).png" alt="" width="563"><figcaption></figcaption></figure>

選擇欲查詢課程世界、時間段，並點選「搜尋」，即可看到競賽之盾的戰役紀錄。進一步得知

1. 競賽 ID
2. 競賽名稱
3. 題目範圍
4. 競賽模式
5. 參加人數（參加人數 / 人數上限）
6. 平均答題數
7. 平均答對與答錯題數

另外，也可以點擊「下載表單」下載 Excel 檔案。
