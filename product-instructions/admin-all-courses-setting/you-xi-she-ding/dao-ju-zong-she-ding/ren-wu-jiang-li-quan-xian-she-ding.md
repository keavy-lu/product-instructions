# 任務獎勵權限設定

### 如何更動全部課程世界的道具派發權限？

<mark style="background-color:purple;">設定位置：管理者後台 -> 遊戲設定 -> 道具總設定 -> 任務獎勵權限設定</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-09 15.14.23.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
這邊的設定值會影響到該教師或是出題者派發任務時能夠選用到的獎勵有哪一些
{% endhint %}

{% hint style="success" %}
在設定權限前，請先確保有設置完道具的 QP 與設定好世界是否能夠使用該道具
{% endhint %}

點擊「編輯」後，即可調整目前各權限者派發任務獎勵的權限設定，包括

* 免費權限：所有能夠取用教師後台的教師
* 付費教師權限：縣市合作案內的教師（通常會開放比較好的道具給教師們）
* 廣達權限：日前特別為廣達設置的權限，僅廣達出題者可使用
* 內容商[^1]權限：身份為內容商的上題者，僅品學堂任務會發的獎勵
* 管理者：身份為內部員工的上題者

編輯完成後，可以點擊「儲存」將變動項目儲存。

{% hint style="info" %}
開道具的票時，須說明該道具開放哪些權限
{% endhint %}

[^1]: 素養A++、素養品學堂、英語素養
