# 本課程道具數值

{% hint style="info" %}
此功能影響範圍僅有目前所在的課程世界
{% endhint %}

### 如何編輯單一課程世界的道具數值？

<mark style="background-color:purple;">設定位置：管理者後台 -> 系統設定 -> 本課程道具數值</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-09 11.20.37.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「編輯」後，即可調整目前所有道具的數值，包括

* 是否存在於此課程：影響到素養任務道具是否能於該課程世界領取
* QP：教師派發任務獎勵時所耗費的 QP，建議為買價的 1/100
* 買價：於商店買到該商品的價格
* 賣價：販賣背包內商品的商品價格
* 可買：是否能夠在商店中買到該商品
* 可使用：是否能夠在該課程世界使用
* 掉落值：影響該道具的掉落率
* 掉落率：該道具的掉落值 / 全部道具掉落值總和
* 鎖定：鎖定該項道具於該世界的設定值，不受道具總設定更動所影響

編輯完成後，須點擊「儲存」將變動項目儲存。

{% hint style="info" %}
某一道具的掉落機率 = 答對掉落率 ＊ 該道具掉落率
{% endhint %}

{% hint style="info" %}
「本課程道具數值」與「道具總設定」皆為道具設定數值的功能，系統會採計最後設置的數值

e.g. 1 \
如果「國中世界」的Ａ道具 QP 數值設定為 20，也沒有鎖定該道具的數值，而後如果有夥伴更改「道具總設定」的Ａ道具數值，將 QP 值調整為 30，代表所有未鎖定Ａ道具數值的世界，其QP 皆會被更改為 30，包括國中世界。

e.g. 2

如果「道具總設定」的Ａ道具 QP 數值設定為 20，而後有夥伴更改「國中世界」的Ａ道具數值，將 QP 值調整為 30，代表國中世界的Ａ道具 QP 數值將不採計道具總設定的數值，而是直接採計最後調整的數值：30。
{% endhint %}
