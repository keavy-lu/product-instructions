# 魔幣商品

### **如何編輯魔幣商店內販售的商品與其定價？**

<mark style="background-color:purple;">設定位置：管理者後台 -> 遊戲設定 -> BOSS 戰設定 -> 魔幣商品</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-09 13.38.12.png" alt="" width="563"><figcaption></figcaption></figure>

如要上架某一商品，請勾選「上架」欄位，並填寫商品單價。也可以選擇進一步設定特價價格（需要同時設定單價才可以設定特價價格。

{% hint style="warning" %}
「魔幣商店的道具販售價格」與「道具總設定」有功能上的重疊，系統會採計最後設置的數值，設置前請洽 PM

e.g. 1 \
如果在「魔幣商品」裡將Ａ魔幣商品售價設定為 200。而後如果有夥伴更改「道具總設定」的Ａ魔幣商品售價，將售價調整為 300，代表所有世界的Ａ魔幣商品售價皆會被更改為 300。

e.g. 2

如果「道具總設定」的Ａ魔幣商品售價設定為 200。而後如果有夥伴更改「魔幣商品」的Ａ道具數值，將售價調整為 300，代表所有世界的Ａ魔幣商品售價皆會被更改為 300。
{% endhint %}
