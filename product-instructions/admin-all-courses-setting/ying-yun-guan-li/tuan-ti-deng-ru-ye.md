# 團體登入頁

{% hint style="info" %}
若將某一 joseal 社群代碼改為與企業世界相同，屆時由企業世界匯入＋創建的帳號和密碼，可以直接利用 One Bonio 帳號登入 joseal 對應社群

* 已登入PaGamO的情況下，在 joseal 對應社群網址下登入時點擊 One Bonio icon 即可直接進入社群
* 未登入PaGamO的情況下，在 joseal 對應社群網址下登入時點擊 One Bonio icon，輸入帳號密碼後也可以進入社群
{% endhint %}

### 如何新增團體登入頁？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 團體登入頁</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-10 12.17.17.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增團體登入頁」
2. 輸入團體登入頁顯示名稱
3. 選擇欲連結的已建立課程的課程代碼
4. 輸入團體代碼
5. 調整是否在清單中顯示入口
6. 上傳團體 LOGO（不得超過 270px \[ 寬 ]  x 80px \[ 高 ]）
7. 是否開放教師在統計分析的功能中查看班級作答數據
8. 設定客製化頁面
   * 頁面背景顏色
   * 頁面背景圖片
   * 中央方框區背景顏色
   * 中央方框區陰影
   * 所有文字顏色
   * 帳號欄位預設文字：繁體中文、簡體中文、英文
   * 密碼欄位預設文字：繁體中文、簡體中文、英文

{% hint style="success" %}
在創建團體登入頁時，請確認已經[創建好課程](../xi-tong-she-ding/ke-cheng-guan-li.md#ru-he-chuang-jian-xin-de-ke-cheng-shi-jie)
{% endhint %}

{% hint style="info" %}
設定項目：團體代碼

1. 建議為團體英文名稱或縮寫，設定好後不可修改，且不可與其他團體登入頁的代碼重複
2. 僅可使用小寫英數字，字數不限，但建議不要超過 10 個字，以免太難記
3. 此名稱將被使用在：
   1. 團體登入頁網址
   2. App 登入時，要輸入的團體代碼
   3. 用來建立 PaGamO 帳號。在團體登入頁用來登入的帳號，其實在 PaGamO 真正的帳號是由 "團體英文名稱" + "\_portal\_" + "帳號" 所組成\
      （因為擔心與現有帳號重複，因此在英文名稱跟工號之間多加"portal" ）
{% endhint %}

{% hint style="info" %}
設定項目：是否在清單中顯示入口

* 使用者未登入時，課程大廳會顯示團體登入入口按鈕，點擊後會看到入口清單
* 若開關打開，會在清單上顯示此團體登入頁名稱，點擊後即可到團體登入頁；若開關關閉，則不顯示在清單上
{% endhint %}

{% hint style="info" %}
設定項目：暱稱

* 如果名單更新者有填暱稱，則 GC 暱稱為該暱稱
* 如果名單更新者沒有填暱稱，則不更新暱稱（原用戶保有暱稱、新用戶暱稱留空給用戶填）
{% endhint %}

### 如何編輯團體登入頁？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 團體登入頁</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-10 12.30.09.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊現有團體登入頁的「編輯」
2. 調整團體登入頁顯示名稱
3. 調整是否在清單中顯示入口
4. 調整團體 LOGO（不得超過 270px \[ 寬 ]  x 80px \[ 高 ]）
5. 調整客製化頁面
   * 頁面背景顏色
   * 頁面背景圖片
   * 中央方框區背景顏色
   * 中央方框區陰影
   * 所有文字顏色
   * 帳號欄位預設文字：繁體中文、簡體中文、英文
   * 密碼欄位預設文字：繁體中文、簡體中文、英文
