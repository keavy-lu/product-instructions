# 一次性折扣碼

### 如何新增一次性折扣碼？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 優惠折扣碼設定 -> 一次性折扣碼</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-14 16.23.53.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增一次性折扣碼」
2. 輸入折扣代碼前綴
3. 新增描述
4. 選擇商務類別
5. 選擇折扣金額
6. 選擇適用產品
7. 選擇優惠的開始時間與結束時間
8. 輸入折扣碼的總數限制
9. 點擊「新增」

{% hint style="info" %}
折扣碼的描述僅用於辨識，不會被玩家看到
{% endhint %}

{% hint style="warning" %}
1. 因為金流平台若為「一元」不會開發票，因此折扣後金額最少為「兩元」。折後金額元以下的商品將無法設定
2. 商品類型有限制：付款類型為「線上金流付款」＋受眾標籤為「toC」的訂閱制商品才可設定使用折扣碼（若是線下虛擬付款的商品就無法使用折扣碼）
{% endhint %}

### 如何編輯一次性折扣碼？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 優惠折扣碼設定 -> 一次性折扣碼</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-14 16.20.11 2.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊現有折扣碼的「編輯」
2. 編輯描述
3. 編輯適用產品
4. 編輯優惠的結束時間
5. 新增折扣碼數量
6. 點擊「更新」
