# 縣市合作案用戶授權查詢

### 如何使用縣市合作案用戶授權查詢？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 縣市合作案用戶授權查詢</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-15 10.10.06.png" alt="" width="563"><figcaption></figcaption></figure>

輸入欲查詢用戶的 UID，即可

1. 查看帳號資訊
2. 查看教育雲或是澔學的連接狀態
3. 第三方授權的資料更新時間
4. 過去 n 筆登入的授權資訊
