# Banner 輪播管理

### 如何新增或編輯素養學習 APP 的 Banner？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 素養 APP 素養新知管理 -> Banner 輪播管理</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-14 14.02.03 (2).png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增輪播 Banner」或現有 Banner 的「編輯」
2. 輸入輪播 Banner 名稱
3. 選擇發佈時間、結束時間
4. 上傳 Banner 圖檔，並選擇是否要導入圖片中的超連結

可進一步調動 Banner 的顯示順序，點擊顯示順序欄位的座標 icon，並拖曳至指定排序位置，即可變更排序。

{% hint style="info" %}
1. 輪播 Banner 名稱僅用於辨識，不會被玩家看到
2. 輪播僅會出現前 3 則發佈中的 Banner
{% endhint %}

{% hint style="warning" %}
有上傳 Banner 圖檔，才能夠進一步設置 Banner 照片的超連結
{% endhint %}
