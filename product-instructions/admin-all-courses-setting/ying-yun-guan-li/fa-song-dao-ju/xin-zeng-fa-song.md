# 新增發送

{% hint style="warning" %}
1. 一次發送太多道具可能會破壞遊戲平衡，發送前請先跟遊戲設計師確認！
2. 有些道具因合約不可發送，特殊道具請先跟 BD 及 PM 確認！
{% endhint %}

### 「新增發送」與 「生成 PaCode」的差異為何？

新增發送為系統直接發送道具到玩家的背包內，玩家不需要額外做「領取」的動作。生成 PaCode 則需要另外由營運同仁生成 PaCode 並額外發送給指定玩家，而玩家也需要另外於遊戲介面做領取。

### **如何發送道具 ？**

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 發送道具 -> 新增發送</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-11 14.35.01.png" alt="" width="563"><figcaption></figcaption></figure>

1. 輸入活動名稱
2. 輸入欲發送道具名稱或是直接選擇欲發送的道具，並選擇數量
3. 選擇發送世界
4. 選擇發送對象：所有已加入指定世界的玩家或指定玩家[^1]
5. 輸入發送給玩家的訊息（選填，可使用 HTML 編輯器）
6. 選擇發送時間：立即發送或預約發送（排程）
7. 點擊「下一步」並確認發送道具的資訊
8. 輸入預計發送的使用者數量，並點擊「發送」

{% hint style="info" %}
1) 活動名稱僅用於查詢發送紀錄，不會被玩家看到
2) 一次最多只能發送 30 種道具，每種道具數量上限 100 個
3) 發送對象的人數限制為 99999 人
4) 若在選擇發送對象時選擇「所有已加入指定世界的玩家」，則會發送給「課程擁有者」、「競賽世界觀賽者」以外的玩家
5) 發送的訊息會出現在玩家的個人訊息中
{% endhint %}

{% hint style="warning" %}
發送道具時，請勿發送到競賽之盾世界或是奪寶信號世界
{% endhint %}

[^1]: 帳號 or email or UID
