# 發送排程

### 如何確認尚未發送或發送中道具的活動？

<mark style="background-color:purple;">檢視位置：管理者後台 -> 營運管理 -> 發送道具 -> 發送排程</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-11 15.02.21.png" alt="" width="563"><figcaption></figcaption></figure>

* 可查看尚未發送或發送中道具的活動，並查看其預定發送時間、建立時間、發送者 Email
* 點擊「詳細內容」，查看該次發送道具、發送世界、發送對象[^1]、發送使用者數、發送訊息
* 點擊「取消」，取消該次發送道具排程

{% hint style="warning" %}
取消發送排程對系統負擔很大，請盡量避免使用
{% endhint %}

[^1]: 依新增發送時選擇的發送對象格式顯示：

    1. 發送對象帳號
    2. 發送對象 email
    3. 發送對象 UID
