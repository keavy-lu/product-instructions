# 首頁輪播圖檔管理頁

### 首頁輪播圖檔的內容會出現在何處？

首頁輪播圖檔的內容會出現在首頁中，最多可呈現出八則圖檔，會自動循環播放，每則秒數為三秒一則。

### 如何新增或編輯輪播圖檔？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 首頁新聞管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-09 18.38.36.png" alt="" width="563"><figcaption></figcaption></figure>

1. 選擇環境：Global、馬來西亞、香港
2. 點擊「新增輪播圖檔」或現有輪播圖檔的「編輯」
3. 輸入廣告名稱
4. 選擇發佈時間、結束時間
5. 上傳網頁版與手機版的圖檔，並選擇是否要導入圖片中的超連結

可進一步調動輪播圖檔的顯示順序，點擊顯示順序欄位的座標 icon，並拖曳至指定排序位置，即可變更排序；如果需要讓輪播圖檔提早關閉顯示，可以點擊「提前結束」

{% hint style="warning" %}
有選擇贊助商照片，才能夠進一步設置贊助商照片的超連結
{% endhint %}
