# 世界任務管理

### 世界任務會出現在何處？

世界任務會出現在遊戲介面中的任務書裡，並呈現出該任務的任務名稱、任務描述、任務開始與結束時間、獎勵。

### 如何新增世界任務？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 世界任務管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-10 11.08.34.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增世界任務」
2. 輸入創建任務時所使用的帳號，並選擇欲派發成世界任務的「任務」
3. 是否開放教師在統計分析的功能中查看班級作答數據
4. 指派任務至課程中

{% hint style="success" %}
在派發世界任務時，請確認已經[創建完「任務」](../../teacher-console/zuo-ye-guan-li/ren-wu/#ru-he-zhi-pai-ren-wu-gei-xue-sheng)（已指定完成時間並選擇完成獎勵）
{% endhint %}

{% hint style="info" %}
1. 若某任務已經以其他任務型式派發至該世界的某一班級中，且Ａ學生「已完成」該任務，則後來再將該任務派發成世界任務時，此任務並不會出現在Ａ學生的世界任務標籤頁中
2. 若某任務已經以其他任務型式派發至該世界的某一班級中，且Ａ學生「尚未完成」該任務，則後來再將該任務派發成世界任務時，此任務會出現在世界任務標籤頁中，而非一般任務中
{% endhint %}

### 如何調整教師查看作答數據權限、指派世界任務的世界？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 世界任務管理 -> 未結束 or 已結束分頁 -> 現有世界任務的編輯</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-10 11.40.34.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊現有世界任務的「編輯」
2. 調整是否開放教師在統計分析的功能中查看班級作答數據
3. 調整指派任務的課程

### 如何編輯世界任務的題目來源？

系統目前不開放編輯世界任務，玩家會以當初派發任務的作業題目來源進行答題，就算編輯原作業的題目來源，也不影響玩家答題的題目範圍。

### 如何提早結束世界任務？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 世界任務管理 -> 未結束分頁 -> 現有世界任務的提前結束</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-10 11.44.03.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊現有世界任務的「提前結束」
2. 閲讀完警示語後點擊「確認」

{% hint style="info" %}
提前結束將導致此任務從所有課程世界的任務書消失，且此任務在世界任務後台會跑到「已結束」的分頁
{% endhint %}

### 如何查看世界任務紀錄？

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 世界任務管理 -> 未結束 or 已結束分頁 -> 數據</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-10 11.48.49.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊現有世界任務的「數據」
2. 可查看到各世界與全部派發世界的作答狀況：總完成帳號數、總開始帳號數、平均正確率、平均完成度、完成帳號數佔比
3. 可以進一步下載 Excel 檔案
