# 課程管理

### 名詞解釋

1. 公開世界：於 PaGamO 首頁能夠看到該課程世界，而使用者們能夠直接進入該課程
2. 顯示課程：該課程能夠在首頁的已加入課程，以及切換課程選單裡顯示
3. 學生上限：為課程世界的總玩家數量
4. 到期日：該課程世界的到期時間

{% hint style="info" %}
若設定該世界為公開世界，則會鎖定顯示課程的設定
{% endhint %}

### 如何編輯課程世界的資訊？

<mark style="background-color:purple;">設定位置：管理者後台 -> 系統設定 -> 課程管理 -> 編輯</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-09 13.52.24 2.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「編輯」後，進而編輯該課程世界的課程名稱、分類名稱、語言設定、語言提示彈窗設定、到期日、學生上限、公開課程、顯示課程、重複答題、課程簡介與圖示，並儲存編輯內容。

### 如何創建新的課程世界？

<mark style="background-color:purple;">設定位置：管理者後台 -> 系統設定 -> 課程管理 -> 建立新課程</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-09 13.52.24.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「建立新課程」後，進而設定該課程世界的課程名稱、分類名稱、語言設定、語言提示彈窗設定、到期日、學生上限、公開課程、顯示課程、重複答題、課程簡介與圖示，並儲存編輯內容。

{% hint style="info" %}
1. 學校名稱可不設定
2. 如分類名稱沒有該課程世界所屬的分類，可以參考[此連結](ke-cheng-fen-lei-guan-li.md)
{% endhint %}

{% hint style="info" %}
* 分類名稱：請選擇這個課程世界所屬的分類
* 語言設定：該課程世界開放給玩家們切換的不同語系
* 語言提示彈窗設定：當開啟開關，使用者第一次進入此課程世界時，會出現語言提示彈窗，告知使用者目前所選語言
* 重複答題：開放該課程世界的玩家能夠重複回答相同題目，若課程世界選擇關閉「重複答題」的功能後，會有以下影響：
  * 如果用戶答錯題目，可以再次答題，直到答對；如果用戶在一開始就答對題目，就不能再重複答這一題
  * 如果用戶選定的範圍已經都答對了，就會跳出重新選擇答題範圍的提示視窗
  * 如果教師剛好出題到玩家答對的題目，玩家還是可以答題
* 課程簡介：會顯示於選定世界後的選擇身份頁面
* 圖示：會顯示於已加入課程欄位、課程列表中
{% endhint %}
