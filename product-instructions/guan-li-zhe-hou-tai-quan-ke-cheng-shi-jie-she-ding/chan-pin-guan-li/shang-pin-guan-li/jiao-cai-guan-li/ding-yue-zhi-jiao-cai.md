# 訂閱制教材

### 如何查詢或複製目前已建立的訂閱制教材？

<mark style="background-color:purple;">設定位置：管理者後台 -> 產品管理 -> 商品管理 -> 教材管理 -> 訂閱制教材</mark>

<figure><img src="../../../../../.gitbook/assets/截圖 2023-08-15 11.35.29.png" alt="" width="563"><figcaption></figcaption></figure>

可直接輸入欲搜尋商品的商品代碼或商品名稱進行精準搜尋，或是使用商品類別、商品類型、受眾標籤進行篩選，便可得知該商品的資訊：商品代碼、商品名稱、首次扣款價、是否自動扣款、扣款週期、可作答的課程世界、受眾標籤與商品標籤。

如需要查看該商品的更多資訊，可以於點擊「更多操作」後點擊「查看商品細節」。

如需要複製商品教材的商品資訊設置，可點擊操作欄位的「複製 icon」。

如有設置內容相似的訂閱制教材，可以於點擊現有的訂閱制商品旁的「更多操作」後點擊「查複製商品」，進一步編輯商品設定與商品價格

{% hint style="info" %}
1. 商品生效日是跟金流相關
2. 教材生效日是跟教材相關，教材生效日起，就可以拿到教材對應的任務
{% endhint %}

{% hint style="info" %}
1. 如果購買時間「早於」商品生效日 -> 預購\
   預購商品的下次扣款時間 = 商品生效時間 + 扣款週期時長\
   e.g. Ａ預購商品的生效時間是`2023/9/23` ，用戶付錢的時間是 `2023/9/9` ，計算下次扣款時間是使用商品生效時間往後算，所以是用 `2023/9/23` + 90 天 = `2023/12/22`
2. 如果購買時間「晚於」商品生效日 -> 非預購商品\
   非預購商品的下次扣款時間 = 付錢時間. + 扣款週期時長\
   e.g. Ｂ非預購商品的生效時間是`2023-08-23` ，用戶付錢的時間是 `2023/9/9` ，計算下次扣款時間是使用付錢時間往後算，所以是用 `2023/9/9` + 90 天 = `2023/12/08`
{% endhint %}

### 如何編輯已建立的訂閱制教材？

<mark style="background-color:purple;">設定位置：管理者後台 -> 產品管理 -> 商品管理 -> 教材管理 -> 訂閱制教材</mark>

<figure><img src="../../../../../.gitbook/assets/截圖 2023-09-27 15.43.04.png" alt=""><figcaption></figcaption></figure>

點擊欲編輯的教材欄位的「更多操作」，而後點擊「查看商品細節」，即可檢視當前教材的資訊，滑動至頁面最下方並點擊「編輯」，即可調整以下資訊

* 商品名稱
* 派送訂閱制任務班級代碼
* 受眾標籤
* 商品標籤
* 可答題與領取任務獎勵的遊戲世界
* 預設答題世界
* 額外任務
* LINE OA 通知型訊息
* 銷售協議
* 商品頁的標題文字
* 販售開始與結束時間
* 商品生效時間
