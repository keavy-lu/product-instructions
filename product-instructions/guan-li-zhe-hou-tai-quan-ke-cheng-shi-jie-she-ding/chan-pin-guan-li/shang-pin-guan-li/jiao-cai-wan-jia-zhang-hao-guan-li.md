# 教材玩家帳號管理

### 如何查詢玩家綁定的教材？

<mark style="background-color:purple;">設定位置：管理者後台 -> 產品管理 -> 商品管理 -> 教材玩家帳號管理</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-15 10.57.53 (1).png" alt="" width="563"><figcaption></figcaption></figure>

可直接輸入玩家的帳號、電子信箱或是真實姓名進行模糊搜尋，或是點擊下拉式選單精準搜單筆或多筆資訊，兩種搜尋方式皆可搭配使用下拉式選單的商品代碼與參與課程的搜尋功能。送出後即可看到搜尋結果，並從「已綁定商品教材」的欄位中得知該帳號已綁定教材的商品代碼。

### 如何增加玩家綁定的教材或調整玩家綁定教材的時間？

<mark style="background-color:purple;">設定位置：管理者後台 -> 產品管理 -> 商品管理 -> 教材玩家帳號管理</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-15 10.57.53 2.png" alt="" width="563"><figcaption></figcaption></figure>

可直接輸入玩家的帳號、電子信箱或是真實姓名進行模糊搜尋，或是點擊下拉式選單精準搜單筆或多筆資訊，兩種搜尋方式皆可搭配使用下拉式選單的商品代碼與參與課程的搜尋功能。送出後即可看到搜尋結果。找到欲增加玩家綁定的教材或調整玩家綁定教材的時間的玩家帳號後，須點擊「編輯 icon」。

如需增加玩家綁定的教材，請點擊「新增商品教材」，選取商品教材，並選擇任務起派日與教材到期日。

如需調整玩家綁定教材的時間，請點擊「調整教材時間」選取想要調整的商品教材，並重新選擇任務起派日與教材到期日。

{% hint style="info" %}
如要批量增加綁定教材或是調整教材時間，請點擊欲執行的帳號左側的核取方塊，並點擊已選取數量旁的「新增商品教材」或「調整教材時間」
{% endhint %}

{% hint style="warning" %}
如果教材列表中並無自己欲新增的教材，請參考新增[訂閱制教材](jiao-cai-guan-li/ding-yue-zhi-jiao-cai.md#ru-he-xin-zeng-ding-yue-zhi-jiao-cai)或是[買斷制教材](jiao-cai-guan-li/mai-duan-zhi-jiao-cai.md#ru-he-xin-zeng-mai-duan-zhi-jiao-cai)的方法
{% endhint %}

### 如何「批量」增加玩家綁定的教材或調整玩家綁定教材的時間？

<mark style="background-color:purple;">設定位置：管理者後台 -> 產品管理 -> 商品管理 -> 教材玩家帳號管理</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-23 13.45.55.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊搜尋列旁的「展開 icon」
2. 將欲綁定教材的帳號的資訊（可用帳號、玩家真實姓名、信箱或 UID）分行貼至「精確搜尋多筆資訊」的搜尋欄位
3. 點擊進階搜尋視窗的「搜尋」，進行精準搜尋（僅完全相符者才符合搜尋條件）
4. 查看搜尋資訊是否與需求相符，如正確，則可全選並點擊「新增商品教材」，並選定商品教材、任務起派日、教材到期日
5. 點擊「確認綁定」，即可儲存該次調整
