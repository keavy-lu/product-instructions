# 個人訊息

{% hint style="info" %}
1. 發送訊息給指定使用者在指定世界中的遊戲角色，若使用者有加入訊息設定的發送世界，便可以收到訊息
2. 發送的訊息將出現在遊戲內「訊息管理」中的「個人訊息」
{% endhint %}

### 如何查看或編輯已新增的個人訊息？ <a href="#ru-he-cha-kan-huo-bian-ji-yi-xin-zeng-de-ge-ren-xun-xi" id="ru-he-cha-kan-huo-bian-ji-yi-xin-zeng-de-ge-ren-xun-xi"></a>

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 訊息管理 -> 個人訊息</mark>

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEcP74yMknDNTcCvarlNh%2Fuploads%2Fwl0STD5qshc8VU8pHsQI%2F%E6%88%AA%E5%9C%96%202023-12-06%20%E4%B8%8A%E5%8D%8810.25.58.png?alt=media&#x26;token=72fe930a-ff62-41c9-8904-8ccea05cfbff" alt="" width="563"><figcaption></figcaption></figure>

1. 可透過訊息標題**模糊搜尋**特定個人訊息
2. 可於介面中查看訊息的訊息標題、發送時間、發送者帳號
3. 如需要進一步查看或編輯個人訊息的內容，須點擊「查看」
   * 可查看訊息標題、訊息內容、發送對象[^1]、發送世界、發送時間、發送者帳號
   * 可點擊「編輯」更改訊息標題、訊息內容、發送時間​

### 如何新增個人訊息？ <a href="#ru-he-xin-zeng-ge-ren-xun-xi" id="ru-he-xin-zeng-ge-ren-xun-xi"></a>

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 訊息管理 -> 個人訊息</mark>

<figure><img src="https://files.gitbook.com/v0/b/gitbook-x-prod.appspot.com/o/spaces%2FEcP74yMknDNTcCvarlNh%2Fuploads%2FiXyQ61TRva9Qrv7ahVzg%2F%E6%88%AA%E5%9C%96%202023-12-06%20%E4%B8%8A%E5%8D%8810.24.12.png?alt=media&#x26;token=9d0e748a-94b0-4153-bae3-2c4a87e39cb5" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增個人訊息」
2. 輸入訊息標題
3. 輸入訊息內容（在編輯時可使用 HTML 編輯器）
4. 選擇發送對象格式，分行貼入資料
5. 選擇發送世界
6. 設定發送時間（未設定則會馬上發送）

[^1]: 依新增訊息時選擇的發送對象格式顯示：

    1. 發送對象帳號
    2. 發送對象 email
    3. 發送對象 UID
