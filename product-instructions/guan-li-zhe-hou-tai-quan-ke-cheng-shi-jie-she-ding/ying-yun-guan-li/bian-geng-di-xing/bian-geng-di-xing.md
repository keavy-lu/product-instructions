# 變更地形

### 如何變更地形？

<mark style="background-color:purple;">檢視位置：管理者後台 -> 營運管理 -> 變更地形 -> 變更地形</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-11 15.33.52.png" alt="" width="563"><figcaption></figcaption></figure>

1. 輸入活動名稱
2. 選擇要變更地形的世界
3. 挑選要變更的位置，並選擇變更成的地形、設定初始血量
   * 隨機指定野地（海洋）
   * 指定PaGamO帳號的所有土地
   * 指定特定地形來變更
   * 透過 CSV 檔指定座標
4. 點擊「下一步」並確認變更地形的資訊
5. 輸入變更地形所屬的課程世界名稱與變更地形的土地名稱，並點擊「確認變更」

{% hint style="info" %}
1) 活動名稱僅用於查詢發送紀錄
2) 一次變更地形僅能變更 10000 個土地
{% endhint %}

{% hint style="warning" %}
變更地形不可復原，請謹慎使用
{% endhint %}
