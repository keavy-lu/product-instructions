# 使用者分群管理

{% hint style="info" %}
使用者分群的使用情境

1. 官方競賽之盾的參賽設定
2. 彈窗消息管理
{% endhint %}

### 如何新增或編輯使用者分群管理

<mark style="background-color:purple;">設定位置：管理者後台 -> 營運管理 -> 使用者分群管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-14 17.32.26.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「新增使用者分群條件」或是現有使用者分群的「編輯」
2. 輸入代碼
3. 輸入名稱（供內部人員辨識所用）
4. 輸入分群描述（目前只有一個選項並預設選取「訂閱中的商品代碼」）
5. 選擇判別條件
   * 納入：勾選要納入此分群的商品代碼
   * 排除：勾選要排除此分群的商品代碼
6. 點擊「新增」或「更新」

{% hint style="info" %}
1) 如果用「排除」時，連「沒有任何商品代碼的人」也會被包含在這個分群中
2) 分群如果正在使用，則無法進行刪除
{% endhint %}
