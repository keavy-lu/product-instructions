# 報名者管理

### 如何查看已報名的參賽者？

<mark style="background-color:purple;">設定 / 檢視位置：競賽管理介面 -> 當前世界 -> 報名者管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-30 12.21.52.png" alt="" width="563"><figcaption></figcaption></figure>

進入報名者管理後，即可

1. 查看目前報名的參賽者人數
2. 依照真實姓名、暱稱、聯絡信箱、學校與隊名（若為單人賽，則沒有此搜尋條件）進行模糊搜尋
3. 點擊「下載回應」，下載參賽者資訊的 Excel 檔案
4. 查看某一參賽者的真實姓名、暱稱、聯絡信箱、學校與隊名、個人排名、個人積分
5. 點擊某一參賽者的「報名資訊」
   * 查看其驗證信箱的狀態、報名來源、驗證信箱時的時間戳記
   * 查看帳號 UID、暱稱、真實姓名、帳號、年齡
   * 查看聯絡信箱、縣市、城市、學校、年級、班級、報名來源
   * 查看隊名
6. 點擊某一參賽者的「戰績」，查看個人排名、個人積分、土地數、答題數、答題次數 (含重複題）、答對數、答對次數 (含重複題）、一次答對數、正確率、能量值、背包內道具
7. 點擊某一參賽者的「設定」
   * 輸入帳號或是電子信箱更改帳號（作為遞補隊員的功能）
   * 寄送驗證信
   * 進行重置，用戶維持報名成功狀態，但需要重新選擇角色與重生土地

{% hint style="info" %}
戰績數據計算方式

1. 答對數：曾經答對過的題目數量（閱讀題組無論子題數量，都只算一題）
2. 答題數：曾經作答過的題目數量（閱讀題組無論子題數量，都只算一題）
3. 答題次數（含重複題）：計算用戶答題次數，無論作答是否正確都列入計算（閱讀題組無論子題數量，都只算一題）
4. 答對次數（含重複題）：計算用戶答題次數，只計算答對（閱讀題組無論子題數量，都只算一題）
5. 第一次答對數：計算用戶答題次數，只計算答對，若該筆作答記錄為閱讀題組，會計算子題答對的數量
6. 正確率：計算用戶答對題數與總題數的比例，會另外把閱讀題組子題拆開來，例如: 用戶答對了選擇題 A，閱讀題組內一對一錯 這樣的正確率會是 2 / 3 => 66 or 67 %
{% endhint %}

### 如何讓已經進入競賽的玩家交換隊伍？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 當前世界 -> 報名者管理 -> 設定 -> 換帳號</mark>

1. 找到要交換的二個 user：Ａ、Ｂ
2. 點擊 User 帳號欄右方操作欄位的「設定」，而後點擊「換帳號」
3. 先將一個不在比賽世界的帳號跟他們做交換：
   * 「Ａ」換成「空閒帳號１」
   * 「Ｂ」換成「空閒帳號２」
4. 再置換一次，將Ａ、Ｂ的所在的隊伍對調：
   * 「空閒帳號１」換成「Ａ」
   * 「空閒帳號２」換成「Ｂ」
