# 黑名單管理

### 如何管理電競賽的黑名單？

<mark style="background-color:purple;">設定位置：競賽管理介面 -> 黑名單管理</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-30 12.31.15.png" alt="" width="563"><figcaption></figcaption></figure>

1. 可透過 UID、帳號、課程代碼，精準搜尋特定的使用者、課程世界
2. 可於列表中查看使用者的 UID、帳號、真實姓名、禁賽時間、禁賽狀態
3. 可點擊「新增名單」，選擇要禁賽的使用者
   1. 輸入使用者的帳號 / Email / UID
   2. 選擇使用者違規的賽場
   3. 填入違規事宜
   4. 選擇禁賽時間段
4. 可點擊「查看」、「歷史紀錄」查看過去禁賽緣由、禁賽時間等紀錄，查看使用者過去的禁賽狀況
5. 可點擊「解禁」，取消該使用者的禁賽狀態

{% hint style="info" %}
1) 此處的黑名單為全 PaGamO 電競世界共用
2) 禁賽設定無法進一步編輯，需要取消當次禁用，再次進行禁賽設定
{% endhint %}
