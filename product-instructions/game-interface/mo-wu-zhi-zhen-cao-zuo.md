# 魔物指針操作

### 如何加入魔王戰役？

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 魔物指針</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-13 11.34.58.png" alt="" width="563"><figcaption></figcaption></figure>

進入魔物指針後，可以看到當前開放的魔王戰役，倘若佈告欄圖示上並無任何字樣標示，代表其為官方魔王；而任務魔王或是強襲魔王的佈告欄圖示皆有文字提示。

點選任一魔王戰役後，會進入魔王的簡短介紹頁，倘若確定參加戰役，即可點擊右下角的「參加戰役」。

### 如何於魔幣商店購買商品？

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 魔物指針 -> 魔幣商店</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-28 15.58.05.png" alt="" width="563"><figcaption></figcaption></figure>

點擊任一欲購買的道具並選擇購買數量後，點擊「送出」即可完成購買
