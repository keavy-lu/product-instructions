# 好友

### 如何新增好友？

<mark style="background-color:purple;">設定位置：遊戲介面 -> 個人頭像 -> 好友 -> 新增</mark>

點擊左上角的「新增」，並輸入暱稱、帳號或是 ID，進行搜尋，即可查詢到符合搜尋條件的玩家，可進一步點擊「邀請」，要求新增為朋友。

{% hint style="info" %}
好友數量上限會隨等級上升而增加，最開始的上限為 10 人，每增加十級能夠提升 10 個數量上限。如：11\~20 等的好友數量上限為 20 人；21\~30 等的好友數量上限為 30 人，以此類推。81 等以上時，可達 90 人
{% endhint %}

### 如何查看好友的個人資訊？

<mark style="background-color:purple;">設定位置：遊戲介面 -> 個人頭像 -> 好友 -> 好友列表</mark>

點擊左上角的「好友」，並點選列表中的任一好友，即可查看好友的角色檔案、遊戲成就，可進一步：

1. 傳訊：傳送訊息至好友的個人訊息中
2. 拜訪：查看好友的土地
3. 刪除：將玩家從好友列表移除。

### 如何收送能量？

<mark style="background-color:purple;">設定位置：遊戲介面 -> 個人頭像 -> 好友 -> 好友列表</mark>

點擊好友列表右方的「贈送能量」即可發送三點能量給好友；或是點擊「領取能量」即可獲得好友發送的三點能量。如需要一次性地發送能量、獲得好友發送的能量，可點擊右上方的「一鍵收送能量」。

{% hint style="info" %}
1. 每日可發送能量的好友數量與好友數量的上限相同，最開始的上限為 10 人，滿級時，上限可達 90 人
2. 在領取好友贈送的能量時，如果點選領取的能量超過上限，多餘的會保存到下一次領取
3. 一天內好友贈送的能量僅能於當日領取，無法累積到隔日繼續使用。建議玩家先用好友贈送的能量，用完了還不夠再用能量道具
{% endhint %}
