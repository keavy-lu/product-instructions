# 小地圖

### 如何在小地圖中利用鼠標與箭頭移動至他處？

<mark style="background-color:purple;">設定位置：遊戲介面 -> 個人頭像 -> 小地圖</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-10-27 14.12.14.png" alt="" width="563"><figcaption></figcaption></figure>

使用鼠標點擊小地圖中的任一地點，大地圖就會跳轉至該地點；使用上下左右的箭頭，小地圖與大地圖也會跟著移動。

### 如何在小地圖的座標面板中輸入位置快速移動至他處？

<mark style="background-color:purple;">設定位置：遊戲介面 -> 個人頭像 -> 小地圖 -> 座標面板</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-10-27 14.12.14拷貝.png" alt="" width="563"><figcaption></figcaption></figure>

於小地圖的座標面板中輸入位置後，點擊搜尋 icon，即可跳轉至指定地點。

{% hint style="info" %}
若沒有出現小地圖的座標面板，原因為該課程世界並不開放使用該功能，詳情可查看[該連結](../../admin-courses/game-setting/shi-jie-she-ding.md#ru-he-she-zhi-you-hu-shi-jie-de-xiao-di-tu-de-zuo-biao-shu-zhi-de-kai-fang-zhuang-kuang)
{% endhint %}

### 如何更換小地圖檢視方式：依土地顯示/依玩家顯示？

<mark style="background-color:purple;">設定位置：遊戲介面 -> 個人頭像 -> 小地圖</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-10-27 14.12.14拷貝2.png" alt="" width="563"><figcaption></figcaption></figure>

小地圖的土地 icon 與領土 icon 為當前土地的顯示方式，各別代表其為「依土地顯示」、「依玩家顯示」的狀態。

欲切換小地圖成「依玩家顯示」的顯示方式，需要點擊小地圖中的土地 icon；欲切換小地圖成「依土地顯示」的顯示方式，需要點擊小地圖中的領土 icon。

### 如何回到自己領土？

<mark style="background-color:purple;">設定位置：遊戲介面 -> 個人頭像 -> 回到自己的領土</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-10-27 14.12.14拷貝3.png" alt="" width="563"><figcaption></figcaption></figure>

點擊人物頭像旁的座標 icon，即可回到玩家自己的領土。同時，小地圖與大地圖皆會切換至玩家領土的所在地。

回到自己領土規則：點擊回到自己領土後，不會特別指定回到那一塊土地，主要依照系統指派為主，所以每次可能都會不一樣。
