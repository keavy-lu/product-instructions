# 帳號資訊

### 如何查看 GC 的角色資訊？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 角色檔案</mark>

玩家可查看帳號的 User ID、暱稱、目前使用的角色、該角色的屬性加成與各項遊玩數據，包含金錢、不重復的答對題數、持有的領土數量、等級與經驗值。

### 如何查看 GC 的戰績排名

<mark style="background-color:purple;">檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 戰績排名</mark>

玩家能以答題數量、領土數量、參與戰役數量作為指標，並以該世界、或單一班級作為篩選範圍了解用戶本身排名與全體用戶的排名（顯示前五十名）。

{% hint style="info" %}
排名資訊排程每 15 分鐘更新一次\
（國小、國中世界於每小時的 05 分及 35 分更新排名，但 12 時與 18 時 05 分及 35 分不算）
{% endhint %}

### 如何查看 GC 過去的答題紀錄與題目內容？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 題目列表</mark>

玩家可查看過去答對與答錯且未訂正與答錯但已訂正的題目。

### 如何重新回答 GC 過去所答過的題目？

<mark style="background-color:purple;">答題位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 題目列表</mark>

玩家可以點選答錯的題目再次答題（訂正），若訂正成功，該題會轉移至訂正列表​。

### 如何查看可於自選章節中答題的章節範圍？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 已購項目</mark>

玩家可查看目前可以於自選章節中答題的章節範圍。

### 如何購買遊戲方案或是查看帳號過去購買的遊戲方案？

<mark style="background-color:purple;">購買 / 檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 已購項目</mark>

在部分世界中，玩家可以進一步購買不同版的本的遊戲方案。過去所購買的遊戲方案資訊也會呈現在已購項目中 e.g. 購買項目、到期日等。

### 如何查看答題的統計資訊？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 學習分析</mark>

學習分析中有四項不同分析可供玩家參考

1. 班級統計：選定班級與題目集（作業）後可以檢視答題正確率與完成度​
2. 自選章節統計：選定自選章節裡的科目與與冊次後（僅會出現玩家已經答題過的科目範圍）可以檢視該冊次下個章節的答題正確率與完成度​
3. 科目統計：以回答過的題目做為統計數據來源，檢視不同科目下的答題狀況與正確率
4. 時間區間統計：了解從過去到現在於不同科目下的答題歷程

### 如何進入學習中心？

<mark style="background-color:purple;">跳轉位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 學習中心</mark>

點擊後，玩家可以跳轉至素養學習中心

{% hint style="info" %}
玩家有訂閱素養教材，才會出現此學習中心的跳轉入口
{% endhint %}

{% hint style="warning" %}
若該玩家有訂閱素養教材，卻沒有出現該入口，請參考[此<mark style="color:blue;">連結</mark>](../../admin-courses/game-setting/xue-xi-zhong-xin-xiang-guan-she-ding.md#ru-he-da-kai-shou-ye-de-xue-xi-zhong-xin-ru-kou)
{% endhint %}

### 如何查看已經搜集到的怪獸？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 怪獸列表</mark>

玩家可查看目前已經搜集到的怪獸。

### 如何查看已經達成的遊戲成就？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> 個人頭像 -> 帳號資訊 -> 遊戲成就</mark>

玩家可查看目前已經達成的遊戲成就。
