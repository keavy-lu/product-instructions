# 背包

### 如何使用道具？

<mark style="background-color:purple;">使用位置：遊戲介面 -> 個人頭像 -> 背包</mark>

背包介面下方欄位為目前持有的遊戲道具，點擊使用後即可生效

* 怪獸：點擊使用後並放置於某一土地。如需要將該怪獸收回背包，須點擊怪獸所在的土地，點選怪獸頭像並點擊「收回」
* 防護罩：點擊使用後、放置於某一土地並確認即可生成該防護罩。不能收回背包
* 其他功能道具：點擊使用後並確認，即可獲得特定效果。不能收回背包
* 地形：點擊使用後、放置於任一符合升級條件的土地，即可進行升級。不能收回背包

### 如何購買道具？

<mark style="background-color:purple;">購買位置：遊戲介面 -> 個人頭像 -> 背包 -> 商店</mark>

背包介面上方欄位為商店所販售的道具，可以以指定價格購買該道具。​

### 如何販賣道具？

<mark style="background-color:purple;">販售位置：遊戲介面 -> 個人頭像 -> 背包</mark>

背包介面下方欄位為目前持有的遊戲道具，點擊販售、確認價格後即可賣出​。

### 如何贈送道具？

<mark style="background-color:purple;">贈送位置：遊戲介面 -> 個人頭像 -> 背包</mark>

背包介面下方欄位為目前持有的遊戲道具，點擊贈送、確認該次送出所需要額外花費的金錢後，並選擇贈送對象後即可送出（不限好友），送出後，系統會分別寄送個人訊息給收件人（寄件人、道具、數量、時間）和寄件人（收件人、道具、數量、時間）。

{% hint style="info" %}
1. 每個玩家送禮給同一對象 1 次 / 日，送禮對象最多 10 人 / 日
2. 送禮 1 次只能選 1 種道具，該道具數量上限為 5
3. 部分道具無法贈送給他人，這類道具點擊後鎖定送禮按鍵（變暗無法使用）
4. 送禮時，將會收取寄件人所選擇的送禮道具買價 1% 的金額作為運送費用
{% endhint %}
