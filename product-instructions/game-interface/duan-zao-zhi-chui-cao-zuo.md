# 鍛造之鎚操作

### 如何鍛造道具？

<mark style="background-color:purple;">鍛造位置：遊戲課程 -> 側邊欄（功能選單） -> 鍛造之鎚</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-31 10.23.23.png" alt="" width="563"><figcaption></figcaption></figure>

點擊欲鍛造的道具後，能夠查看該道具所需要的鍛造材料與加工費用，而每樣道具或地形所產生的鍛造經驗值並不相同，但越珍稀的道具或地形會取得更高的鍛造經驗值。隨著鍛造經驗值的累積，可以達到越高的鍛造等級（目前系統僅開放至等級五），並解鎖更多可鍛造的道具。
