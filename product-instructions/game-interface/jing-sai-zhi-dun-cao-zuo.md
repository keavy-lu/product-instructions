# 競賽之盾操作

### 如何加入競賽之盾？

#### 官方競賽之盾

<mark style="background-color:purple;">加入位置：遊戲課程 -> 側邊欄（功能選單） -> 競賽之盾 -> 官方（紅色大門）</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-13 11.47.29.png" alt="" width="563"><figcaption></figcaption></figure>

進入官方競賽之盾後，可以看到當前開放的官方競賽之盾，點選任一競賽的說明，可了解該場競賽的規則，包括競賽時間與積分計算方式。倘若確定參加競賽，即可點擊右方的「進場」。

{% hint style="info" %}
如果目前沒有任何開放的官方競賽，門上的字樣會呈現「官方競賽尚未開放」
{% endhint %}

* 如果該場競賽為大亂鬥（個人戰），玩家會直接進入賽場並開始答題
* 如果該場競賽為團體戰，玩家會需要另外選擇自己所屬的隊伍，才能夠進入賽場並開始答題

#### 課堂競賽之盾

<mark style="background-color:purple;">加入位置：遊戲課程 -> 側邊欄（功能選單） -> 競賽之盾 -> 課堂（藍色大門）</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-13 11.47.29 2.png" alt="" width="563"><figcaption></figcaption></figure>

進入課堂競賽之盾前會需要取得教師給予的競賽地圖號碼，才可以進入課堂的競賽之盾。

* 如果該場競賽為大亂鬥（個人戰），玩家會直接進入賽場並開始答題
* 如果該場競賽為團體戰，玩家會需要另外選擇自己所屬的隊伍，才能夠進入賽場並開始答題

{% hint style="info" %}
如果沒有教師所提供的競賽地圖號碼，就無法進入該課堂競賽之盾
{% endhint %}
