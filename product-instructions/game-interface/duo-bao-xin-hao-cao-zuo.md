# 奪寶信號操作

### 如何加入奪寶信號？

<mark style="background-color:purple;">加入位置：遊戲課程 -> 側邊欄（功能選單） -> 奪寶信號</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-31 10.16.55.png" alt="" width="563"><figcaption></figcaption></figure>

進入奪寶信號後，可以看到當前開放奪寶信號競賽，點選任一競賽的說明，可了解該場競賽的規則。

於進場前或是設定隊伍前，系統會先透過班級代碼辨識玩家是否持有該場奪寶信號的入場資格，如玩家不在該班級範圍內，亦可輸入進場密碼來入場。

除此之外，玩家也需另外確認競賽的競賽類別：

* 如果該場競賽為個人競賽，隊伍的按鈕處會反灰，無法進一步設定
* 如果該場競賽為分組競賽，進場前會需要另外設定隊伍的相關設定，請先點擊「隊伍」。若要開設新隊伍並邀請其他隊友進入該隊伍，可以點擊「建立隊伍」，輸入與其他隊伍名不重複且合法（4～16個中文或英文字母或數字）的隊名，即可取得隊伍代碼；若要進入某一隊伍，可以輸入隊長給予的隊伍代碼，並點擊「加入夥伴隊伍」，即可加入某一隊伍

玩家取得奪寶信號的入場資格，並設置完隊伍設定後（單人競賽不必設定），即可點擊右方的「進場」，進入該奪寶信號。

{% hint style="info" %}
1. 進入奪寶信號地圖後，如需回到原本的課程世界，可點擊奪寶信號地圖側邊欄的「暫離」
2. 如需再次進入奪寶信號地圖，只要再次從功能列中進入奪寶信號列表，選取該奪寶信號的「進場」，即可再度進入奪寶信號地圖
{% endhint %}
