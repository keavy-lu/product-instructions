# 任務書操作

### 如何接取任務？ <a href="#ban-ji-shi-shen-me" id="ban-ji-shi-shen-me"></a>

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 任務書</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-12 16.57.09.png" alt="" width="563"><figcaption></figcaption></figure>

點選任務書裡的任一任務，可以查看該任務所剩時長、任務目標與任務敘述，若確認接取該任務，可點擊「開始任務」並回到遊戲畫面開始攻擊或訓練土地。

{% hint style="info" %}
在搜尋素養任務時，可以使用教材篩選器搜尋特定類別的素養任務
{% endhint %}

{% hint style="info" %}
當玩家使用任務書內的題目進行答題，需要玩家將任務書內容全部答對後才算是完成任務
{% endhint %}

### 如何暫停任務？ <a href="#ban-ji-shi-shen-me" id="ban-ji-shi-shen-me"></a>

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 任務書</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-12 17.00.01 (1).png" alt="" width="563"><figcaption></figcaption></figure>

點選任務書裡的正在進行的任務，並點擊「暫停任務」，即可暫停使用該任務題庫作為回答的題目，可以另外接取其他任務，或是進入自選章節選擇自己欲練習的章節範圍。

### 如何重啟任務？ <a href="#ban-ji-shi-shen-me" id="ban-ji-shi-shen-me"></a>

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 任務書</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-14 13.49.35.png" alt="" width="563"><figcaption></figcaption></figure>

點選任務書裡之前暫停的任務，並點擊「重啟任務」，即可重新以該任務題庫作為回答的題目。

### 如何完成任務並領取任務獎勵？

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 任務書</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-10-23 15.40.05.png" alt="" width="563"><figcaption></figcaption></figure>

待玩家將任務內的所有題目完全答對時（答錯後補答並答對），進度才會達到 100%，屆時才能夠點擊「完成任務」並領取任務獎勵。
