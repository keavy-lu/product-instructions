# 身份切換

{% hint style="info" %}
僅同時具有多種身份的玩家，才能夠切換至不同的操作環境 e.g. 老師能夠切換至教師後台

若在該世界尚未取得其他權限，可以參考[此連結](shen-fen-qie-huan.md#ru-he-qie-huan-shen-fen)
{% endhint %}

### 如何切換身份？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 身分切換</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-07-31 10.44.08.png" alt="" width="563"><figcaption></figcaption></figure>

點擊上排導航欄中的身份切換後，會出現該帳號於該世界能夠取用的不同操作環境 e.g. 管理者介面、教師後台等，點選任一身份後，即可跳轉至該環境。
