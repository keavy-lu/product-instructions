# 訊息管理

### 如何查看公告訊息？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> Hader -> 訊息管理</mark>

點擊公告訊息的 Tab 後，即可查看系統發佈到單一課程世界的公告訊息。

### 如何查看班級訊息？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> Hader -> 訊息管理</mark>

點擊班級訊息的 Tab 後，即可查看班級教師所發佈的班級訊息。

### 如何查看個人訊息？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> Hader -> 訊息管理</mark>

點擊個人訊息的 Tab 後，即可查看系統發佈或是好友傳送的個人訊息。

### 如何查看帳號訊息？

<mark style="background-color:purple;">檢視位置：遊戲介面 -> Hader -> 訊息管理</mark>

點擊帳號訊息的 Tab 後，即可查看系統發佈的帳號訊息或是素養商品任務的獎勵訊息。

### 如何領取帳號訊息內道具？

<mark style="background-color:purple;">領取位置：遊戲介面 -> Hader -> 訊息管理</mark>

點擊帳號訊息的 Tab 後，可點擊附有獎勵的素養任務獎勵訊息，並領取該獎勵；另外也可以點擊頁面左下方的「全部領取」，領取所有可領取獎勵的帳號訊息中的獎勵。

{% hint style="warning" %}
若所在的課程世界沒有開放此獎勵道具，會出現下列反灰的領取按鈕
{% endhint %}

{% hint style="info" %}
如素養任務所在世界有開放至學習中心答題，素養任務可以在遊戲中進行，也可以在學習中心進行。而在學習中心完成任務當下，遊戲內的任務書就不會再出現這個任務，因此玩家無法在遊戲的任務書中領取獎勵，因此，系統這時會改以使用帳號訊息的功能寄送任務對應的道具，玩家可以在開放領取道具的任一世界中領取該獎勵。因為一個任務只能領取一次獎勵，因此在Ａ課程世界的帳號訊息裡領取過，該訊息消失同時，B、C、D 課程世界的同一封帳號訊息也會消失，無法重複領取
{% endhint %}
