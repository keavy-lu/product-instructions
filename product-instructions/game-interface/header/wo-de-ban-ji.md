# 我的班級

### 如何在遊戲介面加入班級？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 我的班級</mark>

點擊上排導航欄中的「我的班級後，可以看到目前已經加入的班級列表。依序填入老師給予的班級代碼和座號（依照教師設定決定是否需要填寫），並點擊右上方的「加入班級」，即可加入該[班級](/broken/pages/OO38s7w0SDrOcoDA8uAE#ban-ji-shi-shen-me)。
