# 遊戲設定

{% hint style="info" %}
<mark style="color:blue;">本文旨在教導玩家如何進行介面設定，並說明如何兌換 PaCode 所對應的道具或地形</mark>
{% endhint %}

### 如何調整顯示的物件？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 遊戲設定 -> 顯示設定</mark>

可藉由遊戲設定內的開關來調整是否要顯示旗子、防護罩、怪手等物件。

### 如何調整遊戲介面顯示語言？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 遊戲設定 -> 顯示設定</mark>

可藉由遊戲設定內的語言設定調整遊戲介面中的顯示語言。

### 如何調整遊戲答題介面的字體大小？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 遊戲設定 -> 顯示設定</mark>

可藉由遊戲設定內的字體設定調整遊戲答題介面中的字體大小。

### 如何調整背景音樂？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 遊戲設定 -> 聲音設定</mark>

可藉由遊戲設定內的開關來調整是否要開啟遊戲介面裡的背景音樂、答題時的背景音效。

### 如何重設密碼？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 遊戲設定 -> 密碼設定</mark>

輸入正確的原先密碼、輸入新密碼後即可重設密碼。

{% hint style="info" %}
教育雲帳號與澔學帳號無法重設密碼
{% endhint %}

### 如何啟用重生功能？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 遊戲設定 -> 其他設定</mark>

點擊「重生」後再次點擊「確認」後，即可啟用[重生功能](/broken/pages/NF7cSWijuzhQhjpBEiYV)。

### 如何啟用假期模式？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 遊戲設定 -> 其他設定</mark>

點擊「假期模式」後再次點擊「確認」後，即可啟用[假期模式](/broken/pages/ZmKOt1oQsAi9i6evJDZg)。

### 如何兌換 PaCode？

<mark style="background-color:purple;">兌換位置：遊戲介面 -> Hader -> 遊戲設定 -> PaCode</mark>

玩家若有從官方途徑（官方活動或是購買特定商品）獲得 [PaCode](/broken/pages/YncoGr98YbWdtwx3lx9p#pacode-shi-shen-me)，可至遊戲設定內輸入並兌換道具。
