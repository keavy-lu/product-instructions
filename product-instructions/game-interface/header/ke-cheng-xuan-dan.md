# 課程選單

### 如何在遊戲介面切換至不同課程世界？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 課程選單</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-10-24 15.48.57.png" alt="" width="563"><figcaption></figcaption></figure>

點擊當前所在的課程世界後，即可展開已加入的課程世界的選單，點擊其中一個課程世界，及可切換至該課程世界。
