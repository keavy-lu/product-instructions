# 自選章節操作

### 如何選擇自選章節的題目範圍？ <a href="#ban-ji-shi-shen-me" id="ban-ji-shi-shen-me"></a>

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄 -> 自選章節</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-12 17.23.51.png" alt="" width="563"><figcaption></figcaption></figure>

點選[自選章節](/broken/pages/yx5VqO31QAwHKLIpVegG#ban-ji-shi-shen-me)裡的任一出版社<mark style="color:blue;">、</mark>科目、學年度、冊次下的章節（可多選），被選定的題目範圍會呈現於已選章節的欄位內；若需要編輯目前使用的章節範圍，可以於已選章節的欄位點選章節右方的刪除按鈕。

除了第一次設定的範圍外，如需選擇其他的出版社<mark style="color:blue;">、</mark>科目、學年度、冊次作為出題範圍，可於選定完第一次的章節後點擊章節以外的其他 Tab ，重新選擇範圍並加入已選章節。

{% hint style="info" %}
1. 在選擇題目範圍時，可以一次性地選擇多個出版社、科目、學年度、冊次下的章節
2. 如果沒有暫停任務書裡正在進行的任務，選定自選章節的題目範圍，系統會直接暫停玩家本來進行中的任務，以自選章節作為題目來源
{% endhint %}
