# 交易市集操作

### 我要怎麼於市集販售道具？

<mark style="background-color:purple;">販售位置：遊戲課程 -> 側邊欄（功能選單） -> 交易市集 -> 寄賣</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-17 15.28.39.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點選欲販售的非灰階道具（成灰階的道具為不可販售的商品）
2. 輸入該次欲販賣的商品數量與單價
3. 勾選「確認上述資訊」並上架該商品

{% hint style="info" %}
1) 市集會收取售價的 1% 作為服務費
2) 買方支付總價 = 售價 + 市集服務費
3) 每一筆商品只能在只能市集販售五天，時間結束後商品會自動下架，市集所收取的服務費不予退回
4) 若選擇提早下架商品，集所收取的服務費不予退回
5) 已下架的商品仍會佔用寄賣頁的寄賣欄位，需要將商品收回背包，釋放欄位
{% endhint %}

{% hint style="warning" %}
1. 玩家等級達十等以上才可以進行寄賣，最多可寄售 10 筆商品
2. 玩家等級達五十等後，最多可寄售 20 筆商品
3. 部分商品不可交易，僅能選擇可交易的商品做販售
{% endhint %}

### 我要怎麼於市集購買道具？

<mark style="background-color:purple;">購買位置：遊戲課程 -> 側邊欄（功能選單） -> 交易市集 -> 市集</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-17 15.37.10.png" alt="" width="563"><figcaption></figcaption></figure>

1. 使用搜尋、篩選、排序功能找到自己想要購買的商品
2. 點選任一欲購買的道具並確認價格
3. 勾選「確認上述資訊」並購買該商品
