# 每日獎勵操作

### 如何補領當日的月份簽到獎勵？

<mark style="background-color:purple;">補領位置：遊戲課程 -> 側邊欄（功能選單） -> 每日獎勵</mark>

可於遊戲畫面的側邊欄或是功能選單中點選「每日獎勵」，再次進入小夥伴大富翁的的遊玩畫面進行補領。

### 如何補領當日的週次簽到獎勵？

<mark style="color:red;">唯有當日首次登入時才會跳出週次簽到視窗，無法進行補領</mark>

### 如何得知當月的的累積登入進度？

<mark style="background-color:purple;">補領位置：遊戲課程 -> 側邊欄（功能選單） -> 每日獎勵</mark>

可以點擊左下角的「查看累積登入」查看當前進度。
