# 遊戲介面操作

### 如何攻擊土地？ <a href="#jie-shao" id="jie-shao"></a>

玩家需要先選定答題的內容（可選擇任務或是自選章節），而後點擊與自己土地相鄰的野地或是他人的領土，按下「前進」，即可答題，若答對，目標的血量將下降，若下降至 0，即可攻佔該土地。另外，野地經玩家攻佔之後會隨機變成草原、荒沙和密林等不同地形。

{% hint style="info" %}
玩家無法在下列幾種情況攻擊土地：

* 受新手保護或是開啟假期模式的玩家的領土
* 受特殊防護罩（黃金、鑽石、天使防護罩）保護的土地或野地
* 與自己的領土不相連的海或土地
{% endhint %}

### 如何訓練土地？ <a href="#jie-shao" id="jie-shao"></a>

與攻擊土地相同，玩家需要先選定答題的內容（可選擇任務或是自選章節），而後點擊自己的土地，按下「訓練」，即可答題，若答對，目標的血量將上升 60，可以讓該地較不容易被敵人攻陷。

{% hint style="info" %}
玩家無法在下列幾種情況訓練土地：

* 開啟[省力模式](/broken/pages/zXaJNzs9W8ARYFoJByRE#sheng-li-mo-shi-shi-shen-me)
* 開啟[假期模式](/broken/pages/ZmKOt1oQsAi9i6evJDZg#jia-qi-mo-shi-shi-shen-me)且未結束
{% endhint %}

### 如何放置怪獸？

到背包中找尋想要放置的怪獸，按下「使用」，點擊欲放置該怪獸的土地，即可放置怪獸，一塊土地最多可放置三個怪獸。

{% hint style="info" %}
所有怪獸的防守力皆相同，其他玩家多答一題即可消滅土地上的一隻怪獸，建議玩家優先以[商店](https://pagamo.fandom.com/zh-tw/wiki/%E5%95%86%E5%BA%97)中廉價且方便取得的青翼鳥之卵、變色蜥之卵及貝魯獸之卵防禦土地
{% endhint %}

{% hint style="warning" %}
僅部分基礎地形能夠放置怪獸
{% endhint %}

### 如何回收怪獸至背包

點選有放置怪獸的領土、點擊怪獸圖示，最後點擊「收回」，即可將怪獸回收至背包中。

### 如何建造地形？

當訓練地形到第五級時，即可建造地形。到背包中找尋想要建造的地形道具，按下「建造」，若答對，即可將地形升級。可以將自己的地形收藏展示，同時增加血量。

### 如何查看土地資訊？

<figure><img src="../../.gitbook/assets/截圖 2023-10-24 16.37.30.png" alt="" width="563"><figcaption></figcaption></figure>

點擊自己或是他人的領土後，即可查看

1. 該土地的持有人
2. 該土地所屬的地形
3. 該土地上的怪獸
4. 該土地的等級
5. 該土地當前血量 / 該土地、等級下的最高血量

{% hint style="info" %}
玩家能夠直接從外觀上得知該土地是否有防護罩保護
{% endhint %}

{% hint style="info" %}
查看野地的土地資訊時，僅會出現野地地形、血量的資訊
{% endhint %}
