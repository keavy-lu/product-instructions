# 硬體與瀏覽器版本需求

### APP 硬體需求

#### Android

* 最低配備
  * 記憶體 : 2GB
  * 內存空間 : 300MB
  * Android 6.0
* 建議配備
  * 記憶體 : 2GB
  * Android 7.0以上（包含7）

#### IOS

* 最低配備
  * 裝置 : iPhone 6S、iPhone SE
  * 記憶體 : 2GB
  * iOS 13以上（包含13）
* 建議配備
  * 裝置 : iPhone 7 以上裝置（包含7）

### WEB 瀏覽器版本需求

* ie：停止支援
* Edge >= 16
* Chrome >= 57
* Opera >= 44
* Safari >= 11
* Firefox >= 52
