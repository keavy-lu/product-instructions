# 學科挑戰操作

### 如何進入學科挑戰？

<mark style="background-color:purple;">加入位置：遊戲課程 -> 側邊欄（功能選單） -> 學科挑戰</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-14 15.48.01.png" alt="" width="563"><figcaption></figcaption></figure>

進入學科挑戰後，可以看到不同學科的挑戰塔，也可以在介面的右下方看到目前持有的挑戰券、星星幣的數量，隨後可以依照自己所屬的程度進入不同的挑戰塔，並依照手指箭頭的關卡指示點選該關卡。

點選任一挑戰塔的關卡後，會進入關卡的資訊頁，包括該關卡目標，倘若確定挑戰該關卡，即可點擊下方的「開始挑戰」，花費一張挑戰券並開始答題。

### 如何購買額外的挑戰券？

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 學科挑戰 -> 前往購買（挑戰券）</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-07-14 15.58.22.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊自己欲購買的數量
2. 閱讀完購買提醒並同意銷售協議後，且願意購買，可點選「確定購買」
3. 填寫金流頁面的資訊並完成繳款
4. 購買成功！定單資訊將會寄送到綁定的信箱

{% hint style="warning" %}
尚未完成信箱綁定者，需要先綁定將 PaGamO 帳號綁定個人信箱，才能進行購買
{% endhint %}

### 如何於星星商店購買商品？

<mark style="background-color:purple;">設定位置：遊戲課程 -> 側邊欄（功能選單） -> 學科挑戰 -> 星星商店</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-28 16.00.16.png" alt="" width="563"><figcaption></figcaption></figure>

點擊任一欲購買的道具並選擇購買數量、點擊購買，並選擇該道具欲使用的課程世界，完成後，商品會直接進入該課程背包。
