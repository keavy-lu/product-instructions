# 任務魔王戰

### 如何開設任務魔王戰？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單 -> 新增任務</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-06-30 14.50.25.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="warning" %}
請確保

1. [<mark style="color:blue;">已創建作業</mark>](../zuo-ye-guan-li/zuo-ye.md#ru-he-jian-li-zuo-ye)
2. [已有所需魔王](../../game-main-function/mo-wu-zhi-zhen-she-zhi/she-zhi-guan-fang-mo-wang-zhan.md#ru-he-she-zhi-mo-wu-zhan-yi-de-mo-wang)
{% endhint %}

1. 請先選擇「魔王模式」作為任務的模式
2. 填入任務名稱
3. 選擇任務的開始時間與結束時間（時間僅限於 1～20 分鐘）
4. 選擇答案顯示設定（此設定值無意義，皆不會顯示正確答案與詳解）
5. 點選「魔王設定」，並完成以下設定
   * 選擇魔王難度（不同難度的魔王在後台為不同隻魔王，若無法選擇其他難度，代表後台並無設定該難度的魔王）
   * 填入參加人數（參加人數設定越高，魔王血量越高）
6. 點擊「下一步」，將任務指派給班級或小組（屆時就會扣除 QP）

{% hint style="info" %}
1) 開設魔王任務會耗費教師 2000 QP[^1]，用於給予學生參與魔王戰的獎勵
2) 不像是一般任務，學生答對任務內的題目並不會獲得任何 QP
3) 如有玩家已進入任務魔王戰，則使用到的作業或任務都無法刪除或修改（包含引用的作業）
{% endhint %}

[^1]: 題目點數，可以用來購買任務獎勵
