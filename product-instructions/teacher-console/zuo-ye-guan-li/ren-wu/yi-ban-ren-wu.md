# 一般任務

### 如何創建一般任務？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單 -> 新增任務</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-07-05 10.09.48.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="warning" %}
請確保已[<mark style="color:blue;">創建作業</mark>](../zuo-ye.md#ru-he-jian-li-zuo-ye)
{% endhint %}

1. 請先選擇「一般任務」作為任務的模式
2. 填入任務名稱
3. 填入任務概述（學生會在遊戲畫面上看到此段描述，非必填）
4. 上傳任務縮圖（僅有素養任務需要上傳任務縮圖，在「管理者後台 > 遊戲角色管理」開啟「主題式閱讀管理權限」後，即可看見此選項）
5. 選擇任務的開始日期與時間、結束日期與時間
6. 選擇學生完成任務時，是否能夠取得獎勵（會消耗相對應的 QP，非必填）
7. 選擇答案顯示設定
   * 答題後，可查看正確答案與詳解（適合用於不需重複答題的任務類型）
   * 答題後，若答錯，只能看到詳解；若答對，可看到正確答案與詳解（適合用於重複答題的任務類型，且學生可依照詳解的內容於下次作答選出正確答案）
   * 答題後，無法查看正確答案與詳解（適合用於重複答題的任務類型，學生需要額外思考去想出問題的正確答案）
8. 點擊「下一步」，將任務指派給班級或小組（屆時就會扣除 QP）

{% hint style="info" %}
1) 當學生完成任務時，可以領取一個獎勵。因此當你只有購買十個獎勵數量時，只有前十名完成任務的學生可以獲得獎勵
2) 每當學生答對一題一般任務內的題目，教師將會獲得 5QP（當學生按下完成任務時，會立即回饋給老師）
3) 任務獎勵設定後，若要變更，將不會退回之前設定獎勵所花費的 QP
{% endhint %}

### 如何編輯一般任務？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單 -> 欲編輯的一般任務</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-11-01 17.04.10.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊欲編輯的一般任務類型的任務卡片的「編輯」
2. 可進一步更改一下資訊
   * 任務名稱
   * 任務敘述
   * 任務縮圖
   * 任務開始時間 / 任務結束時間
   * 獎勵與獎勵數量
3. 不可更改以下資訊
   * 任務模式
   * 答案顯示設定
