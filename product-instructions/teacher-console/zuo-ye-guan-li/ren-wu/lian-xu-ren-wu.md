# 連續任務

### 如何創建連續任務？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 連續任務</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-04 14.24.52.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="warning" %}
請確保

1. 已[<mark style="color:blue;">創建作業</mark>](../zuo-ye.md#ru-he-jian-li-zuo-ye)
2. 已[開啟連續任務的功能](../../../admin-courses/game-setting/ren-wu-shu.md#ru-he-da-kai-lian-xu-ren-wu-de-gong-neng)
{% endhint %}

1. 命名該次連續任務的名稱
2. 點擊「新增連續任務」，從現有的作業清單中選擇任務
3. 選擇任務模式
   * 答題後，可查看正確答案與詳解\
     適合用於不需重複答題的任務類型
   * 答題後，若答錯，只能看到詳解；若答對，可看到正確答案與詳解\
     適合用於重複答題的任務類型，且學生可依照詳解的內容於下次作答選出正確答案
   * 答題後，無法查看正確答案與詳解\
     適合用於重複答題的任務類型，學生需要額外思考去想出問題的正確答案
4. 調整連續任務的出題順序
5. 編輯連續任務中的每一個任務的資訊：任務名稱、 任務描述、獎勵與任務開始、結束時間
6. 點擊「儲存」，將此次連續任務清單儲存成連續任務（屆時就會扣除 QP）

{% hint style="info" %}
連續任務屬於一般任務
{% endhint %}

### 如何指派連續任務？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 連續任務</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-08-04 14.33.06.png" alt="" width="563"><figcaption></figcaption></figure>

可直接勾選班級指派該連續任務，當成功指派後，連續任務中的第一個任務就會出現在學生的一般任務中，完成第一個任務後，才會接續出現後續的其他任務。

### 如何編輯連續任務？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 連續任務</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-11-01 17.24.36 1.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊欲編輯的測驗任務，進入總覽頁面
2. 可進一步更改一下資訊
   * 連續任務名稱
   * 連續任務內任務的
     * 任務名稱
     * 任務描述
     * 獎勵與獎勵數量
     * 任務開始時間 / 任務結束時間
3. 不可更改以下資訊
   * 答案顯示設定
