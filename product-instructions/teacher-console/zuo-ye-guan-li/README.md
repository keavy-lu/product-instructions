# 作業管理

{% hint style="info" %}
<mark style="color:blue;">作業管理為派發題目給練習的功能</mark>
{% endhint %}

作業管理中有兩大區塊：作業和任務，為教師派發題目給學生練習的必要工具，且兩大功能須配合使用、缺一不可：

1. 作業：確立題目來源、題數與答題方式，建立起「作業」
2. 任務：將作業打包成「任務」，以任務形式派發給欲指派任務的對象
