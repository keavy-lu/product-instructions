# 作業

### 如何建立作業？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 新增作業</mark>

1. 選擇題目來源、科目、學年度、冊次與作業章節作為題目來源
2. 選擇出題模式
   * 全自動：輸入總出題數 系統自動隨機出題
   * 半自動：先依難易挑選題數 系統自動隨機出題
   * 手動：手動勾選題目 完全按照您的意願
3. 選擇出題順序
   * 隨機出題：題目由系統隨機出現，每位學生答題順序都會不同
   * 順序出題：由教師設定出題順序，每位學生答題順序皆同
4. 調整題目數量
   1. 減少題目：點擊題目右邊的垃圾桶符號進行刪除
   2. 增加題目
      1. 自動增加方法：點擊右上角的「上一步」，再按一次「新增作業-全自動/半自動」，輸入希望增加的題目數量。
      2. 手動增加方法：直接點擊「題目來源」（下圖，左上位置）即可看到已選題庫範圍的所有題目，將希望增加的題目左側框框打勾即可。
5. 命名作業
   * 點擊右上角的下一步，為作業命名並按下「儲存」，便可成功建立作業。若不需將其指派為任務，直接離開頁面即可。

{% hint style="info" %}
可選擇繼續創建任務，請[依照需求創建不同任務類型](ren-wu/#bu-tong-ren-wu-de-ou-bie-shi-shen-me)
{% endhint %}

### 如何分享作業給其他教師？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單</mark>

1. 點選欲分享的作業旁的「分享」
2. 閱讀完 Tool Tips 的資訊後點擊「確認」即可分享該作業給其他教師

{% hint style="info" %}
作業若指派給競賽之盾使用，無法再分享給其他教師使用
{% endhint %}

{% hint style="warning" %}
如果無法於作業清單中看到分享作業的「分享」按鈕，原因是系統不開放教師使用這項功能，詳情參考[此連結](../../admin-courses/game-setting/jiao-shi-hou-tai.md#ru-he-da-kai-jiao-shi-hou-tai-de-fen-xiang-zuo-ye-de-gong-neng)
{% endhint %}

### 如何編輯作業的題目範圍與出題方式？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-11-01 15.39.36.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點選欲編輯的作業旁的選單
2. 於選單中點擊「編輯」，即可在既有題庫下重新選擇作業的題目範圍，並決定題目的出題方式
3. 點擊「完成」即可儲存編輯內容

{% hint style="warning" %}
如果該作業被指派為魔王模式或是強襲魔王模式，且有玩家已經進入魔王戰，則使用到的作業無法修改（包含引用的作業）
{% endhint %}

### 如何刪除作業？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單</mark>

1. 點選欲刪除的作業旁的選單
2. 於選單中點擊「刪除」，並於確認視窗點擊「刪除」，即可成功刪除

{% hint style="warning" %}
如果該作業被指派為魔王模式或是強襲魔王模式，且有玩家已經進入魔王戰，則使用到的作業無法刪除（包含引用的作業）
{% endhint %}
