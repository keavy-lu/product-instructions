# 教學資源

{% hint style="info" %}
教學資源<mark style="color:blue;">為教師自行出題的功能</mark>
{% endhint %}

教學管理中有兩項子功能，分別為：

1. 素養教學資源：可供管理學生清單
2. 分享資源：可檢視並引用其他教師所分享的任務，亦可檢視自身所分享的任務

{% hint style="info" %}
<mark style="color:blue;">如要關閉某任務的分享狀態，仍須至任務清單關閉</mark>
{% endhint %}
