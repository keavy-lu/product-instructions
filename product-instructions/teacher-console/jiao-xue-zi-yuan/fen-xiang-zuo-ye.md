# 分享作業

{% hint style="warning" %}
如果無法於教學資源中看到分享作業功能，原因是系統不開放教師使用這項功能，詳情參考[此連結](../../admin-courses/game-setting/jiao-shi-hou-tai.md#ru-he-da-kai-jiao-shi-hou-tai-de-fen-xiang-zuo-ye-de-gong-neng)
{% endhint %}

### 如何引用其他教師的作業？

<mark style="background-color:purple;">設定位置：教師後台 -> 教學資源 -> 分享作業</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-07-10 10.48.37 (1).png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊卡片下方的「預覽」，檢視其他教師所分享的作業內容
2. 如有需要引用某一作業，可於作業的預覽介面裡點擊右下角的「引用」，該作業就會進入作業管理的作業清單

### 如何編輯引用的作業？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-07-10 11.40.32 (1).png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊有引用標示的作業旁的選單，選擇「編輯」
2. 可在「題目來源」的 Tab 中新增該出題範圍下的其他題目至作業中
3. 可在「已選擇題目」的 Tab 中預覽作業內的所有題目，並更改其出題方式（選擇順序出題後可以進一步題目進行排序）

{% hint style="info" %}
如需進一步將引用任務打包成任務，可參閱[創建任務的說明](../zuo-ye-guan-li/ren-wu/#ru-he-zhi-pai-ren-wu-gei-xue-sheng)
{% endhint %}

### 如何取消引用作業？

<mark style="background-color:purple;">設定位置：教師後台 -> 作業管理 -> 作業清單</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-07-10 11.40.32.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊有引用標示的作業旁的選單，選擇「取消引用」
