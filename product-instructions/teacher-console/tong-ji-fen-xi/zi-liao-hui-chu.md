# 資料匯出

{% hint style="info" %}
作業類型

1. 個人作業：課堂競賽之盾的作業類型以外的作業皆屬於此項
2. 共管教師作業：班級內其他教師（共管教師）所出的作業
3. 世界作業：官方所派發的世界任務，其作業則屬於此項
4. 競賽之盾：若將作業分派為課堂競賽之盾，其作業屬於此項
{% endhint %}

### 如何匯出指定班級的作業正確率、完成率、作業分數的 excel 表格？

<mark style="background-color:purple;">匯出位置：教師後台 -> 統計分析 -> 作業清單</mark>

教師選定班級與作業類別後，能夠匯出最多五個作業的作答情形，檔案內含：作業名稱、學生的座號、姓名、暱稱與帳號，與其於該任務的作答表現：正確率、完成度、總答對率

1. 正確率：以「該作業的正確率」作為數據來源
2. 完成度：以「該作業的完成度」作為數據來源
3. 總答對率：即作業分數，為作業完成度＊作業正確率＊100

### 如何匯出指定班級學生的使用時數的 excel 表格？

<mark style="background-color:purple;">匯出位置：教師後台 -> 統計分析 -> 時數下載</mark>

教師選定班級、年份與月份後，能夠匯出學生的使用時數，檔案內含學生所在的組別名稱與組別 ID、座號、真實姓名與該月使用時數（以分鐘計算）
