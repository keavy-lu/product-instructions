# 統計分析

{% hint style="info" %}
<mark style="color:blue;">統計分析為統計學生答題狀況的功能</mark>
{% endhint %}

統計分析中有四項子功能：班級總覽、作業驗收、學生列表與資料匯出，前三者為教師查看學生答題狀況的不同形式，資料匯出則可匯出學生填答的資料

1. 班級總覽：教師能以「解題數量、領土數量、等級排名」等不同面向檢視學生排名，並於查看某「班級」於不同作業的答題狀況，包含完成度、正確度與作業分數​
2. 作業驗收：不同於班級總覽，教師能以「作業」為最小篩選單位，了解班級整體的平均正確率、完成度與作業分數。並取得作業的相關分析
3. 學生列表：與班級總覽相同，教師能以「班級」為最小篩選單位，了解個別學生在不同作業的答題情形
4. 資料匯出：將作業資料或是時數資訊以 excel 的檔案格式匯出
