# 匯出錯題

{% hint style="info" %}
作業類型

1. 個人作業：課堂競賽之盾的作業類型以外的作業皆屬於此項
2. 共管教師作業：班級內其他教師（共管教師）所出的作業
3. 世界作業：官方所派發的世界任務，其作業則屬於此項
4. 競賽之盾：若將作業分派為課堂競賽之盾，其作業屬於此項
{% endhint %}

### 如何匯出指定班級的錯題記錄的 Word 檔？

<mark style="background-color:purple;">匯出位置：教師後台 -> 統計分析 -> 匯出錯題</mark>

教師選定班級與作業類別，並選擇起始日與結束日，就能夠匯出該班所有學生在這段期間的錯題 Word 檔（倘若班級內大於兩人，會使用 Zip 壓縮檔進行下載），可以透過 Word 檔的檔名去辨認此文件是哪一位學生的錯題記錄&#x20;

e.g. 2023.06.21-2023.07.21\_王小明\_95896746。即可得知該檔案為2023.06.21-2023.07.21這段期間，ID 為 95896746 的王小明的錯題紀錄。

{% hint style="warning" %}
如果統計分析的功能列中並無出現匯出錯題的功能，原因是系統不開放教師使用這項功能，詳情參考[此連結](../../admin-courses/game-setting/jiao-shi-hou-tai.md#ru-he-da-kai-jiao-shi-hou-tai-de-yi-jian-chu-ti-de-gong-neng)
{% endhint %}
