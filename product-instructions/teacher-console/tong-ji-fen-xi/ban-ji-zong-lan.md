# 班級總覽

{% hint style="info" %}
作業類型

1. 個人作業：課堂競賽之盾的作業類型以外的作業皆屬於此項
2. 共管教師作業：班級內其他教師（共管教師）所出的作業
3. 世界作業：官方所派發的世界任務，其作業則屬於此項
4. 競賽之盾：若將作業分派為課堂競賽之盾，其作業屬於此項
{% endhint %}

### 如何查看班級學生解題排名、土地排名、等級排名？ <a href="#ban-ji-shi-shen-me" id="ban-ji-shi-shen-me"></a>

<mark style="background-color:purple;">檢視位置：教師後台 -> 統計分析 -> 班級總覽</mark>

教師選定班級與作業類型後，能夠以排名檢視學生在該遊戲世界內的答題狀況

1. 解題排名：以「答對題目的數量」作為排名依據
2. 領土排名：以「領土數量」作為排名依據
3. 等級排名：以「玩家等級」作為排名依據

{% hint style="info" %}
1) 排名與其對應的數值 e.g. 解題數量、領土數量並不會因為選定不同作業類型而改變
2) 解題數量是計算該 GC 在該世界有「答對」的題目數量且同一題答對多次不會重複計算，範圍不局限於四種作業類型裡的題目，在自選章節答對的題目也會被計算到解題數量中
{% endhint %}

### 如何查看班級學生於不同任務下的完成率、正確率、作業分數？

<mark style="background-color:purple;">檢視位置：教師後台 -> 統計分析 -> 班級總覽</mark>

教師選定班級與作業類型後，能夠以答題狀況檢視學生於不同任務裡的情形

1. 完成度：以「全班學生的平均完成度」作為數據來源
2. 正確率：以「全班學生的平均正確率」作為數據來源
3. 作業分數：全班學生的完成度＊全班學生的正確率＊100
