# 學生列表

{% hint style="info" %}
作業類型

1. 個人作業：課堂競賽之盾的作業類型以外的作業皆屬於此項
2. 共管教師作業：班級內其他教師（共管教師）所出的作業
3. 世界作業：官方所派發的世界任務，其作業則屬於此項
4. 競賽之盾：若將作業分派為課堂競賽之盾，其作業屬於此項
{% endhint %}

### 如何查看學生於單一作業類型下不同作業的正確率、完成率、作業分數？

<mark style="background-color:purple;">檢視位置：教師後台 -> 統計分析 -> 學生列表</mark>

教師選定班級與作業類別後，能夠查看該班級內的學生於單一作業類型下不同作業的答題狀況

1. 正確率：以「作業正確率」作為數據來源
2. 完成度：以「作業完成度」作為數據來源
3. 作業分數：作業完成度＊作業正確率＊100

{% hint style="info" %}
1) 此處的作答資訊不同於班級總覽的解題數量[^1]，只能夠看到老師自己所開設的任務
2) 如果某一個任務同時指派到兩個班級，而作答的用戶同時存在於這兩個班級中，系統會把作答資料紀錄給「先建立的那個班級」
{% endhint %}

[^1]: 題目來源包含自選章節、目前登入老師開的任務、其他老師開的任務
