# \[ 情境 ] 介面與課程切換

### 如何於教師後台切換至不同課程的教師後台？ <a href="#ru-he-xin-zeng-ban-ji" id="ru-he-xin-zeng-ban-ji"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> Sidebar -> 目前所在的課程世界名稱</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-11-15 11.03.25.png" alt="" width="563"><figcaption></figcaption></figure>

點擊目前所在的課程世界名稱，並接續點擊欲切換的課程，即可跳轉至該課程世界的教師後台。

{% hint style="info" %}
下拉式選單中僅會顯示已有教師後台權限的課程世界，並不會出現所有課程世界
{% endhint %}

### 如何從教師後台進入遊戲介面或素養學習中心？

<mark style="background-color:purple;">設定位置：教師後台 -> Sidebar -> 目前所在的課程世界名稱</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-11-15 11.13.32.png" alt="" width="563"><figcaption></figcaption></figure>

1. 如欲切換至遊戲介面，需點擊 Sidebar 的「遊戲」，即可跳轉到該課程世界的遊戲介面
2. 如欲切換至素養學習中心，需點擊 Sidebar 的「素養學習中心」，即可跳轉到該課程世界的素養學習中心
