# 語言切換

### 如何切換教師後台的介面語言？

<mark style="background-color:purple;">設定位置：教師後台 -> Header -> 語言切換</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-11-01 10.27.18拷貝.png" alt="" width="563"><figcaption></figcaption></figure>

點擊教師後台 Header 的「語言切換」後，並點選下拉式選單中介面欲呈現的語言後，即可變更語言設定。
