# 班級作答狀況

### 如何查看班級的各任務作答情形？

<mark style="background-color:purple;">檢視位置：素養數據 -> 班級作答狀況 -> 班級任務列表</mark>

教師能夠以任務的形式查看資訊，了解班級於不同任務間的作答狀況。

<figure><img src="../../../.gitbook/assets/image (40).png" alt="" width="563"><figcaption></figcaption></figure>

1. 請選擇教材類型
2. 請選擇班級、時間區間、任務來源（可複選）、並填入任務名稱（非必填）
3. 另外開啟其他篩選器：難易度、領域（非必填）
4. 即可
   * 查看特定班級、特定範圍下的素養數據
   * 點擊「未完成人數」查看未完成任務的學生名單
   * 點擊「查看教學資源」連結至教學資源的素養教學資源[查看連結的教案資源](../jiao-xue-zi-yuan/su-yang-jiao-xue-zi-yuan.md#su-yang-jiao-xue-zi-yuan-you-ti-gong-shen-me-dang-an)
   * 點擊「查看文章」[查看任務連結的文章內容](/broken/pages/xiv79opO266V9foEj0bQ)
   * 點擊「任務數據」查看班級於[單一任務的作答情形](/broken/pages/iN9XbQKZI8ItdqvsFKvF)
   * 點擊「匯出報表」將當前數據匯出成 Excel 檔案

{% hint style="info" %}
1) 單一任務多題目的情況下，難易度會是使用最高的那一個（難>中>易>無）\
   e.g. 任務內有四題題目，難易度分別是：難、中、中、無，則會顯示「難」
2) 單一任務多領域別的情況下，篩選到該任務的某一領域別時，也會被篩選到\
   e.g. Ａ任務有人文史地、社會科學、自然科學的領域別，若篩選器選擇「自然科學」時，所有具有自然科學領域別標籤的任務皆會顯示，包含Ａ任務
{% endhint %}

{% hint style="warning" %}
1. 如果管理者後台的教案資源管理處已有任務的對應教案，原因可能為教案並無綁定此任務，詳請可參考[此連結](../../admin-all-courses-setting/chan-pin-guan-li/jiao-xue-zi-yuan/jiao-an-guan-li.md#ru-he-cha-kan-huo-bian-ji-yi-shang-chuan-de-jiao-xue-zi-yuan)
2. 點擊「查看文章」後，如果是不支援顯示題組的詳解，會跳出「暫時無法此題查看詳解」提醒
{% endhint %}

### 如何查看並比較各個班級的作答狀況？ <a href="#ban-ji-shi-shen-me" id="ban-ji-shi-shen-me"></a>

<mark style="background-color:purple;">檢視位置：素養數據 -> 班級作答狀況 -> 各班答題統計</mark>

教師可以以班級的形式查看資訊，了解不同班級於相同任務來源裡的作答狀況。

<figure><img src="../../../.gitbook/assets/image (34).png" alt="" width="563"><figcaption></figcaption></figure>

1. 請選擇教材類型、任務來源
2. 請選擇欲檢視的班級（多選、可跨年級）
3. 選擇排序方式、參考數據（非必填、多選）
4. 即可
   * 查看各班級作答狀況的比較圖表
   * 以參考數據 e.g. 全國平均正確率、本校七年級平均正確率等作為輔助線，查看班級的作答狀況
   * 點擊「匯出報表」將當前數據匯出成 Excel 檔案
   * 點擊「下載成 JPG 檔」將當前數據圖匯出成 JPG 圖檔

{% hint style="warning" %}
如果各班作答統計的參考數據並無全國平均或全國私校平均，原因是系統不開放教師取用這份數據，詳情參考此[<mark style="color:blue;">連結</mark>](../../admin-courses/game-setting/jiao-shi-hou-tai.md)
{% endhint %}

### 如何查看並比較各個班級的作答趨勢？ <a href="#ban-ji-shi-shen-me" id="ban-ji-shi-shen-me"></a>

<mark style="background-color:purple;">檢視位置：素養數據 -> 班級作答狀況 -> 各班答題趨勢</mark>

與各班作答統計相同，相同教師可以以班級的形式查看資訊，了解不同班級於相同任務來源裡的作答狀況，其差異僅在於是否可可以另外選擇時間區段，各班作答趨勢能以時間作為橫軸，以折線圖的方式查看各班的作答狀況，以便教師查看班級在特定時間內的作答情形的變動。

<figure><img src="../../../.gitbook/assets/image (46).png" alt="" width="563"><figcaption><p>目前本校各年級平均正確率已改為本課程世界各年級平均完成率</p></figcaption></figure>

1. 請選擇教材類型、任務來源
2. 請選擇欲檢視的班級（多選、可跨年級）
3. 選擇排序方式、參考數據（非必填、多選）
4. 即可
   * 查看各班級作答狀況的比較折線圖
   * 以參考數據 e.g. 本課程世界各年級平均完成率、全國平均正確率，查看班級的作答狀況
   * 點擊「匯出報表」將當前數據匯出成 Excel 檔案
   * 點擊「下載成 JPG 檔」將當前數據圖匯出成 JPG 圖檔

{% hint style="warning" %}
如果各班作答趨勢的參考數據並無全國平均或全國私校平均，原因是系統不開放教師取用這份數據，詳情參考此[<mark style="color:blue;">連結</mark>](../../admin-courses/game-setting/jiao-shi-hou-tai.md)
{% endhint %}

### 如何查看任務連結的文章內容？

<mark style="background-color:purple;">檢視位置：素養數據 -> 班級作答狀況 -> 班級任務列表 -> 查看文章</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-07-19 10.35.44.png" alt="" width="563"><figcaption></figcaption></figure>

查看文章分為兩個區塊，分別是<mark style="color:red;">文章或問題</mark>與<mark style="color:red;">答題狀況與詳解</mark>

**文章與問題**

1. 最上方可得知該文章屬於哪一個任務
2. 第二行文字可得知該文章的測驗能力、題型與難度

**答題狀況與詳解**

1. 可查看該文章的影音詳解
2. 若該文章為題組題，可以點擊題目選項下方的數字按鈕，切換至不同題目
3. 每題題目都會呈現出對應的題目選項、正確答案、答題分佈（各選項有幾個學生選）與官方詳解
4. 可點擊「眼睛圖示」決定是否顯示內容

**其他操作**

1. 若該任務有多題，可以於左上方的導覽欄進行切換；也可以選擇用右下角的按鈕進行切換
2. 若問題為長篇文章，可點擊兩個區塊間的顯示按鈕，切換三種不同閱讀方式
   * 僅閱讀<mark style="color:red;">文章或問題</mark>
   * 同時閱讀<mark style="color:red;">文章或問題</mark>與<mark style="color:red;">答題狀況與詳解</mark>
   * 僅閱讀<mark style="color:red;">答題狀況與詳解</mark>
3. 若發現題目有錯，也可使用錯題回報功能，將問題回報給出題的管理層
4. 可點選文章側邊的「全螢幕」，進入全螢幕顯示

### 班級作答狀況-班級任務列表頁面中的數據來源為何？

1. 完成率：在此時間區間內，班級學生完成的任務數量 / 收到的任務數量。注意：學生必須至任務書中按下「完成任務」，才會計入次數
2. 平均正確率：在此時間區間內，班級完成的任務的正確率加總 / 完成的任務數量

### 班級作答狀況-班級任務列表-任務數據頁面中的數據來源為何？

1. 正確率：該任務內的答對題數 / 該任務的總題數
2. 答對題數：該任務內的答對題數 ，以及該任務的總題數
3. 作答時間：為首次開始作答至按下「送出」或「結算」的時間長
4. 完成時間：學生至任務書中按下「完成任務」的時間

1. 當使用者首次開始作答至按下「送出」或「結算」時，就會計算第一次作答記錄，包含但不限於下列數據：如正確率、答對題數、作答時間等
