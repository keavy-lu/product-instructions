# 學生作答狀況

### 如何查看班級內各個學生於不同任務間的作答情況？

<mark style="background-color:purple;">檢視位置：素養數據 -> 學生作答狀況 -> 學生作答記錄</mark>

教師能夠以概覽的方式班級內學生於不同任務間的作答情況。

<figure><img src="../../../.gitbook/assets/image (50).png" alt="" width="563"><figcaption><p>最上面的標題錯植，請忽略</p></figcaption></figure>

1. 請選擇教材類型
2. 請選擇班級、時間區間、任務來源（可複選）、並設定標示顏色的正確率標準
3. 即可
   * 查看班級學生於時間區間內的答題狀況，包括平均正確率、平均作答時間、完成率
   * 查看班級學生於時間區間內、於各任務的正確率狀況（將鼠標移至數字上可以看到詳細的任務名稱）
   * 點擊右下方視窗查看更多任務的答題狀況
   * 點擊任一學生，可以查看該學生於時間區段內的作答表現
   * 點擊「匯出報表」將當前數據匯出成 Excel 檔案

{% hint style="info" %}
1) 在選擇時間區間時，時長限制為１年
2) 當時間區間內的任務數量超過 50 個，建議直接下載報表（因資料量較大，表格右側的各任務正確率將無法完整顯示）
{% endhint %}

### 如何查看班級內學生於時間區段內的作答表現？

<mark style="background-color:purple;">檢視位置：素養數據 -> 學生作答狀況 -> 學生學習分析</mark>

相對於學生作答記錄，教師能夠更仔細地檢視班級內單一學生於指定時間區間內的作答情況，包括作答數據、能力分析、任務表現。另外，也開放各個教師能夠給予不同學生教學上的附註。

<figure><img src="../../../.gitbook/assets/image (48).png" alt="" width="563"><figcaption></figcaption></figure>

1. 請選擇教材類型
2. 請選擇欲檢視的班級、學生、任務派發的時間區間[^1]、任務來源（可複選）
3. 即可
   * 查看個人作答數據
     * 作答篇數 / 應答篇數
     * 平均完成率、平均正確率、平均作答時間
     * 時間區間內的表現折線圖
   * 查看閱讀素養能力分析
     * 五種閱讀素養能裡下的表現雷達圖（可另外調整過去參考數據：半年[^2]、一年[^3]、過去所有時間[^4]、不顯示）
     * 個人、班級平均、年級平均、全國平均的表現長條圖
   * 查看各領域素養任務表現，如平均完成率、平均正確率、平均作答時間等
   * [創建教師觀察附註](xue-sheng-zuo-da-zhuang-kuang.md#ru-he-chuang-jian-jiao-shi-guan-cha-fu-zhu)或是[查看其他教師所留下的教師備註](xue-sheng-zuo-da-zhuang-kuang.md#ru-he-cha-kan-qi-ta-jiao-shi-suo-liu-xia-de-jiao-shi-bei-zhu)

{% hint style="info" %}
1) 當時間區間內學生所完成的任務不滿 16 個時，個人作答數據的「平均正確率」僅供參考，建議等到完成一定數量任務之後再以該數字做為評分標準
2) 老師只可以刪除自己發佈的觀察備註，不可以刪除其他人的
{% endhint %}

{% hint style="warning" %}
如果閱讀素養能力分析的參考數據並無班級平均、年級平均或全國平均，原因是系統不開放教師取用這份數據，詳情參考此[<mark style="color:blue;">連結</mark>](../../admin-courses/game-setting/jiao-shi-hou-tai.md)
{% endhint %}

### 如何創建教師觀察附註？

<mark style="background-color:purple;">設定位置：素養數據 -> 學生作答狀況 -> 學生學習分析 -> 教師觀察備註</mark>

可於文字輸入框輸入觀察備註，點擊「發布」即可留下備註。

{% hint style="info" %}
1. 教師只可以刪除自己發佈的觀察備註，不可以刪除其他人的
2. 觀察備註不具備編輯功能
{% endhint %}

### 如何查看其他教師所留下的教師備註？

<mark style="background-color:purple;">設定位置：素養數據 -> 學生作答狀況 -> 學生學習分析 -> 教師觀察備註</mark>

可於教師備註區塊查看其他教師的教師備註資訊，包含：

1. 教師備註內容
2. 發布該備註的教師
3. 發布該備註的時間點

[^1]: 課程大廳的學習數據是使用「任務完成時間」來判斷時間區間，而非「任務派發時間」，因此教師後台的學習數據與首頁的學習數據會不一致。

[^2]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去半年數據為 2022/03/01～2022/08/31

[^3]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去一年數據為 2021/09/01～2022/08/31

[^4]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去所有時間數據為 啟用日

    ～2022/08/31
