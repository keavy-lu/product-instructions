# 素養數據

{% hint style="info" %}
<mark style="color:blue;">素養數據為檢視學生的素養任務完成狀況的功能</mark>
{% endhint %}

素養數據中有三項子功能，分別為：

1. 班級作答狀況：能以班級為最小單位的方式查看各班級於不同教材類型下的作答狀況
2. 學生作答狀況：能以學生為最小單位的方式查看學生於不同教材類型下的作答狀況
3. 全校作答狀況：查看全校與各年級的作答狀況

{% hint style="warning" %}
如果此功能並無顯示於側邊功能欄，原因可能是沒有權限，若需要打開該權限，可查看[該連結](../../admin-courses/game-setting/jiao-shi-hou-tai.md#ru-he-da-kai-su-yang-shu-ju-de-ru-kou)
{% endhint %}
