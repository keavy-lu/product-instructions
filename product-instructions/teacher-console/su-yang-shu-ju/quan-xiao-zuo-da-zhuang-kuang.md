# 全校作答狀況

### 如何查看全校與各年級的作答數據？

<mark style="background-color:purple;">檢視位置：素養數據 -> 全校作答狀況 -> 全校作答統計</mark>

教師可查看全校整體與各年級個別的作答情況。

<figure><img src="../../../.gitbook/assets/image (47).png" alt="" width="563"><figcaption></figcaption></figure>

1. 請選擇教材類型
2. 請選擇時間區間、任務來源（可複選）
3. 即可
   * 查看全校整體的平均正確率與平均完成率
   * 查看各年級的平均正確率與平均完成率
   * 以參考數據 e.g. 全國平均正確率、全校平均完成率等作為輔助線，查看各年級的作答狀況
   * 點擊「匯出報表」將當前數據匯出成 Excel 檔案

### 如何查看全校與各年級的作答趨勢？

<mark style="background-color:purple;">檢視位置：素養數據 -> 全校作答狀況 -> 全校作答趨勢</mark>

教師可查看全校整體與各年級個別的作答趨勢。

<figure><img src="../../../.gitbook/assets/image (43).png" alt="" width="563"><figcaption></figcaption></figure>

1. 請選擇教材類型
2. 請選擇時間區間、任務來源（可複選）
3. 即可
   * 查看全校整體的平均正確率與平均完成率
   * 以參考數據 e.g. 私校正確率、全國正確率等作為輔助線，查看全校於時間區間內的的作答情況折線圖表
   * 查看各年級的平均正確率與平均完成率折線圖表
   * 以參考數據 e.g. 全校平均正確率、全國平均正確率等作為輔助線，查看各年級的作答情況折線圖表
   * 點擊「匯出報表」將當前數據匯出成 Excel 檔案

{% hint style="warning" %}
如果各年級趨勢圖的參考數據並無全國平均或全國私校平均，原因是系統不開放教師取用這份數據，詳情參考此[<mark style="color:blue;">連結</mark>](../../admin-courses/game-setting/jiao-shi-hou-tai.md)
{% endhint %}
