# 我的班級

### 如何新增班級？ <a href="#ru-he-xin-zeng-ban-ji" id="ru-he-xin-zeng-ban-ji"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 我的班級</mark>

點選「新增班級」，填入班級名稱與人數，選擇是否要求「學生加入班級時需填寫座號」後點選儲存。

{% hint style="info" %}
成功創建一個新班級後，請將右上角的班級代碼分享給學生：學生進入遊戲後，在上方列表點選「我的班級」，輸入班級與指定座號（如有要求）即可加入
{% endhint %}

### 如何刪除班級？ <a href="#ru-he-xin-zeng-ban-ji-zuo-wei" id="ru-he-xin-zeng-ban-ji-zuo-wei"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 我的班級</mark>

點選任一班級標籤為「自建」的班級、Hover 於班級封面，即可點擊垃圾桶 icon，刪除該班級。

### 如何新增班級座位？ <a href="#ru-he-xin-zeng-ban-ji-zuo-wei" id="ru-he-xin-zeng-ban-ji-zuo-wei"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 我的班級 -> 某一班級</mark>

點選「新增學生」後切換至「新增座位」分頁，輸入新增的座位數，選擇是否要求「學生加入班級時需填寫座號」後點選儲存。

### 如何讓學生透過班級代碼加入班級？ <a href="#ru-he-xin-zeng-ban-ji-zuo-wei" id="ru-he-xin-zeng-ban-ji-zuo-wei"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 我的班級</mark>

班級卡片上都會附有班級代碼的資訊，複製該代碼給學生，並由學生[加入班級](../../game-interface/header/wo-de-ban-ji.md#ru-he-jia-ru-ban-ji)。

### 如何批量新增班級成員？ <a href="#ru-he-xin-zeng-ban-ji-zuo-wei" id="ru-he-xin-zeng-ban-ji-zuo-wei"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 我的班級 -> 某一班級</mark>

點選「新增學生」後切換至「匯入學生」分頁，逐行輸入欲新增為班級成員的學生帳號，並點選儲存即可。

{% hint style="warning" %}
如果無法於教師後台看到此匯入學生的功能，原因是目前並無打開批量匯入學生的功能開關，詳情參考[此連結](../../admin-courses/game-setting/jiao-shi-hou-tai.md#ru-he-da-kai-jiao-shi-hou-tai-de-hui-ru-xue-sheng-gong-neng)
{% endhint %}

### 如何將新增分組組別？ <a href="#ru-he-jiang-xin-zeng-fen-zu-zu-bie" id="ru-he-jiang-xin-zeng-fen-zu-zu-bie"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 我的班級 -> 某一班級</mark>

點選「新增小組」，輸入小組名稱創建小組。

### 如何將編輯小組內成員？ <a href="#ru-he-jiang-bian-ji-xiao-zu-nei-cheng-yuan" id="ru-he-jiang-bian-ji-xiao-zu-nei-cheng-yuan"></a>

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 我的班級 -> 某一班級</mark>

隨意將學生拖入或拖離小組區塊即可調整分組狀態。

{% hint style="warning" %}
每個組別至少要有一位學生，當組別內只剩一位學生時，該學生無法被移除該組別。 若需移除該位學生，請先加入其他學生或整個組別刪除再新增。
{% endhint %}
