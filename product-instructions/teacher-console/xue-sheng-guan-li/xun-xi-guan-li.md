# 訊息管理

### 如何發送班級訊息？

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 訊息管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-07-18 13.49.23.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「發送新訊息」後即可進入該新訊息的設定頁面​。選擇發送對象並輸入訊息標題與訊息內容後，即可點擊發送，將訊息傳送給指定班級、組別或學生。發送完的訊息細節（包括標題、內文與發送時間）也可以於訊息管理頁面的訊息發送紀錄中查看。

### 如何編輯班級訊息？

<mark style="background-color:purple;">設定位置：教師後台 -> 學生管理 -> 訊息管理 -> 查看訊息</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-07-18 13.48.12.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「編輯」後即可進入訊息內容的編輯頁面，可進一步編輯訊息標題與訊息內容。

{% hint style="warning" %}
發送對象不可編輯，如需要更動發送對象，需要直接刪除該訊息，並於重新發送訊息的過程中選定正確的發送對象
{% endhint %}
