# 團體登入頁

### 如何從課程快捷進入團體登入頁？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> 課程快捷</mark>（需為未登入狀態）

<figure><img src="../../.gitbook/assets/截圖 2023-11-15 10.28.46.png" alt="" width="563"><figcaption></figcaption></figure>

在尚未登入的狀態下，用戶可從主頁的課程快捷中點擊「團體登入頁」，並於彈窗中選擇欲使用的團體登入頁，即可跳轉至指定的團體登入頁面，並進行登入。

{% hint style="info" %}
彈窗中所顯示的團體登入頁並非所有的團體登入頁，此處僅會顯示公開的團體登入頁，其餘不公開的團體登入頁則會需要有特定網址才可進入
{% endhint %}

### 如何從課程列表進入團體登入頁？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> 課程列表</mark>（需為未登入狀態）

<figure><img src="../../.gitbook/assets/截圖 2023-11-15 10.28.36.png" alt="" width="563"><figcaption></figcaption></figure>

在尚未登入的狀態下，用戶可從主頁的課程列表中點擊欲使用的「團體登入頁」、跳轉至指定的團體登入頁面，並進行登入。

{% hint style="info" %}
課程列表頁僅會顯示公開的團體登入頁，其餘不公開的團體登入頁則會需要有特定網址才可進入
{% endhint %}

### 如何從特定網址進入團體登入頁？

<mark style="background-color:purple;">設定位置：團體登入頁的特定網址</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-11-15 10.35.57.png" alt="" width="563"><figcaption></figcaption></figure>

直接進入特定網址後，即可進行登入。
