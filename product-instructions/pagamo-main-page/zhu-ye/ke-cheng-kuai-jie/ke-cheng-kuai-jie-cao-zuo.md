# 課程快捷 - 操作

### 如何進入上一次進入的課程世界、學習中心與教師後台？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> 課程快捷</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-17 15.45.18.png" alt="" width="563"><figcaption></figcaption></figure>

1. 從快捷區域的文字可以得知上次進入的課程世界
2. 點擊「進入遊戲」後，可以進入該課程世界的遊戲介面
3. 點擊「素養學習中心」後，可以進入素養學習中心查看素養任務、答題
4. 點擊「教師後台」後，可以進入該課程世界的教師後台

{% hint style="info" %}
1) 僅有取得素養商品的玩家能夠於有開放素養學習中心的課程中使用課程快捷進入學習中心
2) 僅有取得課程世界的教師權限的教師能夠使用課程快捷進入該課程世界的教師後台
{% endhint %}
