# 地區

### 如何切換網頁地區設定？

<mark style="background-color:purple;">購買位置：PaGamO 主頁 -> Header -> 所在地區</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-10-17 17.14.30 (1).png" alt="" width="563"><figcaption></figcaption></figure>

點擊「地區」後，並點擊欲切換的網頁地區，即可生效。

{% hint style="info" %}
目前地區選項僅開放

1. 全球 / 台灣
2. 馬來西亞
3. 香港
{% endhint %}
