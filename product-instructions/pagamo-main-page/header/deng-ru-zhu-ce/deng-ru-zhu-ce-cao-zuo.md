# 登入註冊 - 操作

### 如何以信箱註冊並啟用帳號？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 登入註冊</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-16 22.08.42拷貝.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊頁面左側的「沒有帳號？前往 Email 註冊」
2. 輸入帳號、電子信箱、密碼
3. 至電子信箱內收取信箱驗證信並驗證帳號
4. 成功驗證帳號後，則可以順利使用該帳號[從帳號入口登入 PaGamO](/broken/pages/eKPfFw8606CkRxN9pznr#ru-he-shi-yong-zhang-hao-deng-ru)

{% hint style="info" %}
倘若沒有收到驗證信，可能原因為填錯電子信箱或是系統出現問題，用戶可以於註冊完帳號的跳轉頁面中選擇「更改電子信箱」或是「重新寄送」驗證信至電子信箱
{% endhint %}

### 如何以縣市 Open ID 帳號或教育雲註冊並啟用帳號？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 登入註冊</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-16 22.08.42 (1).png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊頁面右側的「使用教育雲或是縣市帳號登入」，進入教育雲介面
2. 點擊「申請教育雲端帳號」，並選擇身份別、填入個人資訊後即可申請教育雲端帳號
3. 回到教育雲介面，點擊「啟用教育雲端帳號」，並選擇啟用帳號的方式
4. 成功啟用帳號後，則可以順利使用該帳號與新密碼從教育雲端入口登入 PaGamO

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-16 22.08.42 (1).png" alt="" width="563"><figcaption></figcaption></figure>

1. 根據用戶已綁定第三方帳號的軟體，點擊頁面右側的對應「第三方登入圖示」
2. 輸入該第三方帳號的帳號密碼，即可登入 PaGamO

### 如何重設一般帳號的密碼？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 登入註冊</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-18 10.49.51拷貝.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊頁面左側的「忘記密碼」
2. 填入當時註冊帳號時所使用的電子信箱並點擊「寄出密碼修正信」
3. 至該電子信箱收取密碼修正信並重設密碼
4. 成功重設密碼後，則可以順利使用該帳號與新密碼[從帳號入口登入 PaGamO](/broken/pages/eKPfFw8606CkRxN9pznr#ru-he-shi-yong-zhang-hao-deng-ru)

### 如何重設縣市 Open ID 帳號或教育雲的密碼？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 登入註冊</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-16 22.08.42 (1).png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊頁面右側的「使用教育雲或是縣市帳號登入」，進入教育雲介面
2. 點擊「忘記教育雲端密碼」，並選擇重設密碼的方式
   * 使用縣市帳號重設
   * 使用個人基本資料重設
   * 使用預設密碼重設
   * 傳送驗證碼至備用電子信箱與電話
3. 成功重設密碼後，則可以順利使用該帳號與新密碼從教育雲端入口登入 PaGamO

{% hint style="info" %}
目前縣市 Open ID 不開放由用戶自己更改密碼，僅能由學校人士或網管進行修改
{% endhint %}
