# 消費紀錄

### 如何檢視過去的消費紀錄？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 用戶資訊 -> 消費紀錄</mark>

<figure><img src="../../../../.gitbook/assets/image (61).png" alt="" width="563"><figcaption></figcaption></figure>

於消費紀錄頁面，可檢視過去的訂單記錄，點擊狀態為「完成」的訂單後，可展開查看更多該已完成訂單的資訊，若點選「發票資訊」，則可跳轉至電子發票明細。如需替未付款訂單付款，可參閱[此區塊](xiao-fei-ji-lu.md#ru-he-ti-wei-fu-kuan-ding-chan-fu-kuan)。

### 如何替未付款訂單付款？



<figure><img src="../../../../.gitbook/assets/消費記錄_1440.png" alt="" width="563"><figcaption></figcaption></figure>

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 用戶資訊 -> 消費紀錄 -> 狀態為等待付款的訂單</mark>

點擊狀態為「等待付款」的訂單後，可展開查看更多該未付款訂單的資訊，若點選「付款資訊」，則可跳轉至付款資訊頁，可接續完成付款。
