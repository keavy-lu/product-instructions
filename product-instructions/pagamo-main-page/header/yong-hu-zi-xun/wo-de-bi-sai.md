# 我的比賽

### 如何查閱或是進入我參與的電競賽事？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 用戶資訊 -> 我的比賽</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-18 14.06.00拷貝2.png" alt="" width="563"><figcaption></figcaption></figure>

於登入後點擊個人頭像，並於選單中點擊「我的比賽」，即可：

* 從此區塊中查閱此帳號已報名的賽事與其競賽名稱與賽制（個人賽或團體賽）
* 點擊任一比賽的「比賽資訊」，查看更多的比賽資訊
* 點擊任一比賽的「前往賽場」，進入競賽

{% hint style="info" %}
進入競賽不代表可以立即進入遊戲界面、開始答題，如果該競賽尚未開賽，則會出現競賽開賽的剩餘時間
{% endhint %}
