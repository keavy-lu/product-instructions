# 學習數據

### 如何檢視素養任務的學習數據？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 用戶資訊 -> 學習數據</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-18 14.06.00.png" alt="" width="563"><figcaption></figcaption></figure>

於登入後點擊個人頭像，並於選單中點擊「學習數據」，並選定教材類型、[**任務完成的時間區間**](#user-content-fn-1)[^1]、任務來源，即可：

* 查看時間區段內的整體表現：總完成任務數、任務平均正確率[^2]、任務平均作答時間[^3]
* 選定篩選條件，查看該難度或是領域下，時間區段內的正確率趨勢
* 查看時間區段內的閱讀素養能力分析：五種閱讀素養能裡下的表現雷達圖（可另外調整過去參考數據：半年[^4]、一年[^5]、過去所有時間[^6]、不顯示）
* 查看時間區段內的素養領域表現分析：平均正確率[^7]、任務完成個數、任務平均作答時間[^8]

{% hint style="info" %}
1. 選定時間時可以調整不同時間範圍來呈現最長一年的作答數據
2. 任務來源的特殊考題為測驗模式的任務
3. 當時間區間內孩子所完成的任務不滿 16 個時，學習數據的「正確率趨勢」、「素養領域表現分析」僅供參考，建議等到完成一定數量任務之後再以該數字做為評分標準
4. 每個數據區塊都可以單獨截圖下載
{% endhint %}

[^1]: 教師後台的學習數據是使用「任務派發時間」來判斷時間區間，而非「任務完成時間」，因此首頁的學習數據與教師後台的數據會不一致。

[^2]: 每個任務首次作答正確率平均

[^3]: 每個任務首次作答時間的平均

[^4]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去半年數據為 2022/03/01～2022/08/31

[^5]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去一年數據為 2021/09/01～2022/08/31

[^6]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去所有時間數據為 啟用日

    ～2022/08/31

[^7]: 任務正確率總和 / 指定時間區間內完成的任務數 x 100%

[^8]: 各任務首次作答時間的平均
