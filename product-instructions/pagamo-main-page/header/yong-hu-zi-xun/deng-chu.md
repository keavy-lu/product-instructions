# 登出

### 如何在首頁登出當前使用的帳號？

<mark style="background-color:purple;">設定位置：遊戲介面 -> Hader -> 用戶資訊 -> 登出</mark>

一點擊「登出」，系統即會替玩家登出當前使用的帳號，並跳轉至首頁大廳。
