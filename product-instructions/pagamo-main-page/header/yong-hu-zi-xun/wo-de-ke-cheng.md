# 我的課程

### 如何查閱或進入已加入的課程世界？

<mark style="background-color:purple;">設定位置：PaGamO 主頁 -> Header -> 用戶資訊 -> 我的課程</mark>

<figure><img src="../../../../.gitbook/assets/截圖 2023-10-18 14.06.00.png" alt="" width="563"><figcaption></figcaption></figure>

於登入後點擊個人頭像，並於選單中點擊「我的課程」，即可：

* 從此區塊中查閱此帳號已加入的課程世界
* 點擊任何一個課程世界後，即可跳轉至該課程世界的遊戲介面

{% hint style="info" %}
此區塊功能與主頁的「[已加入課程](../../zhu-ye/yi-jia-ru-ke-cheng/yi-jia-ru-ke-cheng-cao-zuo.md)」區塊相同
{% endhint %}
