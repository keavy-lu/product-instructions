# 學習數據

{% hint style="info" %}
如果用戶有購買任一世界的任一教材，且這些世界裡有至少一個開放用戶使用學習中心，則該用戶可以進入學習中心進行答題（因學習中心不分世界）
{% endhint %}

### 如何查看素養任務的作答數據？

<mark style="background-color:purple;">檢視位置：學習中心 -> 數據儀表板的更多學習數據 or header 的學習數據</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-22 16.16.46.png" alt="" width="563"><figcaption></figcaption></figure>

點擊「數據儀表板的更多學習數據」 or 「導航欄的學習數據」，進入學習數據（個人學習表現）頁面後，選定教材類型、任務完成的時間區間、任務來源，即可：

* 查看時間區段內的整體表現：總完成任務數、任務平均正確率[^1]、任務平均作答時間[^2]
* 選定篩選條件，查看該難度或是領域下，時間區段內的正確率趨勢
* 查看時間區段內的閱讀素養能力分析：五種閱讀素養能裡下的表現雷達圖（可另外調整過去參考數據：半年[^3]、一年[^4]、過去所有時間[^5]、不顯示）
* 查看時間區段內的素養領域表現分析：平均正確率[^6]、任務完成個數、任務平均作答時間[^7]

{% hint style="info" %}
1. 選定時間時可以調整不同時間範圍來呈現最長一年的作答數據
2. 當時間區間內孩子所完成的任務不滿 16 個時，學習數據的「正確率趨勢」、「素養領域表現分析」僅供參考，建議等到完成一定數量任務之後再以該數字做為評分標準
3. 每個數據區塊都可以單獨截圖下載
{% endhint %}

[^1]: 每個任務首次作答正確率平均

[^2]: 每個任務首次作答時間的平均

[^3]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去半年數據為 2022/03/01～2022/08/31

[^4]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去一年數據為 2021/09/01～2022/08/31

[^5]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去所有時間數據為 啟用日

    ～2022/08/31

[^6]: 任務正確率總和 / 指定時間區間內完成的任務數 x 100%

[^7]: 各任務首次作答時間的平均
