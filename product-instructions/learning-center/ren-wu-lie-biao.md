# 任務列表

{% hint style="info" %}
1. 如果用戶有購買任一世界的任一教材，且這些世界裡有至少一個開放用戶使用學習中心，則該用戶可以進入學習中心進行答題（因學習中心不分世界）
2. 素養學習中心答題不再扣除能量，故完全沒有顯示任何有關能量的資訊
3. 使用者在素養學習中心完成任務後，仍會獲得道具，但中心不會有任何獲得道具提示。=> 只要該世界有開放該獎勵道具，系統會發送道具獎勵的通知信到使用者所在的所有世界中，使用者拿取一次後，其他信件就會自動消失
{% endhint %}

{% hint style="info" %}
TAB 內頁顯示的規則

1. 帳號未綁定教材：顯示教材介紹和導購按鈕
2. 帳號綁定教材，未收到任務：顯示任務即將派發頁面
3. 帳號綁定教材，收到任務：根據任務來源教材所屬的內容供應商（如果商品沒有綁正確的內容供應商，任務還是會出現在全部任務 tab，但不會出現在正確的內容供應商 tab 下的任務清單內）
{% endhint %}

### 如何查看目前持有的任務？

<mark style="background-color:purple;">檢視位置：學習中心 -> 全部任務</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-22 15.27.01.png" alt="" width="563"><figcaption></figcaption></figure>

於熱門任務與數據儀表板的區塊下方可以檢視目前持有的所有任務與其資訊，包括任務名稱、任務到期日、任務的文本領域、題型與題目難度。

### 如何搜尋目前持有的任務？

<mark style="background-color:purple;">檢視位置：學習中心 -> header -> 搜尋任務標題</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-10-30 17.00.11.png" alt="" width="563"><figcaption></figcaption></figure>

輸入該任務名稱即可進行搜尋，任務的搜尋結果則會分為「尚未完成」、「已經完成」。

### 如何回答尚未完成的任務？

<mark style="background-color:purple;">回答位置：學習中心 -> 全部任務 -> 未完成任務</mark>

於熱門任務與數據儀表板的區塊下方可以檢視目前持有的所有任務與其資訊，包括任務名稱、任務到期日、任務的文本領域、題型與題目難度。如點擊未完成的任務，即可進入作答頁面，即可開始答題。

{% hint style="info" %}
做完測驗模式的任務後，系統會在交卷後就直接計算分數，不需要等使用者按下領取獎勵
{% endhint %}

### 如何查看任務作答紀錄？

<mark style="background-color:purple;">檢視位置：學習中心 -> 全部任務 -> 已完成任務</mark>

於熱門任務與數據儀表板的區塊下方可以檢視目前持有的所有任務與其資訊，包括任務名稱、任務到期日、任務的文本領域、題型與題目難度。如點擊已完成的任務，即可進入作答數據頁，進而：

* 查看任務完成時間
* 查看第一次作答記錄：正確率、答對題數、作答時間
* 查看題目對應到的閱讀素養能力與第一次答題情形
* 點擊任務內題目跳轉至該題目，並查看文章、題目、第一次作答選項
