# 已完成任務列表

{% hint style="info" %}
如果用戶有購買任一世界的任一教材，且這些世界裡有至少一個開放用戶使用學習中心，則該用戶可以進入學習中心進行答題（因學習中心不分世界）
{% endhint %}

### 如何查看已完成的任務？

<mark style="background-color:purple;">檢視位置：學習中心 -> 導航欄的個人頭像 -> 已完成任務列表</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-22 16.35.07.png" alt="" width="563"><figcaption></figcaption></figure>

點擊 「個人頭像的已完成任務列表」後，選定任務完成的時間區間，即可：

* 查看任務完成時間、任務名稱、正確率
* 依照完成時間或正確率進行排序
* 輸入任務名稱，尋找特定任務

{% hint style="info" %}
選定時間時可以調整不同時間範圍來呈現時間區段最長一年內的任務
{% endhint %}

### 如何檢視某一已完成任務的作答數據？

<mark style="background-color:purple;">檢視位置：學習中心 -> 導航欄的個人頭像 -> 已完成任務列表 -> 某一任務</mark>

<figure><img src="../../.gitbook/assets/截圖 2024-01-08 17.22.50.png" alt="" width="563"><figcaption></figcaption></figure>

進入已完成任務列表，點選任一已完成任務後，即可跳轉至該任務的作答數據頁，進而：

* 查看任務完成時間
* 查看第一次作答記錄：正確率、答對題數、作答時間
* 查看題目對應到的閱讀素養能力與第一次答題情形
* 點擊「查看文章」，查看該任務對應文章
* 點擊任務內題目跳轉至該題目，並查看文章、題目、第一次作答選項
