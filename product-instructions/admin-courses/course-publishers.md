# 引用題庫管理

### 如何引用題庫至目前課程？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 引用題庫管理 -> 引用題庫列表</mark>



<figure><img src="../../.gitbook/assets/截圖 2023-08-09 13.46.07.png" alt="" width="563"><figcaption></figcaption></figure>

點擊自己欲引用的題庫右方的「引用此題庫」，並選擇該題庫的使用權限，即可將該題庫匯入「目前課程使用的題庫」的列表中。

### 如何新增引用題庫？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 引用題庫管理 -> 引用題庫列表</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-09 13.46.41.png" alt="" width="563"><figcaption></figcaption></figure>

如果在引用題庫列表中並無自己欲引用的題庫，可以選擇自行增加題庫至列表。

1. 點擊列表上方的「新增引用題庫」
2. 選擇**精準搜尋**的類別：課程名稱（查詢該課程內的題庫）、教師帳號（查詢該教師的個人題庫）
3. 點擊「引用」，新增該題庫至引用題庫列表

### 如何編輯目前課程所使用的引用題庫？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 引用題庫管理 ->目前課程所使用的引用題庫</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-09 13.48.17.png" alt="" width="563"><figcaption></figcaption></figure>

1. 如需改變某題庫的使用權限，可點擊「編輯」並選擇權限：僅限教師使用 / 師生皆可使用。
2. 如需變更題庫的排序，可以點擊排序欄位的「操作icon」，並拖曳至指定排序位置，即可變更排序。
3. 如需將某題庫從目前課程使用的題庫清單中移除，可點擊「刪除」進行移除。

{% hint style="info" %}
權限

1. 僅限教師使用：僅有教師能夠在教師後台創建作業時能夠選用
2. 師生皆可使用：除了教師能夠在教師後台創建作業時能夠選用外，學生也可以於自選章節中選擇，進行答題
{% endhint %}

{% hint style="warning" %}
付費題庫如有被後台管理者引用至課程題庫中，但倘若玩家沒有購買，一樣不會出現在自選章節
{% endhint %}
