# 學習中心相關設定

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.48.14.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何打開首頁的學習中心入口？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程設定 -> 進階設定 -> 學習中心區塊</mark>

開啟「學習中心」的 Toggle

### 如何開放素養學習的遊戲入口？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程設定 -> 進階設定 -> 學習中心區塊</mark>

1. 先開啟「學習中心」的 Toggle
2. 開啟「開放遊戲入口」的 Toggle

### 如何打開暫時關閉遊戲入口的功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程設定 -> 進階設定 -> 學習中心區塊</mark>

1. 先開啟「學習中心」的 Toggle
2. 開啟「遊戲入口管理」的 Toggle
