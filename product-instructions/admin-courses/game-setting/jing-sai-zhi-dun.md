# 競賽之盾

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.47.27.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何打開競賽之盾的入口？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 競賽之盾區塊</mark>

依照需求打開欲開放入口的競賽類型（兩個項目可以單選也可複選）

* 開放官方競賽
* 開放課堂競賽

{% hint style="info" %}
若僅開啟課堂或官方競賽之盾的其中一個，遊戲介面只會顯示有開啟的一扇門
{% endhint %}

### 如何設定教師開設的課堂競賽總數量上限？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 競賽之盾區塊</mark>

依照需求填入該課程世界能夠開設的課堂競賽數量上限（該數量為過去到現在所開設的競賽數量總和）

{% hint style="info" %}
1. 此功能試營運販售項目，僅適合私人課程、教師數量少的情境
2. 每次使用課堂競賽都會告知剩下使用次數
{% endhint %}

### 如何設定教師開設的單場課堂競賽人數上限？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 競賽之盾區塊</mark>

依照需求填入該課程世界的單場課堂競賽人數上限

{% hint style="info" %}
最高人數上限為 500 人，如果要進行調整的話，仍需要通知 PM，以免流量過載
{% endhint %}
