# 遊戲教學

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.47.01.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何開啟遊戲介面中的「新手教學」功能

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 遊戲教學區塊</mark>

開啟「開啟遊戲教學」的 Toggle。

### 如何選擇「新手教學」的內容

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 遊戲教學區塊</mark>

於「新手教學種類」欄位選擇新手教學內容。

{% hint style="info" %}
更動遊戲教學的設置項目後，請確保已開啟新手教學的功能
{% endhint %}

### 如何強制玩家進入接取任務的教學？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 遊戲教學區塊</mark>

開啟「強制接任務教學」的 Toggle。

{% hint style="info" %}
如果玩家沒做過任務，且任務書有新任務可以接取，跑完新手教學後將強制進入接任務教學
{% endhint %}

### 如何強制玩家於點擊領地時進入教學

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 遊戲教學區塊</mark>

開啟「點擊領地觸發教學」的 Toggle。

### 如何於用戶第一次進入世界時播放片頭影片

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 遊戲教學區塊</mark>

開啟「片投影片播放」的 Toggle。
