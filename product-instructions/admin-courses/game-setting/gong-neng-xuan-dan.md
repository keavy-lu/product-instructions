### 如何打開或關閉課程世界裡不同功能的入口

（自選章節、挑戰塔、交易市集、大富翁版每日獎勵、奪寶信號、裝備）

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程設定 -> 進階設定 ->  功能選單區塊</mark>

請依照需求打開或關閉該世界所需的對應功能入口。

{% hint style="info" %}
1. 如需開啟或關閉魔王戰役（魔物指針）與競賽之盾的遊戲入口，請至各別的設置區塊進行設置
2. 僅建議開啟挑戰塔的課程世界打開裝備的入口
{% endhint %}
