# 課程管理介面開關

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.48.54.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何打開課程管理介面的素養數據功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程管理介面開關區塊</mark>

開啟「是否開放素養數據」的 Toggle

### 如何打開課程管理介面中課程題庫的批量上傳功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程管理介面開關區塊</mark>

開啟「開放課程題庫批量上傳」的 Toggle

### 如何打開課程管理介面的任務獎勵檢視權限？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程管理介面開關區塊</mark>

開啟「任務獎勵狀況可檢視」的 Toggle

### 如何打開課程管理介面的 APP 推播管理功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程管理介面開關區塊</mark>

開啟「是否開放 APP 推播管理」的 Toggle

{% hint style="info" %}
開啟此開關後，課程管理介面會出現 APP 推播管理的功能入口；APP 內的遊戲設定則會出現通知設定，並於 APP 內取得玩家是否同意推播通知的設定值
{% endhint %}
