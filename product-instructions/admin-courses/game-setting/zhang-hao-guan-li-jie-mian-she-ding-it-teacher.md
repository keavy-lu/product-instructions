# 帳號管理介面設定 (IT Teacher)

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.47.37.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何打開帳號管理介面？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 帳號管理介面設定區塊</mark>

於「帳號管理介面」欄位選擇設定項目

{% hint style="info" %}
目前「開放且有批次管理」的選項並無實質意義，請使用「開啟」的選項
{% endhint %}

{% hint style="info" %}
僅有課程管理者或是課程擁有者才能夠使用帳號管理介面
{% endhint %}

### 如何強制用帳號管理創建的帳號在登入時輸入Email？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 帳號管理介面設定區塊</mark>

開啟「IT Teacher 帳號輸入郵箱以找回密碼」的 Toggle。
