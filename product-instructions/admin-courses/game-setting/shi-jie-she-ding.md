<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 世界設定區塊</mark>

### 如何為遊戲世界設定合適出生點？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 世界設定區塊</mark>

調整「合適出生點」的海洋圈數。

{% hint style="info" %}
可透過「合適出生點」，設定「系統搜尋玩家出生點的範圍」，預設為 7 塊土地 + 2 圈海洋，此處可輸入 1\~16 之間的數字，僅適用於一般課程。系統會在設定範圍搜尋玩家適合出生或重生的位置，若經過 50 次仍找不到，地圖將自動擴大 5 圈再搜尋。

\
調整「合適出生點」的機制說明：

* 讓玩家在出生或重生時，可以與其他玩家保持一定的距離。
* 調整後，無論玩家是自行按下重生、使用重生道具，還是新出生在世界上，皆會套用設定。
* 舉例來說，當合適出生點設定為「7 塊土地 + 16 圈海洋」，則可確保玩家出生或重生時，至少會與其他玩家保有 16 圈的距離。
{% endhint %}
