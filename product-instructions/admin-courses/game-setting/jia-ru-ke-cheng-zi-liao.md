# 加入課程資料

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.45.44.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何在單一課程中，取得玩家的學校資訊？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 加入課程資訊區塊</mark>

開啟「填寫學校資訊」的 Toggle。

### 如何在單一課程中，取得玩家的家長資訊？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 加入課程資訊區塊</mark>

開啟「填寫學生家長資料」的 Toggle。

{% hint style="info" %}
家長資訊為：家長姓名、家長電子信箱、家長電話（選填）
{% endhint %}

### 如何在單一課程中，取得玩家的手機號碼資訊？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 加入課程資訊區塊</mark>

開啟「填寫用戶手機號碼」的 Toggle。
