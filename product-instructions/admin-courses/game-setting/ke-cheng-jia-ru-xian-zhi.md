# 課程加入限制

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.45.26.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何讓用戶在課程教材全部到期時無法進入遊戲或是教師後台？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程加入限制區塊</mark>

開啟「教材全到期後，強制回到課程頁面」的 Toggle。

### 如何限制只有持有有效教材的用戶才能進入遊戲？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程加入限制區塊</mark>

關閉「開放直接進入遊戲按鈕」的 Toggle。

{% hint style="info" %}
如開啟此 Toggle，則所有已加入該世界的玩家皆可以進入遊戲
{% endhint %}

### 如何限制用戶利用課程代碼加入課程世界？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程加入限制區塊</mark>

開啟「使用課程代碼加入世界」的 Toggle。

{% hint style="info" %}
如需要限制用戶加入課程世界，請同步將課程調整為非公開世界
{% endhint %}

### 如何選定該課程世界的到期時間？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 課程加入限制區塊</mark>

於「到期時間」的日期選擇器中選定到期時間，並填寫到期訊息。如不填寫到期訊息，將會使用系統的原始預設訊息。

{% hint style="info" %}
如不設定到期時間，可不選定該時間，或是將之前設定的數值刪除
{% endhint %}

{% hint style="info" %}
如需要限制用戶加入課程世界，請同步將課程調整為非公開世界
{% endhint %}
