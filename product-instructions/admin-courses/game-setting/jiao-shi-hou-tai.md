# 教師後台

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.47.47.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何打開教師後台的「聯絡客服」功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「客服按鈕」的 Toggle。

### 如何打開教師後台的「匯入學生」功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「匯入學生」的 Toggle。

### 如何打開遊戲介面、教師後台的「座號相關設定」？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「打開座號」的 Toggle。

### 如何打開教師後台的「分享作業」的功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「分享作業」的 Toggle。

### 如何打開教師後台的「素養教學資源」的入口？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「素養教學資源」的 Toggle。

{% hint style="info" %}
在素養數據的班級作答狀況裡，倘若班級任務列表中若該任務有教學資源，便會顯示「查看教學資源」，點擊後可到素養教學資源詳細資訊頁
{% endhint %}

### 如何打開教師後台的「一鍵出題」（匯出錯題）的功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「教師後台可匯出錯題」的 Toggle。

### 如何開放課程世界使用強襲魔王？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「這個世界是否能用強襲魔王（魔王副本）」的 Toggle。

### 如何打開「素養數據」的入口？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「素養數據」的 Toggle。

{% hint style="warning" %}
公開世界如國小天地、國中世界等，因後台無完整設定年級與班級資料，故不建議開啟
{% endhint %}

### 如何在素養數據的數據圖裡加上全國平均或是全國私校平均的參考數值欄位？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

於「新手教學種類」欄位設定勾選其他參考值（可多選）。

### 如何打開素養數據的「學生作答狀況」或「全校作答狀況」的入口？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「學生作答狀況分頁」、與「全校作答狀況分頁」的 Toggle。

### 如何選擇素養數據中校級數據的計算邏輯？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

於「校級數據計算方式」欄位設定計算邏輯。

{% hint style="info" %}
* 課程：直接計算課程世界內的數據
* 課程（2S）：依照 2S 規定預先排除部分數據後，計算課程世界內的數據
* 學校 ID：僅適用於[<mark style="color:red;">澔學</mark>](#user-content-fn-1)[^1]，一個世界內會有多個學校，會以學校 ID 作為取用數據的辨別依據，並計算學校內的數據
{% endhint %}

### 如何在學生作答狀況的素養能力長條圖加上班級平均、年級平均、全國平均的參考數值欄位？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

於「是否顯示平均數值」的設定勾選其他參考值（可多選）

### 如何讓課程世界內的所有教師皆擁有付費教師權限？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

開啟「使用付費教師權限」的 Toggle。

{% hint style="info" %}
開啟後，所有教師都能夠在選擇任務獎勵時，選擇到部分付費道具
{% endhint %}

### 如何打開「題目顯示學生上一次作答內容」的設定？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

{% hint style="info" %}
開啟後，具有該課程世界「課程管理權限」的教師帳號，出任務時可開啟「題目顯示學生上一次作答內容」功能
{% endhint %}

### 如何打開「學生答對題目後才可作答下一題」的設定？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

{% hint style="info" %}
開啟後，具有該課程世界「課程管理權限」的教師帳號，在出作業流程中可設定「學生答對題目後才可作答下一題」功能
{% endhint %}

### 如何打開「申論題型 AI 批改」的設定？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 教師後台區塊</mark>

{% hint style="info" %}
開啟後，教師在出作業流程中可設定申論題 AI 批改功能，需要進一步選擇教師後台能使用的 AI 批改方式
{% endhint %}

[^1]: 新北澔學、花蓮澔學
