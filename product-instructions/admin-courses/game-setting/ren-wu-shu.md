# 任務書

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.46.15.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何針對舊版教師後台打開任務中「完成門檻」的功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 任務書區塊</mark>

開啟「完成門檻」的 Toggle。

{% hint style="info" %}
如關閉「完成門檻」的功能，則用戶需要 100％ 完成任務，則可以點擊「完成任務」
{% endhint %}

### 如何打開教師後台裡「連續任務」的功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 任務書區塊</mark>

開啟「開放連續任務」的 Toggle。

### 如何打開教師後台裡的任務的「測驗模式」功能？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 任務書區塊</mark>

開啟「開放測驗模式」的 Toggle

{% hint style="info" %}
若開啟「測驗模式」的功能，則教師能夠在新增任務時選取測驗模式
{% endhint %}

### 如何打開遊戲介面裡「素養任務」的標籤？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 任務書區塊</mark>

開啟「任務書顯示素養任務標籤」的 Toggle

{% hint style="info" %}
若關閉「任務書顯示素養任務標籤」的功能，且此課程還是有素養任務時，則會出現在一般任務的標籤內
{% endhint %}
