# 班級設定

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.46.05.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何開啟遊戲介面中的「我的班級」功能

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 班級設定區塊</mark>

開啟「顯示我的班級」的 Toggle。

{% hint style="info" %}
若關閉此功能，則玩家無法透過教師給予的班級代碼加入班級
{% endhint %}

### 如何讓遊戲介面中的「我的班級」功能顯示教師名稱？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 班級設定區塊</mark>

開啟「班級顯示教師名稱」的 Toggle。

{% hint style="info" %}
若關閉此功能，則玩家無法從已加入班級的清單中查看班級教師的名稱
{% endhint %}

### 如何讓遊戲介面中的「我的班級」功能同時顯示教師名稱、學生座號？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 班級設定區塊</mark>

開啟「我的班級顯示班級詳細資訊」的 Toggle。

{% hint style="info" %}
班級顯示教師名稱、我的班級顯示班級詳細資訊的功能部分重疊，如需顯示班級的教師資訊，請確保兩個 toggle 皆開啟
{% endhint %}
