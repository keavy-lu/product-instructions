# 魔王戰

<figure><img src="../../../.gitbook/assets/截圖 2023-08-04 17.47.14.png" alt="" width="563"><figcaption></figcaption></figure>

### 如何打開魔物指針的入口？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 魔王戰區塊</mark>

開啟「開啟魔物指針」的 Toggle

### 如何開放玩家在參與世界魔王戰時使用一般任務？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 魔王戰區塊</mark>

開啟「世界魔王答題可選擇一般任務」的 Toggle

### 如何設定管理者能夠開設魔王任務時使用的魔王？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 魔王戰區塊</mark>

從「管理者權限魔王任務」的下拉式選單中選擇後台開放使用的魔王類型

{% hint style="info" %}
若打開此 toggle，需要具備管理者與教師的身份才可以開啟魔王任務
{% endhint %}

{% hint style="warning" %}
如果沒有下拉式選單中並無自己欲設定的魔王類型，請先至[此連結](../../game-main-function/mo-wu-zhi-zhen-she-zhi/she-zhi-guan-fang-mo-wang-zhan.md#ge-mo-wang-zhan-yi-mo-shi-de-cao-zuo-liu-cheng-wei-he)查看詳情
{% endhint %}

### 如何設定教師能夠開設魔王任務時使用的魔王？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 進階設定 -> 魔王戰區塊</mark>

從「教師權限魔王任務」的下拉式選單中選擇後台開放使用的魔王類型

{% hint style="warning" %}
1. 此功能目前公開課程都不會開放給一般老師使用，因為無法預期同時有多少老師開戰，會造成流量不可控的問題
2. 須先設定上方「管理者權限魔王任務」的魔王，此欄位才可設定
{% endhint %}
