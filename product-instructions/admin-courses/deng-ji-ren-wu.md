# 等級任務

### 如何設置該課程的等級任務？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 ->等級任務</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-07 17.05.38.png" alt="" width="563"><figcaption></figcaption></figure>

1. 輸入欲派發的任務的所屬帳號
2. 選擇欲派發的任務
3. 選擇該任務在任務書顯示的條件
4. 設定該任務的開始時間與結束時間
5. 如果該任務有設置
6. 點擊「Submit」即可創建等級任務

{% hint style="info" %}
如果要在等級任務中附帶完成獎勵，需要於創建任務時確保

> 該任務必須是任務所屬的作業中第一個創建的任務
{% endhint %}

### 如何編輯該課程的等級任務？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 ->等級任務</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-11-10 14.06.24.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊欲編輯的等級任務烈的「編輯」
2. 於頁面上方可：
   * 重新挑選任務（作業）
   * 調整該任務在任務書顯示的條件
   * 條鞥該任務的開始時間與結束時間
3. 點擊「Submit」即可儲存變更
