# 遊戲角色管理

### 如何新增帳號至課程中？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 遊戲角色管理 -> 新增帳號至課程</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-09 13.42.39.png" alt="" width="563"><figcaption></figcaption></figure>

如有已經創立的 PaGamO 帳號玩家需要加入所在的課程，可以點擊「新增帳號至課程」，輸入欲新增的帳號（請完整輸入）並檢查，即可進一步設定該角色權限、遊戲角色的到期日

{% hint style="info" %}
1. 新增帳號時，唯有帳號完全相符者才可新增至課程
2. 每一個使用者僅會有一個帳號，但在不同世界中可分別建立不同的遊戲角色
{% endhint %}

### 如何打開遊戲角色的權限？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 遊戲角色管理</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-09 13.42.39 2.png" alt="" width="563"><figcaption></figcaption></figure>

找到需要調動遊戲角色權限的帳號，可以從角色權限的欄位中查看該帳號當前所持有的權限類別 e.g. 課程擁有者、課程管理權限、老師權限、主題式閱讀管理權限

1. 課程擁有者：開啟後，即可在課程管理介面使用世界任務排序功能、素養數據功能、統計分析功能、權限調整功能、調整是否為公開世界的功能，並具有刪除課程題庫內的所有題目與改變出題者的權限，目前此角色在一個課程中僅能有一位
2. 課程管理權限：開啟後，即可在課程管理介面進行任務設置、成員管理及課程題庫管理等功能
3. 老師權限：開啟後，即可在教師後台進行學生管理、作業管理及個人題庫管理等功能
4. 主題式閱讀管理權限：開啟後，即可在題庫管理介面綁定內容供應商，且在派發任務時可上傳任務縮圖
5. 派獎勵免 QP 權限：開啟後，在教師後台派發作業時，不需消耗 QP 即可設定並派發獎勵

如需要更動遊戲角色的權限，則需要點擊操作欄位的編輯 Icon，並於角色權限欄位進行新增或刪除

{% hint style="info" %}
僅有課程擁有者才可以調整角色權限
{% endhint %}

### 如何批量修改 GC 到期日？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 遊戲角色管理</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-10-23 11.28.52.png" alt="" width="563"><figcaption></figcaption></figure>

1. 點擊「進階搜尋」，將欲修改 GC 到期日的帳號的資訊（可用帳號或是信箱）分行貼至搜尋欄
2. 點擊進階搜尋視窗的「搜尋」，進行精準搜尋（僅完全相符者才符合搜尋條件）
3. 查看搜尋資訊是否與需求相符，如正確，則可全選並點擊「批次編輯」，並選定特定的 GC 到期日
4. 點擊「更新」，即可儲存該次調整
