# 上傳題目上限設定

### 如何變更課程世界上傳題目的上限值？

<mark style="background-color:purple;">設定位置：管理者後台 -> 課程世界設定 -> 上傳題目上限設定</mark>

<figure><img src="../../.gitbook/assets/截圖 2023-08-07 16.54.06.png" alt="" width="563"><figcaption></figcaption></figure>

輸入該課程世界上傳題目的上限值，如須設定為無上限，請輸入「 -1 」

{% hint style="info" %}
此上限值的應用範圍包括：

1. 該課程世界的課程題庫管理
2. 該課程世界的教師個人題庫
{% endhint %}
