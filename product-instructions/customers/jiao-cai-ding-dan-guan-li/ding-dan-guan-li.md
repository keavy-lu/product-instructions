# 訂單管理

### 如何檢視教材購買者的訂單資訊？

<mark style="background-color:purple;">檢視位置：客服後台 -> 教材訂單管理 -> 訂單管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-21 17.17.39 (2).png" alt="" width="563"><figcaption></figcaption></figure>

1. 可透過購買者資料、玩家資料、訂單或交易編號，模糊搜尋特定的教材購買者
2. 可於列表中確認家長的購買者姓名、玩家姓名、UID、帳號、Email資訊，若確認無誤，點擊「訂單管理」，即可
   * 於列表中查看每一筆訂單的訂單編號、商品名稱、金額、訂閱狀態、下次扣款日、教材到期日、該教材所綁定到帳號 UID&#x20;
   * 如需查看訂單的扣款記錄，須點擊該訂單的「查看扣款紀錄」，查看該教材過去的扣款資訊，包括交易編號、金額、扣款時間、交易方式、交易狀態

### 如何設定教材購買者的訂單或綁定帳號？

<mark style="background-color:purple;">檢視位置：客服後台 -> 教材訂單管理 -> 訂單管理</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-21 17.17.39 (2).png" alt="" width="563"><figcaption></figcaption></figure>

1. 可透過購買者資料、玩家資料、訂單或交易編號，模糊搜尋特定的教材購買者
2. 可於列表中確認家長的購買者姓名、玩家姓名、UID、帳號、Email資訊，若確認無誤，點擊「訂單管理」，即可
   * 解除綁定帳號，須點擊該訂單的「⋯」、「解除綁定帳號」，並解除綁定
   * 設定綁定帳號，須點擊該訂單的「⋯」、「設定綁定帳號」，並選擇欲綁定的帳號。原訂單的綁定帳號的 UID 會從原本的空值改為有效值
   * 調整教材日期，須點擊該訂單的「⋯」、「調整教材日期」，並選定日期進行設置
   * 轉換訂單教材，須點擊該訂單的「⋯」、「轉換訂單教材」，並選擇週期相同或是更低的教材做轉換。原訂單的訂閱狀態會從原本的訂閱中改為「已取消」
   * 退訂訂單，須點擊該訂單的「⋯」、「退訂此訂單」，並進行設置。原訂單取消後，此付款週期後，原綁定的 PaGamO 帳號將收不到後續派發任務，且右方服務記錄會出現以下字樣「已將訂單 {訂單編號}「{商品名稱}」進行退訂。」
   * 轉移訂單，須點擊該訂單的「⋯」、「轉移此訂單」，並選擇欲接受訂單的帳號、確認資訊後即可完成移轉

{% hint style="info" %}
1) 解除綁定帳號後，將無法從家長後台看到此帳號的學習進度
2) 在調整教材到期日時，支持線上續訂的信用卡支付訂單可以調整下次扣款日；不支援線上續訂的訂單則只能調整教材到期日；若調整下次扣款日，其教材到期日會在訂單訂閱中時設為五天後，取消訂閱時設為同一天
{% endhint %}

{% hint style="info" %}
**訂單管理功能於此情形下才會是 Enable 狀態**

* 解除綁定帳號：已購買教材且已綁定至某一孩子的帳號的訂單才可設定
* 設定綁定帳號：未綁定的訂單才可設定
* 調整教材日期：已購買教材的訂單才可設定
* 轉換訂單教材：設定為可線上續訂、透過線上金流付款的訂閱制商品，且教材尚未到期，才可以轉換
* 退訂此訂單：可線上續訂的信用卡支付訂單在訂閱中時才可退訂
* 轉移此訂單：未綁定帳號且尚未到期的訂單才可以轉移
{% endhint %}
