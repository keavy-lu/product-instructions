# 學習數據

### 如何於客服後台檢視孩子的學習數據？

<mark style="background-color:purple;">檢視位置：客服後台 -> 教材訂單管理 -> 學習數據</mark>

<figure><img src="../../../.gitbook/assets/截圖 2023-08-22 13.59.11.png" alt="" width="563"><figcaption></figcaption></figure>

1. 可透過購買者資料、玩家資料、訂單或交易編號，模糊搜尋特定的教材購買者
2. 可於列表中確認家長的購買者姓名、玩家姓名、UID、帳號、Email資訊，若確認無誤，點擊「學習數據」，即可近一步查詢該教材購買者所綁定的孩子帳號的作答狀況

#### 個人學習表現

選定欲查詢的時間，並選擇教材類型、題庫類型、科目、任務來源，即可：

* 查看時間區段內的派發任務完成數與獲得的任務總數
* 查看時間區段內的任務作答狀況：總任務完成數、任務平均正確率[^1]、任務平均作答時間[^2]、總學習時數[^3]
* 查看時間區段內的閱讀素養能力分析：五種閱讀素養能裡下的表現雷達圖（可另外調整過去參考數據：半年[^4]、一年[^5]、過去所有時間[^6]、不顯示）
* 查看時間區段內的素養領域表現分析：以完成任務平均正確率[^7]排序孩子前五名的領域表現、各領域表現（任務完成個數、完成任務平均正確[^7]、派發完成率[^8]）

#### 學習表現趨勢

選定欲查詢的時間，並選擇教材類型、題庫類型、科目、任務來源，並選定欲比較的群體：全國[^9]、全國個人[^10]、全國公校[^11]、全國私校[^12]，與欲查看的數據類型：完成任務數、平均正確率、平均作答時間，即可：

* 查看個人作答情形比較折線圖
* 設定學生每個月須完成的任務數量，以月份做劃分，查看學生是否做到定期定量與穩定成長

{% hint style="info" %}
1. 作答數據是計算指定時間區間內所有完成任務的作答記錄，並不會受到訂單轉移或訂單方案改變的影響
2. 選定時間時可以調整不同時間範圍來呈現最長三年的作答數據
3. 任務來源的特殊考題為測驗模式的任務
4. 每個數據區塊都可以單獨截圖下載，方便課程顧問將圖片素材用作日常溝通以及放在學習成果表內
{% endhint %}

[^1]: 每個任務首次作答正確率平均

[^2]: 每個任務首次作答時間的平均

[^3]: 每個任務每次作答時間加總（點擊任務後到按下送出答案後的時間）

[^4]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去半年數據為 2022/03/01～2022/08/31

[^5]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去一年數據為 2021/09/01～2022/08/31

[^6]: 例：選擇時間區間為 2022/09/01～2022/12/21，過去所有時間數據為 啟用日

    ～2022/08/31

[^7]: 任務正確率總和 / 指定時間區間內完成的任務數 x 100%

[^8]: 完成任務數 / 指定時間內派發的任務數 x 100%

[^9]: 直接計算接收並完成任務的數據，不進行任何排除

[^10]: 完成任務的商品目標群眾標記為 2C&#x20;

[^11]: 使用者有被綁定標記 To G 的素養教材且有GC 在設定世界類別為 To G 的世界內

[^12]: 計算位在使用中的 2S 學校世界、身份非特殊年級、也未在被刪除的班級內的學生的數據（僅持有學生身份）
